## JpegLsPresetCodingParameters class

Définit les paramètres de codage prédéfinis JPEG-LS tels que définis dans ISO/IEC 14495-1, C.2.4.1.1. JPEG-LS définit un ensemble de paramètres par défaut, mais des paramètres personnalisés peuvent être utilisés. Lorsqu'ils sont utilisés, ces paramètres sont écrits dans le flux binaire codé car ils sont nécessaires au processus de décodage.
