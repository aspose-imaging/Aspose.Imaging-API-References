## WmfPenStyle enumeration

L'énumération PenStyle 16 bits est utilisée pour spécifier différents types de stylos pouvant être utilisés dans les opérations graphiques.

```csharp
public enum WmfPenStyle : short
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Cosmetic | `0` | Le cosmétique |
| EndcapRound | `0` | Les embouts de ligne sont ronds. |
| JoinRound | `0` | Les jointures de ligne sont rondes. |
| Solid | `0` | Le stylo est solide. |
| Dash | `1` | Le stylo est en pointillés. |
| Dot | `2` | Le stylo est pointillé. |
| Dashdot | `3` | Le stylo a des tirets et des points alternés. |
| Dashdotdot | `4` | Le stylo a des tirets et des points doubles. |
| Null | `5` | Le stylo est invisible. |
| Insideframe | `6` | Le stylo est solide. Lorsque ce stylo est utilisé dans n'importe quel enregistrement de dessin qui prend un rectangle englobant, les dimensions de la figure sont réduites afin qu'elle tienne entièrement dans le rectangle englobant, en tenant compte de la largeur du stylo. |
| Userstyle | `7` | Le stylet utilise un tableau de style fourni par l'utilisateur. |
| Alternate | `8` | Le stylet définit un pixel sur deux (ce style s'applique uniquement aux stylos cosmétiques). |
| EndcapSquare | `256` | Les embouts de ligne sont carrés. |
| EndcapFlat | `512` | Les embouts de ligne sont plats. |
| JoinBevel | `4096` | Les jointures de ligne sont biseautées. |
| JoinMiter | `8192` | Les jointures de ligne sont mitigées lorsqu'elles se situent dans la limite actuelle définie par l'enregistrement SETMITERLIMIT META_ESCAPE. Une jointure est biseautée lorsqu'elle dépasserait la limite. |
