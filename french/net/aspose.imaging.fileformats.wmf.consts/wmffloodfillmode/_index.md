## WmfFloodFillMode enumeration

L'énumération FloodFill spécifie le type d'opération de remplissage à effectuer .

```csharp
public enum WmfFloodFillMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| FloodFillBorder | `0` | La zone de remplissage est délimitée par la couleur spécifiée par le membre Color. Ce style est identique au remplissage effectué par l'enregistrement META_FLOODFILL. |
| FloodFillSurface | `1` | La zone de remplissage est délimitée par la couleur spécifiée par le membre Color . Le remplissage continue vers l'extérieur dans toutes les directions tant que la couleur est rencontrée. Ce style est utile pour remplir des zones avec bordures multicolores. |
