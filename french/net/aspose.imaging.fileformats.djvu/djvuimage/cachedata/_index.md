## DjvuImage.CacheData method

Cache les données privées.

```csharp
public override void CacheData()
```

### Exemples

L'exemple suivant montre comment mettre en cache toutes les pages d'une image DJVU.

```csharp
[C#]

string dir = "c:\\temp\\";

// Charge une image depuis un fichier DJVU.
using (Aspose.Imaging.FileFormats.Djvu.DjvuImage image = (Aspose.Imaging.FileFormats.Djvu.DjvuImage)Aspose.Imaging.Image.Load(dir + "sample.djvu"))
{
    // Cet appel met en cache toutes les pages afin qu'aucun chargement de données supplémentaire ne soit effectué à partir du flux de données sous-jacent.
    image.CacheData();

    // Ou vous pouvez mettre en cache les pages individuellement.
    foreach (Aspose.Imaging.FileFormats.Djvu.DjvuPage page in image.Pages)
    {
        page.CacheData();
    }
}
```
