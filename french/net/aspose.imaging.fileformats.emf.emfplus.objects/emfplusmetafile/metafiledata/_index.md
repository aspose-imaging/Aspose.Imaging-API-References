## EmfPlusMetafile.MetafileData property

Obtient ou définit des données de longueur variable qui spécifient le métafichier incorporé. Le contenu et le format des données peuvent être différents pour chaque type de métafichier.

```csharp
public byte[] MetafileData { get; set; }
```

### Remarques

Les images graphiques sont spécifiées par les objets EmfPlusImage (section 2.2.1.4). Un objet EmfPlusMetafile DOIT être présent dans le champ ImageData d'un objet EmfPlusImage si ImageTypeMetafile est spécifié dans son champ Type. Cet objet est générique et est utilisé pour différents types de données, notamment : Un métafichier WMF [MS-WMF] ; Métafichier WMF qui peut être placé ; Un métafichier EMF [MS-EMF] ; Un métafichier EMF+ qui spécifie les opérations graphiques avec des enregistrements EMF+ uniquement ; and Un métafichier EMF+ qui spécifie les opérations graphiques avec les enregistrements EMF+ et EMF. Voir la section 2.2.2 pour la spécification d'objets de structure supplémentaires.
