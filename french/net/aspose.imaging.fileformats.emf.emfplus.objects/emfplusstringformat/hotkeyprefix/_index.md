## EmfPlusStringFormat.HotkeyPrefix property

Obtient ou définit un entier signé 32 bits qui spécifie le type de traitement qui est effectué sur une chaîne lorsqu'un préfixe de raccourci clavier (c'est-à-dire une esperluette) est rencontré. Fondamentalement, ce champ spécifie s'il faut afficher les préfixes de raccourci clavier qui se rapportent au texte. La valeur DOIT être définie dans l'énumération HotkeyPrefix (section 2.1.1.14).
