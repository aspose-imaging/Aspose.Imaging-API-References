## EmfPlusColorBalanceEffect.MagentaGreen property

Obtient ou définit un entier signé 32 bits qui spécifie un changement dans la quantité de vert dans l'image. Cette valeur DOIT être comprise entre -100 et 100, avec les effets suivants : -100 ≤ valeur &lt; 0 Lorsque la valeur diminue, la quantité de vert dans l'image DEVRAIT diminuer et la quantité de magenta DEVRAIT augmenter. 0 Une valeur de 0 spécifie que les quantités de vert et de magenta NE DOIVENT PAS changer. 0 &lt; valeur ≤ 100 Lorsque la valeur augmente, la quantité de vert dans l'image DEVRAIT augmenter et la quantité de magenta DEVRAIT diminuer.
