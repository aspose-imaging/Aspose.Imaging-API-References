## EmfPlusGraphicsVersion class

L'objet EmfPlusGraphicsVersion spécifie la version des graphiques du système d'exploitation utilisée pour créer un métafichier EMF+ .

### Remarques

Les versions graphiques sont extensibles par le fournisseur ; cependant, pour assurer l'interopérabilité, toute extension de ce type DOIT être implémentée à la fois dans les clients et les serveurs des métafichiers EMF+.
