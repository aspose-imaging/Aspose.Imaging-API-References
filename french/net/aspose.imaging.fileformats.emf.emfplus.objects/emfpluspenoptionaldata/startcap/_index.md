## EmfPlusPenOptionalData.StartCap property

Obtient ou définit un entier signé 32 bits facultatif qui spécifie la forme pour le début d'une ligne dans le champ CustomStartCapData. Ce champ DOIT être présent si l'indicateur PenDataStartCap est défini dans le champ PenDataFlags de l'objet EmfPlusPenData, et la valeur DOIT être définie dans l'énumération LineCapType (section 2.1.1.18).
