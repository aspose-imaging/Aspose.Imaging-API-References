## EmfPlusCompressedImage.CompressedImageData property

Obtient ou définit un tableau d'octets, qui spécifient l'image compressée. Le type de compression DOIT être déterminé à partir des données elles-mêmes.

```csharp
public byte[] CompressedImageData { get; set; }
```

### Remarques

Les bitmaps sont spécifiés par les objets EmfPlusBitmap (section 2.2.2.2). Un objet EmfPlusCompressedImage DOIT être présent dans le champ BitmapData d'un objet EmfPlusBitmap si BitmapDataTypeCompressed est spécifié dans son champ Type. Cet objet est générique et est utilisé pour différents types de données compressées, notamment :  Exchangeable Image File Format (EXIF), comme spécifié dans [EXIF] ;  Graphics Interchange Format (GIF), comme spécifié dans [GIF] ;  Joint Photographic Experts Group (JPEG), comme spécifié dans [JFIF] ;  Portable Network Graphics (PNG), comme spécifié dans [RFC2083] et [W3C - PNG] ; and  Tag Image File Format (TIFF), comme spécifié dans [RFC3302] et [TIFF].
