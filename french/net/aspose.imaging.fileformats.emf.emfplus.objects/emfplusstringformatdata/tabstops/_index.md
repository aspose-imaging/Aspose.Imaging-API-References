## EmfPlusStringFormatData.TabStops property

Obtient ou définit un tableau facultatif de valeurs à virgule flottante qui spécifient les emplacements des taquets de tabulation facultatifs pour cet objet. Chaque valeur de taquet de tabulation représente le nombre d'espaces entre les taquets de tabulation ou, pour le premier taquet de tabulation, le nombre d'espaces entre le début d'une ligne de texte et le premier taquet de tabulation. Ce champ DOIT être présent si la valeur du champ TabStopCount dans l'objet EmfPlusStringFormat est supérieure à 0.

public float[] TabStops { get; set; }
