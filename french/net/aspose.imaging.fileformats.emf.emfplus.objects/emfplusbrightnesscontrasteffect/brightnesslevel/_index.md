## EmfPlusBrightnessContrastEffect.BrightnessLevel property

Obtient ou définit un entier signé 32 bits qui spécifie le niveau de luminosité. Cette valeur DOIT être comprise entre -255 et 255, avec les effets suivants : -255 ≤ valeur &lt; 0 Lorsque la valeur diminue, la luminosité de l'image DEVRAIT diminuer. 0 Une valeur de 0 indique que la luminosité NE DOIT PAS changer . 0 &lt; valeur ≤ 255 À mesure que la valeur augmente, la luminosité de l'image DEVRAIT augmenter.
