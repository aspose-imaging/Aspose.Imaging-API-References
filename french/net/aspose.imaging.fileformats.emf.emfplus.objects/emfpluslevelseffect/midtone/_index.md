## EmfPlusLevelsEffect.MidTone property

Obtient ou définit le Spécifie le degré d'éclaircissement ou d'assombrissement des tons moyens d'une image. Les valeurs de canal Color au milieu de la plage d'intensité sont plus modifiées que les valeurs proches des extrémités hautes ou basses, ce qui signifie qu'une image peut être éclaircie ou assombrie sans perdre le contraste entre les parties les plus sombres et les plus claires de l'image. -100 ≤ valeur &lt; 0 Spécifie que les tons moyens sont rendus plus sombres. 0 Spécifie que les tons moyens NE DOIVENT PAS changer. 0 &lt; valeur ≤ 100 Spécifie que les tons moyens sont rendus plus clairs.

public int MidTone { get; set; }
