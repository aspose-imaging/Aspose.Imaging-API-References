## CmxImage.Pages property

Obtient les pages.

```csharp
public override Image[] Pages { get; }
```

### Valeur de la propriété

Les pages.

### Exemples

L'exemple suivant montre comment mettre en cache toutes les pages d'une image CMX.

```csharp
[C#]

string dir = "c:\\temp\\";

// Charge une image à partir d'un fichier CMX.
using (Aspose.Imaging.FileFormats.Cmx.CmxImage image = (Aspose.Imaging.FileFormats.Cmx.CmxImage)Aspose.Imaging.Image.Load(dir + "sample.cmx"))
{
    // Cet appel met en cache uniquement la page par défaut.
    image.CacheData();

    // Mettez en cache toutes les pages afin qu'aucun chargement de données supplémentaire ne soit effectué à partir du flux de données sous-jacent.
    foreach (Aspose.Imaging.FileFormats.Cmx.CmxImagePage page in image.Pages)
    {
        page.CacheData();
    }
}
```
