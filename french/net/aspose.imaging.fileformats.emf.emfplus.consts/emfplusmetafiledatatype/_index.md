## EmfPlusMetafileDataType enumeration

L'énumération MetafileDataType définit les types de données de métafichiers qui peuvent être incorporées dans un métafichier EMF+.

```csharp
public enum EmfPlusMetafileDataType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| MetafileDataTypeWmf | `1` | Spécifie que le métafichier est un métafichier WMF qui spécifie les opérations graphiques avec des enregistrements WMF, comme spécifié dans [MS-WMF]. |
| MetafileDataTypeWmfPlaceable | `2` | Spécifie que le métafichier est un métafichier WMF qui spécifie les opérations graphiques avec des enregistrements WMF et qui contient des informations d'en-tête supplémentaires qui rendent le métafichier WMF indépendant du périphérique, comme spécifié dans [MS-WMF]. |
| MetafileDataTypeEmf | `3` | Spécifie que le métafichier est un métafichier EMF qui spécifie les opérations graphiques avec des enregistrements EMF, comme spécifié dans [MS-EMF]. |
| MetafileDataTypeEmfPlusOnly | `4` | Spécifie que le métafichier est un métafichier EMF+ qui spécifie les opérations graphiques avec des enregistrements EMF+ uniquement. |
| MetafileDataTypeEmfPlusDual | `5` | Spécifie que le métafichier est un métafichier EMF+ qui spécifie les opérations graphiques avec les enregistrements EMF et EMF+. |

### Remarques

Les données de métafichier incorporées sont spécifiées par les objets EmfPlusMetafileData (section 2.2.2.27).
