## EmfPlusDriverStringOptionsFlags enumeration

Les drapeaux DriverStringOptions spécifient les propriétés de positionnement et de rendu du texte graphique. Ces drapeaux peuvent être combinés pour spécifier plusieurs options.

```csharp
[Flags]
public enum EmfPlusDriverStringOptionsFlags
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| DriverStringOptionsCmapLookup | `1` | Si elle est définie, les positions des glyphes de caractères DEVRAIENT être spécifiées dans une table de recherche de carte de caractères. Si elle est claire, les positions des glyphes DEVRAIENT être obtenues à partir d'un tableau de coordonnées. |
| DriverStringOptionsVertical | `2` | Si défini, la chaîne DEVRAIT être rendue verticalement. Si elle est claire, la chaîne DEVRAIT être rendue horizontalement. |
| DriverStringOptionsRealizedAdvance | `4` | Si elle est définie, les positions des glyphes de caractères DEVRAIENT être calculées par rapport à la position du premier glyphe. Si elle est désactivée, les positions des glyphes DEVRAIENT être obtenues à partir d'un tableau de coordonnées. |
| DriverStringOptionsLimitSubpixel | `8` | Si défini, moins de mémoire DEVRAIT être utilisée pour mettre en cache les glyphes anti-aliasés, ce qui produit un rendu de texte de qualité inférieure. Si clair, plus de mémoire DEVRAIT être utilisée, ce qui produit un rendu de texte de meilleure qualité. |
