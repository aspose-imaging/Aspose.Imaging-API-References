## EmfPlusCompositingMode enumeration

L'énumération CompositingMode définit les modes de combinaison des couleurs source avec les couleurs d'arrière-plan. Le mode de composition représente l'état d'activation du mélange alpha.

```csharp
public enum EmfPlusCompositingMode : byte
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| CompositingModeSourceOver | `0` | Active le mélange alpha, qui spécifie que lorsqu'une couleur est rendue, elle est mélangée avec la couleur d'arrière-plan. L'étendue du mélange est déterminée par la valeur de la composante alpha de la couleur en cours de rendu. |
| CompositingModeSourceCopy | `1` | Désactive le mélange alpha, ce qui signifie que lorsqu'une couleur source est rendue, elle écrase la couleur d'arrière-plan. |
