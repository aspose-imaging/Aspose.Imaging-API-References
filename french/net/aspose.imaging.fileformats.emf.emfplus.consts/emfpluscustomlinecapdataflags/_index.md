## EmfPlusCustomLineCapDataFlags enumeration

Les drapeaux CustomLineCapData spécifient les données pour les bouchons de ligne personnalisés. Ces drapeaux peuvent être combinés pour spécifier plusieurs options.

```csharp
[Flags]
public enum EmfPlusCustomLineCapDataFlags
```
