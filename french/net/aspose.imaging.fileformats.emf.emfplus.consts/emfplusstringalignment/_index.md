## EmfPlusStringAlignment enumeration

L'énumération StringAlignment définit les moyens d'aligner les chaînes par rapport à un rectangle de mise en page du texte.

```csharp
public enum EmfPlusStringAlignment
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| StringAlignmentNear | `0` | Spécifie que l'alignement des chaînes se fait vers l'origine du rectangle de mise en page. Cela peut être utilisé pour aligner des caractères le long d'une ligne ou pour aligner du texte dans un rectangle. Pour un rectangle de mise en page de droite à gauche, l'origine DEVRAIT être en haut à droite. |
| StringAlignmentCenter | `1` | Spécifie que l'alignement est centré entre l'origine et l'étendue du rectangle de mise en page. |
| StringAlignmentFar | `2` | Spécifie que l'alignement est sur le côté droit du rectangle de mise en page. |
