## EmfPlusLineJoinType enumeration

L'énumération LineJoinType définit les manières de joindre deux lignes dessinées par le même stylet graphique et dont les extrémités se rejoignent.

```csharp
public enum EmfPlusLineJoinType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| LineJoinTypeMiter | `0` | Spécifie une jointure en onglet. |
| LineJoinTypeBevel | `1` | Spécifie une jointure de ligne biseautée. |
| LineJoinTypeRound | `2` | Spécifie une jointure de ligne arrondie. |
| LineJoinTypeMiterClipped | `3` | Spécifie une jointure en onglet coupé. |

### Remarques

Les lignes graphiques sont spécifiées par[`EmfPlusPen`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluspen) objets (section 2.2.1.7). Une jointure de ligne rend l'intersection des deux extrémités de ligne plus continue.
