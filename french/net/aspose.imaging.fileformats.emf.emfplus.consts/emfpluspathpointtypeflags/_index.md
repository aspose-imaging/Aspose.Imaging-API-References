## EmfPlusPathPointTypeFlags enumeration

Les drapeaux PathPointType spécifient les propriétés de type des points sur les chemins graphiques. Ces drapeaux peuvent être combinés pour spécifier plusieurs options.

```csharp
[Flags]
public enum EmfPlusPathPointTypeFlags
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| PathPointTypeDashMode | `1` | Spécifie qu'un segment de ligne qui passe par le point est en pointillés. |
| PathPointTypePathMarker | `2` | Spécifie que le point est un marqueur de position. |
| PathPointTypeCloseSubpath | `8` | Spécifie que le point est l'extrémité d'un sous-chemin. |
