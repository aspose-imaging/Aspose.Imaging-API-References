## EmfPlusFilterType enumeration

L'énumération FilterType définit les types d'algorithmes de filtrage qui peuvent être utilisés pour l'amélioration de la qualité du texte et des graphiques et le rendu des images.

```csharp
public enum EmfPlusFilterType : byte
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| FilterTypeNone | `0` | Spécifie que le filtrage n'est pas effectué. |
| FilterTypePoint | `1` | Spécifie que chaque pixel de destination est calculé en échantillonnant le pixel le plus proche de l'image source. |
| FilterTypeLinear | `2` | Spécifie que l'interpolation linéaire est effectuée en utilisant la moyenne pondérée d'une zone 2x2 de pixels entourant le pixel source. |
| FilterTypeTriangle | `3` | Spécifie que chaque pixel de l'image source contribue de manière égale à l'image de destination. C'est le plus lent des algorithmes de filtrage. |
| FilterTypeBox | `4` | Spécifie un algorithme de filtre de boîte, dans lequel chaque pixel de destination est calculé en faisant la moyenne d'un rectangle de pixels source. Cet algorithme n'est utile que pour réduire la taille d'une image. |
| FilterTypePyramidalQuad | `6` | Spécifie qu'un filtre de tente à 4 échantillons est utilisé. |
| FilterTypeGaussianQuad | `7` | Spécifie qu'un filtre gaussien à 4 échantillons est utilisé, ce qui crée un effet de flou sur une image. |
