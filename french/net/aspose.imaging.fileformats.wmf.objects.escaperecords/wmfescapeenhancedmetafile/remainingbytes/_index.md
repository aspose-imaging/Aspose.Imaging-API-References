## WmfEscapeEnhancedMetafile.RemainingBytes property

Obtient ou définit les octets restants.

```csharp
public int RemainingBytes { get; set; }
```

### Valeur de la propriété

Un entier non signé 32 bits qui spécifie le nombre d'octets dans le flux EMF qui restent à traiter après cet enregistrement. Ces octets EMF supplémentaires DOIVENT suivre dans les champs EnhancedMetafileData des enregistrements d'échappement META_ESCAPE_ENHANDED_METAFILE suivants.
