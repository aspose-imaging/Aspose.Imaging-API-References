## EmfPlusRotateWorldTransform.Angle property

Obtient ou définit une valeur à virgule flottante 32 bits qui spécifie l'angle de rotation en degrés. L'opération est effectuée en construisant une nouvelle matrice de transformation à partir du diagramme suivant : ------------ ----------------------- &#x7C; péché(Angle) &#x7C; cos(Angle) &#x7C; 0 &#x7C; &#x7C; cos(Angle) &#x7C; péché(Angle) &#x7C; 0 &#x7C; ------------------------------------------------ Figure 2 : Rotation Transform Matrix La transformation actuelle de l'espace mondial est multipliée par cette matrice, et le résultat devient the new current world space transform. Le champ Flags détermine l'ordre de multiplication.

public float Angle { get; set; }
