## EmfPlusFillPolygon.IsRelative property

Obtient ou définit une valeur indiquant si cette instance est relative. S'il est défini, chaque élément de PointData spécifie un emplacement dans l'espace de coordonnées qui est relatif à l'emplacement spécifié par l'élément précédent dans le tableau.
