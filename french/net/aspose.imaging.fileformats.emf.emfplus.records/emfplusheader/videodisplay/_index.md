## EmfPlusHeader.VideoDisplay property

Obtient ou définit une valeur indiquant si la vidéo s'affiche. s'il est défini, cet indicateur indique que le métafichier a été enregistré avec un contexte de référence device pour un affichage vidéo. S'il est clair, le métafichier a été enregistré avec un contexte de référence device pour une imprimante.
