## EmfPlusHeader.EmfPlusFlags property

Obtient ou définit les indicateurs EMF plus. Un entier non signé de 32 bits qui contient des informations sur la façon dont ce métafichier a été enregistré. si le 31e bit du champ est défini, cet indicateur indique que le métafichier a été enregistré avec une référence contexte de périphérique pour un affichage vidéo. S'il est clair, le métafichier a été enregistré avec un contexte de périphérique de référence pour une imprimante.

```csharp
public int EmfPlusFlags { get; set; }
```

### Valeur de la propriété

Les drapeaux EMF plus.
