## EmfLogPenEx.Width property

Obtient ou définit un entier non signé 32 bits qui spécifie la largeur de la ligne tracée par le stylo. Si le type de stylo dans le champ PenStyle est PS_GEOMETRIC, cette valeur est la largeur en unités logiques ; sinon, la largeur est spécifiée en unités de périphérique. Si le type de stylo dans le champ PenStyle est PS_COSMETIC, cette valeur DOIT être 0x00000001.
