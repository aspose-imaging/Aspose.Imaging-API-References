## EmfLogPenEx.BrushStyle property

Obtient ou définit un entier non signé 32 bits qui spécifie un style de pinceau pour le stylet à partir de l'énumération WMF BrushStyle ([MS-WMF] section 2.1.1.4). Si le type de plume dans le champ PenStyle est PS_GEOMETRIC, cette valeur DOIT être BS_SOLID ou BS_HATCHED. La valeur de ce champ peut être BS_NULL, mais uniquement si le style de trait spécifié dans PenStyle est PS_NULL. Le style BS_NULL DEVRAIT être utilisé pour spécifier un pinceau qui n'a aucun effet.
