## EmfPixelFormatDescriptor.DwVisibleMask property

Gets ou sets spécifie la couleur transparente ou l'indice d'un plan sous-jacent. Lorsque le type de pixel est RGBA, dwVisibleMask est une valeur de couleur RVB transparente. Lorsque le type de pixel est un index de couleur, il s'agit d'une valeur d'index transparent.
