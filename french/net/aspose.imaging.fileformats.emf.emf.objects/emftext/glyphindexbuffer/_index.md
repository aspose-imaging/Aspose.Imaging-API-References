## EmfText.GlyphIndexBuffer property

Obtient ou définit le tampon d'index de glyphe facultatif. Si les options ont l'indicateur ETO_GLYPH_INDEX, les codes des caractères dans une chaîne de texte de sortie sont en fait des index des glyphes de caractères dans une police TrueType (énumération ExtTextOutOptions 2.1.11). Les index glyphes sont spécifiques à la police, donc pour afficher les caractères corrects lors de la lecture, la police utilisée DOIT être identique à la police utilisée pour générer les index.
