## InterpolationMode enumeration

Le[`InterpolationMode`](../interpolationmode) l'énumération spécifie l'algorithme utilisé lorsque les images sont mises à l'échelle ou pivotées.

```csharp
public enum InterpolationMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Invalid | `-1` | Mode d'interpolation invalide. |
| Default | `0` | Spécifie le mode par défaut. |
| Low | `1` | Spécifie une interpolation de faible qualité. |
| High | `2` | Spécifie une interpolation de haute qualité. |
| Bilinear | `3` | Spécifie l'interpolation bilinéaire. Aucun préfiltrage n'est effectué. Ce mode ne convient pas pour réduire une image à moins de 50 % de sa taille d'origine. |
| Bicubic | `4` | Spécifie l'interpolation bicubique. Aucun préfiltrage n'est effectué. Ce mode ne convient pas pour réduire une image à moins de 25 % de sa taille d'origine. |
| NearestNeighbor | `5` | Spécifie l'interpolation du plus proche voisin. |
| HighQualityBilinear | `6` | Spécifie une interpolation bilinéaire de haute qualité. Un préfiltrage est effectué pour garantir un rétrécissement de haute qualité. |
| HighQualityBicubic | `7` | Spécifie une interpolation bicubique de haute qualité. Un préfiltrage est effectué pour assurer un rétrécissement de haute qualité. Ce mode produit des images transformées de la plus haute qualité. |
