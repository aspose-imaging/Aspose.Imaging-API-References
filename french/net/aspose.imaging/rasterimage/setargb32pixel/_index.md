## RasterImage.SetArgb32Pixel method

Définit un pixel ARGB 32 bits d'image pour la position spécifiée.

```csharp
public void SetArgb32Pixel(int x, int y, int argb32Color)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| x | Int32 | L'emplacement pixel x. |
| y | Int32 | L'emplacement du pixel y. |
| argb32Color | Int32 | Le pixel ARGB 32 bits pour la position spécifiée. |

### Exemples

L'exemple suivant charge une image raster et définit la couleur d'un pixel arbitraire.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Définit la couleur du pixel en haut à gauche.
    rasterImage.SetArgb32Pixel(0, 0, Aspose.Imaging.Color.Aqua.ToArgb());

    // Une autre méthode consiste à passer directement une instance de Aspose.Imaging.Color
    rasterImage.SetPixel(0, 0, Aspose.Imaging.Color.Aqua);
}
```
