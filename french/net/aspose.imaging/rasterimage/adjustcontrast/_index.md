## RasterImage.AdjustContrast method

Image contrastée

```csharp
public virtual void AdjustContrast(float contrast)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| contrast | Single | Valeur de contraste (dans la plage [-100 ; 100]) |

### Exemples

L'exemple suivant effectue la correction du contraste d'une image.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Définit la valeur de contraste. Les valeurs de contraste acceptées sont dans la plage [-100f, 100f].
    rasterImage.AdjustContrast(50);
    rasterImage.Save(dir + "sample.AdjustContrast.png");
}
```
