## RasterImage.HasAlpha property

Obtient une valeur indiquant si cette instance a alpha.

```csharp
public virtual bool HasAlpha { get; }
```

### Valeur de la propriété

`vrai` si cette instance a alpha ; Par ailleurs,`faux` .

### Exemples

L'exemple suivant charge des images raster et imprime des informations sur le format des données brutes et le canal alpha.

```csharp
[C#]

// Les fichiers image à charger.
string[] fileNames = new string[]
{
    @"c:\temp\sample.bmp",
    @"c:\temp\alpha.png",
};

foreach (string fileName in fileNames)
{
    using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(fileName))
    {
        Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;
        System.Console.WriteLine("ImageFile={0}, FileFormat={1}, HasAlpha={2}", fileName, rasterImage.RawDataFormat, rasterImage.HasAlpha);
    }
}

// La sortie peut ressembler à ceci :
// ImageFile=c:\temp\sample.bmp, FileFormat=Rgb24Bpp, canaux utilisés : 8,8,8, HasAlpha=False
// ImageFile=c:\temp\alpha.png, FileFormat=RGBA32Bpp, canaux utilisés : 8,8,8,8, HasAlpha=True
```

L'exemple suivant montre comment extraire des informations sur le format des données brutes et le canal alpha d'une image BMP.

```csharp
[C#]

// Crée une image BMP 32-bpp de 100 x 100 px.
using (Aspose.Imaging.FileFormats.Bmp.BmpImage bmpImage = new Aspose.Imaging.FileFormats.Bmp.BmpImage(100, 100, 32, null))
{
    System.Console.WriteLine("FileFormat={0}, RawDataFormat={1}, HasAlpha={2}", bmpImage.FileFormat, bmpImage.RawDataFormat, bmpImage.HasAlpha);
};

// Crée une image BMP 24-bpp de 100 x 100 px.
using (Aspose.Imaging.FileFormats.Bmp.BmpImage bmpImage = new Aspose.Imaging.FileFormats.Bmp.BmpImage(100, 100, 24, null))
{
    System.Console.WriteLine("FileFormat={0}, RawDataFormat={1}, HasAlpha={2}", bmpImage.FileFormat, bmpImage.RawDataFormat, bmpImage.HasAlpha);
};

// En règle générale, BMP ne prend pas en charge le canal alpha, la sortie ressemblera donc à ceci :
// FileFormat = Bmp, RawDataFormat = Rgb32Bpp, canaux utilisés : 8,8,8,8, HasAlpha = False
// FileFormat = Bmp, RawDataFormat = Rgb24Bpp, canaux utilisés : 8,8,8, HasAlpha = False
```
