## RasterImage.GetPixel method

Obtient un pixel d'image.

```csharp
public Color GetPixel(int x, int y)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| x | Int32 | L'emplacement pixel x. |
| y | Int32 | L'emplacement du pixel y. |

### Return_Value

La couleur du pixel pour l'emplacement spécifié.

### Exemples

L'exemple suivant charge une image raster et obtient la couleur d'un pixel arbitraire.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Récupère la couleur du pixel en haut à gauche de l'image.
    Color color = rasterImage.GetPixel(0, 0);

    // Obtenir les valeurs des composants de couleur individuels
    byte alpha = color.A;
    byte red = color.R;
    int green = color.G;
    int blue = color.B;

    System.Console.WriteLine("The color of the pixel(0,0) is A={0},R={1},G={2},B={3}", alpha, red, green, blue);
}
```
