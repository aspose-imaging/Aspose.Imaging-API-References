## DataRecoveryMode enumeration

Le mode de récupération de données.

```csharp
public enum DataRecoveryMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Aucune récupération de données n'est implicite. Chaque fois que le format de fichier contient des données corrompues, l'exception appropriée est levée. |
| ConsistentRecover | `1` | Le mode de récupération cohérente tente de récupérer toutes les données tant que la corruption ne casse pas le format de fichier et permet un traitement ultérieur correct. |
| MaximalRecover | `2` | Le mode de récupération maximal récupère toutes les données même si le format de fichier a une structure corrompue et un traitement ultérieur peut produire des effets sans surveillance. |
