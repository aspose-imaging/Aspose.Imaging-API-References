## TiffImage.HasAlpha property

Obtient le canal alpha A.

```csharp
public override bool HasAlpha { get; }
```

### Valeur de la propriété

Le canal alpha a.

### Exemples

L'exemple suivant charge une image TIFF et imprime des informations sur le format des données brutes et le canal alpha.

```csharp
[C#]

string dir = "c:\\temp\\";

string fileName = dir + "sample.tif";
using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(fileName))
{
    Aspose.Imaging.FileFormats.Tiff.TiffImage tiffImage = (Aspose.Imaging.FileFormats.Tiff.TiffImage)image;

    // Si l'image TIFF active a un canal alpha, alors l'image TIFF entière est considérée comme ayant un canal alpha.
    System.Console.WriteLine("ImageFile={0}, FileFormat={1}, HasAlpha={2}", fileName, tiffImage.RawDataFormat, tiffImage.HasAlpha);

    int i = 0;
    foreach (Aspose.Imaging.FileFormats.Tiff.TiffFrame frame in tiffImage.Frames)
    {
        System.Console.WriteLine("Frame={0}, FileFormat={1}, HasAlpha={2}", ++i, frame.RawDataFormat, frame.HasAlpha);
    }
}

// La sortie peut ressembler à ceci :
// ImageFile=c:\temp\sample.tif, FileFormat=RgbIndexed1Bpp, canaux utilisés : 1, HasAlpha=False
// Frame=1, FileFormat=RgbIndexed1Bpp, canaux utilisés : 1, HasAlpha=False
// Frame=2, FileFormat=RgbIndexed1Bpp, canaux utilisés : 1, HasAlpha=False
```
