## PngImage.RawDataFormat property

Obtient le format des données brutes.


### Exemples

L'exemple suivant charge des images PNG et imprime des informations sur le format des données brutes et le canal alpha.

```csharp
[C#]

// Les images PNG à charger.
string[] fileNames = new string[]
{
    @"c:\temp\sample.png",
    @"c:\temp\alpha.png",
};

foreach (string fileName in fileNames)
{
    using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(fileName))
    {
        Aspose.Imaging.FileFormats.Png.PngImage pngImage = (Aspose.Imaging.FileFormats.Png.PngImage)image;
        System.Console.WriteLine("ImageFile={0}, FileFormat={1}, HasAlpha={2}", fileName, pngImage.RawDataFormat, pngImage.HasAlpha);
    }
}

// La sortie peut ressembler à ceci :
// ImageFile=c:\temp\sample.png, FileFormat=Rgb24Bpp, canaux utilisés : 8,8,8, HasAlpha=False
// ImageFile=c:\temp\alpha.png, FileFormat=RGBA32Bpp, canaux utilisés : 8,8,8,8, HasAlpha=True
```
