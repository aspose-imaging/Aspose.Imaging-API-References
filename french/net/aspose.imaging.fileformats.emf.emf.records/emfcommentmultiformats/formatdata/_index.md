## EmfCommentMultiFormats.FormatData property

Obtient ou définit un tableau de longueur variable d'octets de données d'image pour tous les formats graphiques contenus dans cet enregistrement. La taille des données pour chaque image est fournie par le champ DataSize dans l'objet EmrFormat correspondant. Ainsi, la taille totale de ce champ est la somme des valeurs DataSize dans tous les objets EmrFormat. Le format graphique des données de chaque image est spécifié par le champ Signature dans l'objet EmrFormat correspondant.
