## EmfStretchDiBits.BitBltRasterOperation property

Obtient ou définit un entier non signé 32 bits qui spécifie un code d'opération raster . Ces codes définissent comment les données de couleur du rectangle source doivent être combinées avec les données de couleur du rectangle de destination et éventuellement un motif de pinceau, pour obtenir la couleur finale.
