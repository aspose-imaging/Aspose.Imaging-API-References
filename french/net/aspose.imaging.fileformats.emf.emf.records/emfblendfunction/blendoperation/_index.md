## EmfBlendFunction.BlendOperation property

Obtient le code d'opération de mélange. La seule opération de mélange source et destination qui a été définie est 0x00, qui spécifie que le bitmap source DOIT être combiné avec le bitmap de destination en fonction des valeurs de transparence alpha des pixels source.
