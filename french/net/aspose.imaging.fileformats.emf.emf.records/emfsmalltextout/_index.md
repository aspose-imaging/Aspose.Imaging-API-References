### Remarques

Si ETO_SMALL_CHARS est défini dans le champ fuOptions, TextString contient des codes 8 bits pour les caractères , dérivés des octets de poids faible des codes de caractères Unicode UTF16-LE 16 bits, dans lesquels l'octet de poids fort est supposé être 0. Si ETO_NO_RECT est défini dans le champ fuOptions, le champ Limites n'est pas inclus dans l'enregistrement.
