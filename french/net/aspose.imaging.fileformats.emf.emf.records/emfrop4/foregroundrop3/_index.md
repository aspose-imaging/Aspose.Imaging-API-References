## EmfRop4.ForegroundRop3 property

Obtient le ROP3 de premier plan. Les 8 bits non signés les plus significatifs d'une valeur d'opération raster ternaire 24 bits à partir de l'énumération WMF Ternary Raster Operation. Ce code définit comment combiner les données de couleur de premier plan des bitmaps source et destination et du motif de pinceau.

```csharp
public byte ForegroundRop3 { get; }
```

### Valeur de la propriété
