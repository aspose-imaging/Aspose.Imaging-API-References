## EmfRop4 class

Une opération raster quaternaire, qui spécifie les opérations raster ternaires pour les couleurs de premier plan et d'arrière-plan d'un bitmap. Ces valeurs définissent comment les données de couleur de le rectangle source doivent être combinées avec les données de couleur du rectangle de destination.
