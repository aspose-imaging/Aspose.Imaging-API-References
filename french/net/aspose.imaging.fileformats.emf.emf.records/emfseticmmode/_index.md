### Remarques

Lorsque le mode ICM est activé, les couleurs spécifiées dans les enregistrements EMF DEVRAIENT être assorties, alors que le profil de couleur par défaut dans le contexte du périphérique de lecture DEVRAIT être utilisé lorsqu'un transfert de bloc de bits est effectué. Si le profil de couleur par défaut n'est pas souhaité, le mode ICM DEVRAIT être désactivé avant d'effectuer le transfert de blocs de bits.
