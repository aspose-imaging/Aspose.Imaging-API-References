## EmfMetafileHeaderExtension1 class

L'enregistrement EmfMetafileHeaderExtension1 est l'enregistrement d'en-tête utilisé dans la première extension des métafichiers EMF. Après le champ EmfHeaderExtension1, les champs restants sont facultatifs et peuvent être présents dans n'importe quel ordre.
