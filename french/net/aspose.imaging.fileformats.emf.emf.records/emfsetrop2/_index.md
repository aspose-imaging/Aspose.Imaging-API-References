### Remarques

Les modes de mélange des opérations raster binaires définissent comment combiner les couleurs source et de destination lors du dessin avec le stylet actuel. Les modes mixtes sont des codes d'opération raster binaires, représentant toutes les fonctions booléennes possibles de deux variables, utilisant les opérations binaires AND, OR et XOR (OU exclusif) et l'opération unaire NOT. Le mode de mixage concerne uniquement les périphériques raster ; il n'est pas disponible pour les périphériques vectoriels.
