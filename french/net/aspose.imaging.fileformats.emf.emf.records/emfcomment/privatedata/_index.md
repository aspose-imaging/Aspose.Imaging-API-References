## EmfComment.PrivateData property

Obtient ou définit un tableau facultatif d'octets qui spécifie les données privées. Le premier DWORD de ces données NE DOIT PAS être l'une des valeurs d'identifiant de commentaire prédéfinies spécifiées dans la section 2.3.3. Les données privées sont inconnues d'EMF ; il n'a de sens que pour les applications qui connaissent le format des données et comment les utiliser. Les enregistrements de données privées EMR_COMMENT PEUVENT être ignorés.
