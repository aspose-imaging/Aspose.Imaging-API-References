## WmfSetTextCharExtra class

L'enregistrement META_SETTEXTCHAREXTRA définit l'espacement entre les caractères pour la justification du texte dans le contexte du périphérique de lecture. L'espacement est ajouté à l'espace blanc entre chaque caractère, y compris `break` caractères, lorsqu'une ligne de texte justifié est sortie.
