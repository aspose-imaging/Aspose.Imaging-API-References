## WmfSetTextCharExtra.CharExtra property

Obtient ou définit le caractère extra.

```csharp
public int CharExtra { get; set; }
```

### Valeur de la propriété

La quantité d'espace supplémentaire, en unités logiques, à ajouter à chaque caractère. Si le mode de mappage actuel n'est pas MM_TEXT, cette valeur est transformée et arrondie au pixel le plus proche. Pour plus de détails sur la définition du mode de mappage, consultez META_SETMAPMODE (section 2.3.5.17).
