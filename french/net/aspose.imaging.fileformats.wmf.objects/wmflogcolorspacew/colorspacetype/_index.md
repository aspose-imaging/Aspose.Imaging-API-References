## WmfLogColorSpaceW.ColorSpaceType property

Obtient ou définit un entier signé 32 bits qui spécifie le type d'espace colorimétrique . Il DOIT être défini dans l'énumération LogicalColorSpace (section 2.1.1.14). Si cette valeur est LCS_sRGB ou LCS_WINDOWS_COLOR_SPACE, l'espace colorimétrique sRGB DOIT être utilisé.
