## DicomImage.CacheData method

Cache les données privées.

```csharp
public override void CacheData()
```

### Exemples

L'exemple suivant montre comment mettre en cache toutes les pages d'une image DICOM.

```csharp
[C#]

string dir = "c:\\temp\\";

// Charge une image à partir d'un fichier DICOM.
using (Aspose.Imaging.FileFormats.Dicom.DicomImage image = (Aspose.Imaging.FileFormats.Dicom.DicomImage)Aspose.Imaging.Image.Load(dir + "sample.dicom"))
{
    // Cet appel met en cache toutes les pages afin qu'aucun chargement de données supplémentaire ne soit effectué à partir du flux de données sous-jacent.
    image.CacheData();

    // Ou vous pouvez mettre en cache les pages individuellement.
    foreach (Aspose.Imaging.FileFormats.Dicom.DicomPage page in image.DicomPages)
    {
        page.CacheData();
    }
}
```
