### Exemples

Toute image vectorielle (SVG, WMF, CMX, etc.) peut être utilisée comme source pour vos images Canvas. Le code suivant crée une simple image Canvas.

```csharp
[C#]

using (var image = Image.Load(@"Sample.svg"))
{
    image.Save(@"Canvas.html", new Html5CanvasOptions
    {
        VectorRasterizationOptions = new SvgRasterizationOptions()
    });
}
```

Vous pouvez intégrer plusieurs images Canvas dans une page HTML ou mettre à jour une page déjà existante. Pour ce faire, vous devez exporter uniquement la balise Canvas.

```csharp
[C#]

using (var image = Image.Load(@"Sample.svg"))
{
    image.Save(@"Canvas.html", new Html5CanvasOptions
    {
        VectorRasterizationOptions = new SvgRasterizationOptions(),
        FullHtmlPage = false
    });
}
```
