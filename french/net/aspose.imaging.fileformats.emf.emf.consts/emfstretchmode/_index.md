## EmfStretchMode enumeration

L'énumération StretchMode est utilisée pour spécifier comment les données de couleur sont ajoutées ou supprimées des bitmaps qui sont étirés ou compressés.

```csharp
public enum EmfStretchMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| STRETCH_ANDSCANS | `1` | Effectue une opération booléenne ET en utilisant les valeurs de couleur pour les pixels éliminés et existants. Si le bitmap est un bitmap monochrome, ce mode préserve les pixels noirs au détriment des pixels blancs |
| STRETCH_ORSCANS | `2` | Effectue une opération booléenne OU en utilisant les valeurs de couleur pour les pixels éliminés et existants. Si le bitmap est un bitmap monochrome, ce mode préserve les pixels blancs au détriment des pixels noirs. |
| STRETCH_DELETESCANS | `3` | Supprime les pixels. Ce mode supprime toutes les lignes de pixels éliminées sans essayer de conserver leurs informations. |
| STRETCH_HALFTONE | `4` | Mappe les pixels du rectangle source en blocs de pixels dans le rectangle de destination. La couleur moyenne sur le bloc de pixels de destination se rapproche de la couleur des pixels source. |
