## EmfModifyWorldTransformMode enumeration

L'énumération ModifyWorldTransformMode définit les modes d'utilisation des données de transformation spécifiées pour modifier la transformation de l'espace universel en espace de page actuellement définie dans le contexte du périphérique de lecture.

```csharp
public enum EmfModifyWorldTransformMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| MWT_IDENTITY | `1` | Réinitialisez la transformation actuelle à l'aide de la matrice d'identité. Dans ce mode, les données de transformation spécifiées sont ignorées |
| MWT_LEFTMULTIPLY | `2` | Multipliez la transformation actuelle. Dans ce mode, les données de transformation spécifiées sont le multiplicande gauche et la transformation actuellement définie dans le contexte du périphérique de lecture est le multiplicande droit |
| MWT_RIGHTMULTIPLY | `3` | Multipliez la transformation actuelle. Dans ce mode, les données de transformation spécifiées sont le multiplicande droit, et la transformation actuellement définie dans le contexte du périphérique de lecture est le multiplicande gauche |
| MWT_SET | `4` | Exécute la fonction d'un enregistrement EMR_SETWORLDTRANSFORM (section 2.3.12.2). |
