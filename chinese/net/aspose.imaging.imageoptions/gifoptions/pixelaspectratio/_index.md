## GifOptions.PixelAspectRatio property

获取或设置 GIF 像素纵横比。

```csharp
public byte PixelAspectRatio { get; set; }
```

### 适当的价值

GIF 像素纵横比。

### 评论

像素纵横比 - 用于计算纵横比 近似值的因子原始图像中像素的比例。如果该字段的 值不为 0，则该纵横比的近似值 根据以下公式计算: Aspect Ratio = (Pixel Aspect Ratio + 15) / 64 像素纵横比定义为像素的 宽度除以其高度的商。此字段中的值范围允许 以 1/64 为增量指定 4:1 的最宽像素到 1:4 的最高像素。 值: 0 - 没有给出纵横比信息。 1..255 - 计算中使用的值。
