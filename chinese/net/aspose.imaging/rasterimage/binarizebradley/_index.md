## BinarizeBradley(double, int) {#binarizebradley_1}

使用 Bradley 自适应阈值算法对图像进行二值化，使用积分图像阈值

```csharp
public virtual void BinarizeBradley(double brightnessDifference, int windowSize)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| brightnessDifference | Double | 像素与以该像素为中心的 sxs 像素窗口的平均值之间的亮度差。 |
| windowSize | Int32 | 以该像素为中心的像素的 sxs 窗口的大小 |

### 例子

以下示例使用 Bradley 的自适应阈值算法和指定的窗口大小对光栅图像进行二值化。二值化图像仅包含 2 种颜色 - 黑色和白色。

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // 将亮度差为 5 的图像二值化。亮度是一个像素与以该像素为中心的 10 x 10 像素窗口的平均值之间的差。
    rasterImage.BinarizeBradley(5, 10);
    rasterImage.Save(dir + "sample.BinarizeBradley5_10x10.png");
}
```
