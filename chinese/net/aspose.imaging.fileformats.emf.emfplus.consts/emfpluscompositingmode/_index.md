## EmfPlusCompositingMode enumeration

CompositingMode 枚举定义了将源颜色与背景颜色组合的模式。合成模式表示 Alpha 混合的启用状态。

```csharp
public enum EmfPlusCompositingMode : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| CompositingModeSourceOver | `0` | 启用 alpha 混合，它指定在渲染颜色时，它与背景颜色混合。混合的程度由正在渲染的颜色的 alpha 分量的值决定。 |
| CompositingModeSourceCopy | `1` | 禁用 alpha 混合，这意味着当渲染源颜色时，它会覆盖背景颜色。 |
