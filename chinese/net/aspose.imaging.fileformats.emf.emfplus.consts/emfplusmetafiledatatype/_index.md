## EmfPlusMetafileDataType enumeration

MetafileDataType 枚举定义了可以嵌入到 EMF+ 元文件中的元文件数据类型。


### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| MetafileDataTypeWmf | `1` | 指定元文件是一个 WMF 元文件，它指定对 WMF 记录的图形操作，如 [MS-WMF] 中所指定。 |
| MetafileDataTypeWmfPlaceable | `2` | 指定元文件是一个 WMF 元文件，它指定使用 WMF 记录的图形操作，并包含使 WMF 元文件独立于设备的附加标头信息，如 [MS-WMF] 中所指定。 |
| MetafileDataTypeEmf | `3` | 指定元文件是一个 EMF 元文件，它指定使用 EMF 记录的图形操作，如 [MS-EMF] 中所指定。 |
| MetafileDataTypeEmfPlusOnly | `4` | 指定元文件是 EMF+ 元文件，它指定仅使用 EMF+ 记录的图形操作。 |
| MetafileDataTypeEmfPlusDual | `5` | 指定元文件是一个 EMF+ 元文件，它指定 EMF 和 EMF+ 记录的图形操作。 |
