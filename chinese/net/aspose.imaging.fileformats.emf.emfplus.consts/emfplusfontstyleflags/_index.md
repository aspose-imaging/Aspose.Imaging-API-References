## EmfPlusFontStyleFlags enumeration

FontStyle 标志指定图形字体字体的样式。这些标志可以组合起来指定多个选项。

```csharp
[Flags]
public enum EmfPlusFontStyleFlags
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| FontStyleBold | `1` | 如果设置，字体字体必须以较重的重量或厚度呈现。 如果清除，字体必须以正常粗细呈现。 |
| FontStyleItalic | `2` | 如果设置，字体字体必须以相对于基线增加的角度或倾斜的字符的垂直词干呈现。 如果清除，字体字体必须以正常角度的字符垂直词干呈现。 |
| FontStyleUnderline | `4` | 如果设置，字体字体必须在字符基线下方呈现一行。 如果清除，字体字体必须在基线下方没有线条的情况下呈现。 |
| FontStyleStrikeout | `8` | 如果设置，字体字体必须以平行于通过字符中间绘制的基线的线呈现。 如果清除，字体字体必须在字符中没有一行的情况下呈现。 |
