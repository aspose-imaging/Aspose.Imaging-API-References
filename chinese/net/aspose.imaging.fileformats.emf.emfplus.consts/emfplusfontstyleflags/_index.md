## EmfPlusFontStyleFlags enumeration

FontStyle 标志指定图形字体字体的样式。这些标志可以组合起来指定多个选项。

```csharp
[Flags]
public enum EmfPlusFontStyleFlags
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| FontStyleBold | `1` | 如果设置，字体必须以较重的粗细或粗细呈现。 如果清除，字体必须以正常粗细呈现。 |
| FontStyleItalic | `2` | 如果设置，字体字体必须使用字符的垂直词干以相对于基线增加的角度或倾斜来呈现。 如果清除，字体字体必须使用字符的垂直词干以正常角度呈现。 |
| FontStyleUnderline | `4` | 如果设置，字体字体必须在字符基线下方使用线条呈现。 如果清除，字体字体必须在基线下方没有线条呈现。 |
| FontStyleStrikeout | `8` | 如果设置，字体必须使用平行于通过字符中间绘制的基线的线来呈现。 如果清除，字体必须在没有穿过字符的线的情况下呈现。 |
