## EmfPlusColorBalanceEffect.MagentaGreen property

获取或设置一个 32 位有符号整数，它指定图像中 green 数量的变化。此值必须在 -100 到 100 的范围内，效果如下 ： -100 ≤ value &lt; 0 随着值的减小，图像中的绿色量应该减少， 洋红色的量应该增加。 0 值0 表示绿色和洋红色的量不得改变。 0 &lt; value ≤ 100 随着值的增加，图像中的绿色量应该增加， 洋红色的量应该减少。
