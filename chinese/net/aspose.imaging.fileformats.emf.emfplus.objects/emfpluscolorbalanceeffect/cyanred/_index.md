## EmfPlusColorBalanceEffect.CyanRed property

获取或设置一个 32 位有符号整数，指定 图像中红色量的变化。该值必须在 -100 到 100 的范围内，其效果如下: -100 ≤ value &lt; 0 随着值的减小，图像中的红色量应该减少，而 青色的量应该增加。 0 值 0 指定红色和青色的数量不得改变。 0 &lt; value ≤ 100 随着值的增加，图像中的红色量应该增加，而 青色的量应该减少。
