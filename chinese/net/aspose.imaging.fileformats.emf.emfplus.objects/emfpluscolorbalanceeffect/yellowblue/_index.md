## EmfPlusColorBalanceEffect.YellowBlue property

获取或设置一个 32 位有符号整数，指定图像中蓝色量的变化 。该值必须在 -100 到 100 的范围内，效果如下： -100 ≤ value &lt; 0 随着值的减小，图像中的蓝色量应该减少， 黄色的量应该增加。 0 一个值为 0 表示蓝色和黄色的量不得改变。 0 &lt; value ≤ 100 随着值的增加，图像中的蓝色量应该增加，而 黄色的量应该减少。
