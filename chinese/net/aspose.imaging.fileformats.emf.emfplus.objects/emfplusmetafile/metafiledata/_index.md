## EmfPlusMetafile.MetafileData property

获取或设置指定嵌入元文件的可变长度数据。每个元文件类型的数据内容 和格式可以不同。

```csharp
public byte[] MetafileData { get; set; }
```

### 评论

图形图像由 EmfPlusImage 对象指定（第 2.2.1.4 节）。 EmfPlusMetafile 对象必须出现在 EmfPlusImage 对象的 ImageData 字段中，如果 ImageTypeMetafile 在其类型字段中指定。 此对象是通用的，用于不同类型的数据，包括： WMF 元文件 [MS-WMF]; 可以放置的 WMF 元文件； 一个 EMF 元文件 [MS-EMF]； 一个 EMF+ 元文件，它指定仅使用 EMF+ 记录的图形操作； and 一个 EMF+ 元文件，它指定具有 EMF+ 和 EMF 记录的图形操作。 有关其他结构对象的规范，请参见第 2.2.2 节。
