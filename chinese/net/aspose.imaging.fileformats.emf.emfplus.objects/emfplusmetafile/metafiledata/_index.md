## EmfPlusMetafile.MetafileData property

获取或设置指定嵌入元文件的可变长度数据。每种元文件类型的数据内容 和格式可能不同。

```csharp
public byte[] MetafileData { get; set; }
```

### 评论

图形图像由 EmfPlusImage 对象指定（第 2.2.1.4 节）。如果在其 Type 字段中指定了 ImageTypeMetafile，则 EmfPlusMetafile 对象必须存在于 EmfPlusImage 对象的 ImageData 字段中。 该对象是通用的，用于不同类型的数据，包括: WMF 元文件 [MS-WMF]； 可以放置的WMF图元文件； 一个 EMF 元文件 [MS-EMF]； 一个 EMF+ 元文件，指定仅使用 EMF+ 记录的图形操作；和 一个 EMF+ 元文件，它指定 EMF+ 和 EMF 记录的图形操作。 附加结构对象的规范见第 2.2.2 节。
