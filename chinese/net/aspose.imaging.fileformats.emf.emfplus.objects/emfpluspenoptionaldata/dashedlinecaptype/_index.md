## EmfPlusPenOptionalData.DashedLineCapType property

获取或设置可选的 32 位有符号整数，该整数指定虚线中每个短划线的 两端的形状。如果在 EmfPlusPenData 对象的 PenDataFlags 字段中设置了 PenDataDashedLineCap 标志，则该字段必须存在 ，并且必须在 DashedLineCapType 枚举 （第 2.1.1.10 节）中定义 值。

public EmfPlusDashedLineCapType DashedLineCapType { get; set; }
