## EmfPlusLevelsEffect.MidTone property

获取或设置 指定图像的中间色调变亮或变暗的程度。强度范围中间的 Color 通道值比高 或低端附近的值变化更多，这意味着图像可以变亮或变暗而不会丢失图像最暗部分和最亮部分之间的对比度 。 -100 ≤ 值&lt; 0 指定中间调变暗。 0 指定中间调不能改变。 0 &lt; value ≤ 100 指定中间调变亮。
