## EmfPlusColorCurveEffect.AdjustmentIntensity property

获取或设置一个 32 位有符号整数，该整数指定 曲线调整到由 CurveChannel 指定的颜色通道的强度。该字段有意义的 值的范围根据 CurveAdjustment 值而有所不同: 曝光调整范围: -255 ≤ value &lt; 0 随着值的减小，图像的曝光应该减少。 0 值 0 指定曝光不能改变。 0 &lt; value ≤ 255 随着值的增加，图像的曝光应该增加。 密度调整范围: -255 ≤ value &lt; 0 随着值的减小，图像的密度应该减小，导致 图像更暗。 0 值 0 指定密度不能改变。 0 &lt; value ≤ 255 随着值的增加，图像的密度应该增加。 对比度调整范围: -100 ≤ value &lt; 0 随着值的减小，图像的对比度应该降低。 0 值 0 指定对比度不能改变。 0 &lt; value ≤ 100 随着值的增加，图像的对比度应该增加。 高光调整范围: -100 ≤ value &lt; 0 随着值的减小，图像的明亮区域应该显得更暗。 0 值 0 指定高亮不能改变。 0 &lt; value ≤ 100 随着值的增加，图像的亮区应该看起来更亮。 阴影调整范围: -100 ≤ value &lt; 0 随着值的减小，图像的暗区应该显得更暗。 0 值 0 指定阴影不能改变。 0 &lt; value ≤ 100 随着值的增加，图像的暗区应该看起来更亮。 白色饱和度调整范围: 0 — 255 随着值的增加，颜色通道强度范围的上限也会增加。 黑色饱和度调整范围: 0 — 255 随着值的增加，颜色通道强度范围的下限增加。
