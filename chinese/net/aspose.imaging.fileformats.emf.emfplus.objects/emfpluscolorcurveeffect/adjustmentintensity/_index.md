## EmfPlusColorCurveEffect.AdjustmentIntensity property

获取或设置一个 32 位有符号整数，该整数指定 曲线调整到由 CurveChannel 指定的颜色通道的强度。此字段的meaning 值的范围根据CurveAdjustment值而有所不同，如下所示： 曝光调整范围： -255 ≤ value &lt; 0 随着值的减小，图像的曝光应该减少。 0 值为 0 表示曝光不能改变。 0 &lt; value ≤ 255 随着值的增加，图像的曝光应该增加。 浓度调整范围： -255 ≤ value &lt; 0 随着值的减小，图像的浓度应该减小, 导致 图像更暗。 0 值 0 指定密度不能改变。 0 &lt; 值 ≤ 255 随着值的增加，图像的密度应该增加。 对比度调整范围： -100 ≤ 值&lt; 0 随着值的减小，图像的对比度应该降低。 0 值 0 指定对比度不能改变。 0 &lt; 值 ≤ 100 随着值的增加，图像的对比度应该增加e. 高光调整范围： -100 ≤ value &lt; 0 随着值的减小，图像的明亮区域应该显得更暗。 0 值 0 表示高光不能改变。 0 &lt; value ≤ 100 As值增加，图像的亮区应该显得更亮。 阴影调整范围： -100 ≤ value &lt; 0 随着值的减小，图像的暗区应该显得更暗。 0 值 0 表示shadow MUST NOT change. 0 &lt; value ≤ 100 随着值的增加，图像的暗区应该变亮。 白色饱和度调整范围： 0 — 255 随着值的增加，颜色通道范围的上限强度增加。 黑色饱和度调整范围： 0 — 255 随着值的增加，颜色通道强度范围的下限增加。
