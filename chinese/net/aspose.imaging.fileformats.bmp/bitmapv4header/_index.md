## BitmapV4Header class

BitmapV4Header结构是位图信息头文件。它是 BITMAPINFOHEADER 结构的扩展版本。  BitmapV4Header 结构扩展为允许将 JPEG 或 PNG 图像作为源图像传递给 StretchDIBits。

```csharp
public class BitmapV4Header : BitmapInfoHeader
```
