## EmfPlusDrawBeziers.Compressed property

获取或设置一个指示PointData是否被压缩的值。 如果设置，PointData 使用 16 位整数坐标指定坐标空间中的绝对位置。如果清除，PointData 指定绝对位置 在具有 32 位浮点坐标的坐标空间中。 注意如果设置了相对标志（如下），则该标志未定义并且必须被忽略。

```csharp
public bool Compressed { get; set; }
```

### 适当的价值

` true` 如果压缩；否则，` false` 。
