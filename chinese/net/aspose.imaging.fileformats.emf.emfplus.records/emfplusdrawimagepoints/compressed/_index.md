## EmfPlusDrawImagePoints.Compressed property

获取或设置一个指示PointData是否被压缩的值。 该位指示PointData字段是否指定压缩数据。 如果设置，PointData 使用 16 位整数 坐标指定坐标空间中的绝对位置。如果清除，PointData 使用 32 位浮点坐标指定坐标空间中的绝对位置。 注意如果设置了 P 标志（如下），则该标志未定义并且必须被忽略。

public bool Compressed { get; set; }
