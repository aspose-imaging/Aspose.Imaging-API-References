## EmfPlusHeader.EmfPlusFlags property

获取或设置 EMF plus 标志。 一个 32 位无符号整数，包含有关如何记录此元文件的信息。 如果设置了该字段的第 31 位，则此标志指示元文件是使用 视频显示的参考设备上下文记录的。如果清除，则元文件记录为 打印机的参考设备上下文。

```csharp
public int EmfPlusFlags { get; set; }
```

### 适当的价值

EMF 加标志。
