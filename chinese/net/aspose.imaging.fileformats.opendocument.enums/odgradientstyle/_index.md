### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Axial | `0` | 轴向 定义了双线性渐变，也称为反射渐变或镜像线性渐变。 它被创建为沿其轴镜像（或反射）的线性渐变。 |
| Ellipsoid | `1` | 椭圆体 定义了一个渐变，其中颜色沿着定义的 椭圆体中心的半径混合使用 draw:cx 和 draw:cy 属性。 半长轴的长度是填充区域的宽度和半短轴的长度 |

| Radial | `3` | 径向 定义了一个渐变，其中颜色沿着定义的圆的中心 沿半径混合使用 draw:cx 和 draw:cy 属性。 圆的外侧用结束色填充。 |
| Rectangle | `4` | 矩形 定义了一个渐变，它产生从矩形中心到最短 的矩形混合4 个边界。矩形的中心由属性 draw:cx 和 draw:cy 定义。 矩形的宽度就是填充区域的宽度， 矩形的高度就是填充区域的高度。 正方形的外面用结束色填充。 |
| Square | `5` | 正方形 定义产生正方形混合的渐变，模仿走廊或 空中的视觉透视金字塔的视图。 也称为“盒子渐变”和“金字塔渐变”。 正方形的中心由 draw:cx 和 draw:cy 属性定义。 正方形的宽度和高度是填充区域的宽度或高度的最小值。 正方形的外面用结束色填充。 |
| None | `6` | 渐变样式为无 |
