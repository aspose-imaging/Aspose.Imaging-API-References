## EmfStretchMode enumeration

StretchMode 枚举用于指定如何在拉伸或压缩的位图中添加或删除颜色数据。

```csharp
public enum EmfStretchMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| STRETCH_ANDSCANS | `1` | 使用已消除像素和现有像素的颜色值执行布尔 AND 运算。 如果位图是单色位图，此模式会保留黑色像素，但会牺牲白色像素 |
| STRETCH_ORSCANS | `2` | 使用已消除像素和现有像素的颜色值执行布尔 OR 运算。 如果位图是单色位图，此模式会以黑色像素为代价保留白色像素。 |
| STRETCH_DELETESCANS | `3` | 删除像素。此模式会删除所有消除的像素线，而不尝试保留其信息。 |
| STRETCH_HALFTONE | `4` | 将源矩形中的像素映射到目标矩形中的像素块。 目标像素块上的平均颜色接近源像素的颜色。 |
