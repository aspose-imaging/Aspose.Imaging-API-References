## WmfFontQuality enumeration

FontQuality Enumeration 指定在渲染文本时逻辑字体的属性应该与物理字体的属性匹配到何种程度。

```csharp
public enum WmfFontQuality : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Default | `0` | 指定字体的字符质量无所谓，所以 DRAFT可以用 |
| Draft | `1` | 指定字体的字符质量不如逻辑属性的 匹配重要。对于光栅化字体，应该启用缩放，这 意味着更多的字体大小可用。 |
| Proof | `2` | 指定字体的字符质量比逻辑属性的 匹配更重要。对于光栅化字体，应该禁用缩放，并且应该选择大小最接近的 font 。 |
| Nonantialiased | `3` | 指定当 渲染文本 时不应使用抗锯齿 |
| Antialiased | `4` | 指定在渲染文本时应该使用抗锯齿，如果 字体支持它。 |
| Cleartype | `5` | 指定在 渲染文本时应该使用 ClearType 抗锯齿，如果字体支持的话。 |
