## WmfBrushStyle enumeration

BrushStyle 枚举指定可用于图形操作的不同可能的画笔类型 。有关详细信息，请参阅画笔对象的 规范（第 2.2.1.1 节）。

```csharp
public enum WmfBrushStyle
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Solid | `0` | 绘制单一、恒定颜色（纯色或抖动）的画笔。 |
| Null | `1` | 什么都不做的画笔。在图形操作中使用 NULL 画笔 必须与完全不使用画笔具有相同的效果 |
| Hatched | `2` | 在纯色背景上绘制预定义的简单图案或“影线”的画笔 |
| Pattern | `3` | 绘制由位图定义的图案的画笔，位图可以是 Bitmap16 对象 或 DeviceIndependentBitmap (DIB) 对象。 |
| Indexed | `4` | 不支持。 |
| Dibpattern | `5` | 由 DIB 指定的图案画笔。 |
| Dibpatternpt | `6` | 由 DIB 指定的图案画笔。 |
| Pattern8X8 | `7` | 不支持。 |
| Dibpattern8X8 | `8` | 不支持。 |
| Monopattern | `9` | 不支持。 |
| Gradient | `10` | 渐变 wmf 中不支持，odg 中支持。 |
