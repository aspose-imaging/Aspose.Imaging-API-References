## WmfPitchFont enumeration

PitchFont 枚举定义了用于指定字体特征的值。 这些值用于指示字体中的字符是具有固定宽度还是可变宽度或间距。

```csharp
public enum WmfPitchFont : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| DEFAULT_PITCH | `0` | 默认音高，取决于实现。 |
| FIXED_PITCH | `1` | 固定间距，即字体中的所有字符以字符串形式输出时占用相同的 宽度。 |
| VARIABLE_PITCH | `2` | 可变间距，这意味着字体中的字符占用宽度 与输出时字形的实际宽度成正比细绳。例如， "i" 和空格字符的宽度通常比 "W" 或 "O" 字符小得多。 |
