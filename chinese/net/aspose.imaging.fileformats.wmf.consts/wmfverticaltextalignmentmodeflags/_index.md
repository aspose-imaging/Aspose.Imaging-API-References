## WmfVerticalTextAlignmentModeFlags enumeration

VerticalTextAlignmentMode 标志指定参考点和 a 边界矩形之间的关系，用于文本对齐。这些标志可以组合以指定多个选项， 限制只能选择一个标志来改变播放中的绘图位置 设备上下文。 当字体具有垂直默认基线时执行垂直文本对齐，例如汉字.

```csharp
public enum WmfVerticalTextAlignmentModeFlags
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| VTA_TOP | `0` | 参考点必须位于边界矩形的上边缘。 |
| VTA_RIGHT | `0` | 参考点必须在边界矩形的右边缘。 |
| VTA_BOTTOM | `2` | 参考点必须位于边界矩形的底部边缘。 |
| VTA_CENTER | `6` | 参考点必须与边界矩形的中心垂直对齐。 |
| VTA_LEFT | `8` | 参考点必须在边界矩形的左边缘。 |
| VTA_BASELINE | `24` | 参考点必须在文本的基线上。 |
