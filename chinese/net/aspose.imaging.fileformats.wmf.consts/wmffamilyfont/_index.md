## WmfFamilyFont enumeration

FamilyFont 枚举指定字体系列。字体系列以 描述字体的外观。它们用于在所需的确切字体不可用时指定字体。

```csharp
public enum WmfFamilyFont : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Dontcare | `0` | 指定默认字体，具体取决于实现。 |
| Roman | `1` | 具有可变笔划宽度的字体，与 字形的实际宽度成比例，并且带有衬线。 “MS Serif”就是一个例子。 |
| Swiss | `2` | 具有可变笔划宽度的字体，与 字形的实际宽度成比例，并且没有衬线。 “MS Sans Serif”就是一个例子。 |
| Modern | `3` | 具有恒定笔画宽度的字体，带或不带衬线。固定宽度字体是 通常是现代的。 “Pica”、“Elite”和“Courier New”就是示例。 |
| Script | `4` | 字体设计为看起来像手写。 “脚本”和“草书”就是例子。 |
