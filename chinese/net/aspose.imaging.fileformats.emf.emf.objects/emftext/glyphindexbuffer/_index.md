## EmfText.GlyphIndexBuffer property

获取或设置可选的字形索引缓冲区。 如果选项具有 ETO_GLYPH_INDEX 标志，则输出文本字符串中的字符代码实际上是 TrueType 字体（2.1.11 ExtTextOutOptions 枚举）中字符字形的索引 。字形索引是字体特定的， 因此要在播放时显示正确的字符，使用的字体必须与用于 生成索引的字体相同。

public int[] GlyphIndexBuffer { get; set; }
