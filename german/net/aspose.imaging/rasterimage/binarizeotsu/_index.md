## RasterImage.BinarizeOtsu method

Binarisierung eines Bildes mit Otsu-Thresholding

```csharp
public virtual void BinarizeOtsu()
```

### Beispiele

Das folgende Beispiel digitalisiert ein Rasterbild mit Otsu-Schwellenwertbildung. Binarisierte Bilder enthalten nur 2 Farben - Schwarz und Weiß.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Das Bild mit Otsu-Schwellenwerten binarisieren.
    rasterImage.BinarizeOtsu();
    rasterImage.Save(dir + "sample.BinarizeOtsu.png");
}
```
