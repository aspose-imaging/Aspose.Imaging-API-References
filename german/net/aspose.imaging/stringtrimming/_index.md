## StringTrimming enumeration

Gibt an, wie Zeichen aus einer Zeichenfolge abgeschnitten werden, die nicht vollständig in eine Layoutform passt.

```csharp
public enum StringTrimming
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Gibt kein Trimmen an. |
| Character | `1` | Gibt an, dass der Text auf das nächste Zeichen gekürzt wird. |
