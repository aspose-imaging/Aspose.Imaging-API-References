## SetWrapMode(WrapMode) {#setwrapmode}

Legt den Umbruchmodus fest, der verwendet wird, um zu entscheiden, wie eine Textur über eine Form oder an Formgrenzen gekachelt wird. Eine Textur wird über eine Form gekachelt, um sie auszufüllen, wenn die Textur kleiner ist als die Form, die sie ausfüllt.

```csharp
public void SetWrapMode(WrapMode mode)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| mode | WrapMode | Ein Element von[`WrapMode`](../../wrapmode) die angibt, wie wiederholte Kopien eines Bildes verwendet werden, um einen Bereich zu kacheln. |
