## DicomImage.CacheData method

Speichert die Daten privat.

```csharp
public override void CacheData()
```

### Beispiele

Das folgende Beispiel zeigt, wie alle Seiten eines DICOM-Bildes zwischengespeichert werden.

```csharp
[C#]

string dir = "c:\\temp\\";

// Laden Sie ein Bild aus einer DICOM-Datei.
using (Aspose.Imaging.FileFormats.Dicom.DicomImage image = (Aspose.Imaging.FileFormats.Dicom.DicomImage)Aspose.Imaging.Image.Load(dir + "sample.dicom"))
{
    // Dieser Aufruf speichert alle Seiten im Cache, sodass kein zusätzliches Laden von Daten aus dem zugrunde liegenden Datenstrom durchgeführt wird.
    image.CacheData();

    // Oder Sie können die Seiten einzeln zwischenspeichern.
    foreach (Aspose.Imaging.FileFormats.Dicom.DicomPage page in image.DicomPages)
    {
        page.CacheData();
    }
}
```
