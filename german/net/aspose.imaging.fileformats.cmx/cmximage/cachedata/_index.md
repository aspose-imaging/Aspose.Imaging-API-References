## CmxImage.CacheData method

Zwischenspeichert die Daten und stellt sicher, dass kein zusätzliches Laden von Daten aus der zugrunde liegenden Datei durchgeführt wird[`DataStreamContainer`](../../../aspose.imaging/datastreamsupporter/datastreamcontainer) .

```csharp
public override void CacheData()
```

### Beispiele

Das folgende Beispiel zeigt, wie alle Seiten eines CMX-Bildes zwischengespeichert werden.

```csharp
[C#]

string dir = "c:\\temp\\";

// Laden Sie ein Bild aus einer CMX-Datei.
using (Aspose.Imaging.FileFormats.Cmx.CmxImage image = (Aspose.Imaging.FileFormats.Cmx.CmxImage)Aspose.Imaging.Image.Load(dir + "sample.cmx"))
{
    // Dieser Aufruf speichert nur die Standardseite.
    image.CacheData();

    // Alle Seiten zwischenspeichern, damit kein zusätzliches Laden von Daten aus dem zugrunde liegenden Datenstrom durchgeführt wird.
    foreach (Aspose.Imaging.FileFormats.Cmx.CmxImagePage page in image.Pages)
    {
        page.CacheData();
    }
}
```
