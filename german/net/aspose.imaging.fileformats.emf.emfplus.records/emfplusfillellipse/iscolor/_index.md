## EmfPlusFillEllipse.IsColor property

Ruft einen Wert ab oder setzt einen Wert, der angibt, ob diese Instanz eine Farbe ist. Falls gesetzt, gibt BrushId eine Farbe als EmfPlusARGB-Objekt an (Abschnitt 2.2.2.1). Wenn leer, enthält BrushId den Index eines EmfPlusBrush-Objekts (Abschnitt 2.2.1.1 ) in der EMF+-Objekttabelle.

public bool IsColor { get; set; }
