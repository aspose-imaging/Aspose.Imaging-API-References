## EmfPlusRotateWorldTransform.Angle property

Ruft einen 32-Bit-Gleitkommawert ab oder legt ihn fest, der den Drehwinkel in Grad angibt. Die Operation wird ausgeführt, indem eine neue Transformationsmatrix aus dem folgenden -Diagramm konstruiert wird: ------------ ---------------------- &#x7C; Sünde(Winkel) &#x7C; cos(Winkel) &#x7C; 0 &#x7C; &#x7C; cos(Winkel) &#x7C; Sünde(Winkel) &#x7C; 0 &#x7C; ---------------------------------- Abbildung 2: Rotationstransformationsmatrix Die aktuelle Weltraumtransformation wird multipliziert durch diese Matrix, und das Ergebnis wird die neue aktuelle Weltraumtransformation. Das Feld Flags bestimmt die Reihenfolge der Multiplikation.
