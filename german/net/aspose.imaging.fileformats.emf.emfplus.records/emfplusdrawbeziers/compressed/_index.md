## EmfPlusDrawBeziers.Compressed property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob PointData komprimiert ist. Falls gesetzt, gibt PointData absolute Positionen im Koordinatenraum mit 16-Bit-Ganzzahlkoordinaten an. Wenn leer, gibt PointData absolute Positionen im Koordinatenraum mit 32-Bit-Gleitkommakoordinaten an. Hinweis Wenn das Relative-Flag (unten) gesetzt ist, ist dieses Flag undefiniert und MUSS ignoriert werden.

```csharp
public bool Compressed { get; set; }
```

### Eigentumswert

`Stimmt` wenn komprimiert; Andernfalls,`FALSCH` .
