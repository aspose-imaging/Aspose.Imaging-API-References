## EmfPlusFillPolygon.IsCompressed property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob diese Instanz komprimiert ist. Falls gesetzt, gibt PointData absolute Positionen im Koordinatenraum mit 16-Bit- -Ganzzahlkoordinaten an. Wenn leer, gibt PointData absolute Positionen im Koordinatenraum mit 32-Bit-Gleitkommakoordinaten an.

public bool IsCompressed { get; set; }
