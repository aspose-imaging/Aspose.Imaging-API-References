## EmfPlusDrawClosedCurve.Tension property

Ruft die Spannung ab oder legt sie fest. Eine 32-Bit-Gleitkommazahl, die angibt, wie stark sich der Spline biegt, wenn er durch die Punkte läuft. Ein Wert von 0 gibt an, dass der Spline eine Folge von geraden Linien ist. Mit steigendem Wert wird die Kurve runder. Weitere Informationen finden Sie unter [SPLINE77] und [PETZOLD].
