## EmfPlusRegionNode.RegionNodeData property

Ruft optionale Daten variabler Länge ab oder legt diese fest, die das im Feld Typ angegebene Datenobjekt region node definieren. Inhalt und Format der Daten können für jeden region Knotentyp unterschiedlich sein. Dieses Feld DARF NICHT vorhanden sein, wenn der node -Typ RegionNodeDataTypeEmpty oder RegionNodeDataTypeInfinite ist. Dieses Objekt ist generisch und wird verwendet, um verschiedene Arten von Regionsknotendaten anzugeben, einschließlich: Ein EmfPlusRegionNodePath-Objekt (Abschnitt 2.2.2.42) für einen Endknoten; An EmfPlusRectF-Objekt (Abschnitt 2.2.2.39) für einen Endknoten; and Ein EmfPlusRegionNodeChildNodes-Objekt (Abschnitt 2.2.2.41) für einen Nicht-Endknoten.
