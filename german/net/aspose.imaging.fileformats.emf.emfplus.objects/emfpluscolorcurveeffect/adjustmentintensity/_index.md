## EmfPlusColorCurveEffect.AdjustmentIntensity property

Ruft eine 32-Bit-Ganzzahl mit Vorzeichen ab oder legt diese fest, die die Intensität der -Kurvenanpassung an den durch CurveChannel angegebenen Farbkanal angibt. Die Bereiche sinnvoller Werte für dieses Feld variieren je nach CurveAdjustment-Wert wie folgt: Belichtungsanpassungsbereich: -255 ≤ Wert &lt; 0 Wenn der Wert abnimmt, SOLLTE die Belichtung des Bildes abnehmen. 0 Ein Wert von 0 gibt an dass sich die Belichtung NICHT ändern darf. 0 &lt; Wert ≤ 255 Wenn der Wert zunimmt, SOLLTE die Belichtung des Bildes zunehmen. Dichteanpassungsbereich: -255 ≤ Wert &lt; 0 Wenn der Wert abnimmt, SOLLTE die Dichte des Bildes abnehmen , was zu einem dunkleren Bild führt. 0 Ein Wert von 0 gibt an, dass sich die Dichte NICHT ändern darf. 0 &lt; Wert ≤ 255 Wenn der Wert zunimmt, SOLLTE die Dichte des Bildes zunehmen. Kontrastanpassungsbereich: -100 ≤ Wert &lt; 0 Mit abnehmendem Wert SOLLTE der Kontrast des Bildes abnehmen. 0 Ein Wert von 0 gibt an, dass sich der Kontrast NICHT ändern DARF. 0 &lt; Wert ≤ 100 Mit zunehmendem Wert SOLLTE der Kontrast des Bildes zunehmen e. Hervorhebungsanpassungsbereich: -100 ≤ Wert &lt; 0 Wenn der Wert abnimmt, SOLLTEN die hellen Bereiche des Bildes dunkler erscheinen. 0 Ein Wert von 0 gibt an, dass sich die Hervorhebung NICHT ändern darf. 0 &lt; Wert ≤ 100 As Wenn der Wert zunimmt, SOLLTEN die hellen Bereiche des Bilds heller erscheinen. Schattenanpassungsbereich: -100 ≤ Wert &lt; 0 Wenn der Wert abnimmt, SOLLTEN die dunklen Bereiche des Bilds dunkler erscheinen. 0 Ein Wert von 0 gibt an, dass die Schatten darf sich nicht ändern. 0 &lt; Wert ≤ 100 Wenn der Wert zunimmt, SOLLTEN die dunklen Bereiche des Bildes heller erscheinen. Einstellbereich für die Weißsättigung: 0 — 255 Wenn der Wert zunimmt, die obere Grenze des Bereichs des Farbkanals Intensitäten erhöht. Schwarzsättigungs-Einstellbereich: 0 — 255 Wenn der Wert zunimmt, erhöht sich die untere Grenze des Bereichs der Farbkanalintensitäten.
