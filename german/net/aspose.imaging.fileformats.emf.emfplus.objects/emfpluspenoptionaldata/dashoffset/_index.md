## EmfPlusPenOptionalData.DashOffset property

Ruft einen optionalen 32-Bit-Gleitkommawert ab oder legt diesen fest, der den -Abstand vom Anfang einer Zeile bis zum Anfang des ersten Leerzeichens in einem gestrichelten Linienmuster angibt. Dieses Feld MUSS vorhanden sein, wenn das PenDataDashedLineOffset-Flag im Feld PenDataFlags des EmfPlusPenData-Objekts gesetzt ist.

public float DashOffset { get; set; }
