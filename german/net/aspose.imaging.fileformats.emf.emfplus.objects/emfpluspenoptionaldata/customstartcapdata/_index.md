## EmfPlusPenOptionalData.CustomStartCapData property

Ruft das optionale EmfPlusCustomStartCapData-Objekt (Abschnitt 2.2.2.15) ab oder legt es fest, das die benutzerdefinierte Anfangskappenform definiert, die die Form ist, die am Anfang einer mit diesem Stift gezeichneten Linie verwendet. Es kann ein beliebiges in verschiedenen Formen sein, z. B. ein Quadrat, ein Kreis oder eine Raute. Dieses Feld MUSS vorhanden sein, wenn das PenDataCustomStartCap-Flag im PenDataFlags-Feld des EmfPlusPenData-Objekts gesetzt ist

public EmfPlusCustomStartCapData CustomStartCapData { get; set; }
