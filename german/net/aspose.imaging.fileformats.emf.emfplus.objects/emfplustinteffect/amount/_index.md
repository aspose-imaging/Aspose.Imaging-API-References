## EmfPlusTintEffect.Amount property

Ruft ab oder legt eine 32-Bit-Ganzzahl mit Vorzeichen fest, die angibt, wie stark der Farbton verstärkt oder abgeschwächt wird. -100 ≤ Wert &lt; 0 Negative Werte geben an, wie stark der Farbton abgeschwächt wird, was der Hinzufügung von Schwarz entspricht. 0 Ein Wert von 0 gibt an, dass sich der Farbton NICHT ändern darf. 0 &lt; Wert ≤ 100 Positive Werte geben an, wie stark der Farbton verstärkt wird, was der Zugabe von Weiß entspricht.
