## EmfPlusCompressedImage.CompressedImageData property

Holt oder setzt ein Array von Bytes, die das komprimierte Bild angeben. Die Art der Komprimierung MUSS aus den Daten selbst bestimmt werden.

```csharp
public byte[] CompressedImageData { get; set; }
```

### Bemerkungen

Bitmaps werden durch EmfPlusBitmap-Objekte (Abschnitt 2.2.2.2) spezifiziert. Ein EmfPlusCompressedImage -Objekt MUSS im BitmapData-Feld eines EmfPlusBitmap-Objekts vorhanden sein, wenn BitmapDataTypeCompressed in seinem Type-Feld angegeben ist. Dieses Objekt ist generisch und wird für verschiedene Arten komprimierter Daten verwendet, einschließlich:  Exchangeable Image File Format (EXIF), wie in [EXIF] spezifiziert;  Graphics Interchange Format (GIF), wie in [GIF] spezifiziert;  Joint Photographic Experts Group (JPEG), wie in [JFIF] spezifiziert;  Portable Network Graphics (PNG), as spezifiziert in [RFC2083] und [W3C - PNG]; and  Tag Image File Format (TIFF), wie in [RFC3302] und[TIFF]. angegeben
