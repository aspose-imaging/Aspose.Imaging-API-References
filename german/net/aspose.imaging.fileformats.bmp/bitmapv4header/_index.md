## BitmapV4Header class

Die BitmapV4Header-Struktur ist die Bitmap-Informations-Header-Datei. Es ist eine erweiterte Version der BITMAPINFOHEADER-Struktur. Die BitmapV4Header-Struktur wird erweitert, damit ein JPEG- oder PNG-Bild als Quellbild an StretchDIBits übergeben werden kann.

```csharp
public class BitmapV4Header : BitmapInfoHeader
```
