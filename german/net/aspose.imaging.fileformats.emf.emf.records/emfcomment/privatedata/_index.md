## EmfComment.PrivateData property

Ruft ein optionales Array von Bytes ab oder legt es fest, das die privaten Daten angibt. Das erste DWORD dieser Daten DARF KEINEN der vordefinierten Kommentarkennungswerte sein, die in Abschnitt 2.3.3 angegeben sind. Private Daten sind EMF unbekannt; Es ist nur für Anwendungen von Bedeutung, die das Format der -Daten kennen und wissen, wie es verwendet wird. EMR_COMMENT private Datensätze KÖNNEN ignoriert werden.

public byte[] PrivateData { get; set; }
