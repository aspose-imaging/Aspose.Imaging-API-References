## EmfPolygon16 class

Der Datensatz EMR_POLYGON16 gibt ein Polygon an, das aus zwei oder mehr Scheitelpunkten besteht, die durch gerade Linien verbunden sind. Das Polygon wird mit dem aktuellen Stift umrissen und mit dem aktuellen Pinsel und Polygon-Füllmodus gefüllt. Das Polygon wird automatisch geschlossen, indem eine Linie vom letzten Eckpunkt zum ersten gezogen wird.

public sealed class EmfPolygon16 : EmfDrawingRecordType
