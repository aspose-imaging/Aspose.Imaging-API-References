## EmfSelectPalette.IhPal property

Ruft eine 32-Bit-Ganzzahl ohne Vorzeichen ab oder legt sie fest, die entweder den Index eines LogPalette-Objekts (Abschnitt 2.2.17) in der EMF-Objekttabelle oder den Wert DEFAULT_PALETTE angibt, bei dem es sich um den Index einer Bestandsobjektpalette aus der StockObject-Enumeration (Abschnitt 2.1.31).

```csharp
public int IhPal { get; set; }
```

### Bemerkungen

Dieser Wert DARF NICHT Null oder der Index eines anderen Bestandsobjekts sein.
