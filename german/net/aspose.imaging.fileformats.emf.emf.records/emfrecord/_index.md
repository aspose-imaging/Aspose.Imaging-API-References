## EmfRecord class

Basisklasse für EMF-Records Alle EMF-Records MÜSSEN eine Länge haben, die ein Vielfaches von 4 Bytes ist. Dies wird in den generischen Strukturen der vorhergehenden EMF-Record-Typen dargestellt, indem an den Enden dieser Strukturen gegebenenfalls AlignmentPadding fields eingefügt werden. Der Inhalt von AlignmentPadding fields MUSS immer ignoriert werden. Der Kürze halber werden diese Felder nicht in jeder einzelnen EMF -Datensatzdefinition angezeigt.
