## EmfBeginPath class

Dieser Datensatz öffnet eine Pfadklammer im Kontext des aktuellen Wiedergabegeräts. Nachdem eine Pfadklammer geöffnet ist, kann eine Anwendung mit der Verarbeitung von Datensätzen beginnen, um die Punkte zu definieren, die im Pfad liegen. Eine Anwendung MUSS eine offene Pfadklammer schließen, indem sie den EMR_ENDPATH verarbeitet Datensatz. Wenn eine Anwendung den Datensatz EMR_BEGINPATH verarbeitet, MÜSSEN alle vorherigen Pfade aus dem Kontext des Wiedergabegeräts verworfen werden.
