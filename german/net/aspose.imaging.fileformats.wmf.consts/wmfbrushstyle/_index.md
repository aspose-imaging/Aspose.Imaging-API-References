## WmfBrushStyle enumeration

Die BrushStyle-Enumeration spezifiziert die verschiedenen möglichen Pinseltypen , die in Grafikoperationen verwendet werden können. Weitere Informationen finden Sie in der -Spezifikation des Brush-Objekts (Abschnitt 2.2.1.1).

```csharp
public enum WmfBrushStyle
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Solid | `0` | Ein Pinsel, der eine einzelne, konstante Farbe malt, entweder durchgehend oder gedithert. |
| Null | `1` | Ein Pinsel, der nichts tut. Die Verwendung eines NULL-Pinsels in einer Grafikoperation MUSS denselben Effekt haben wie die Verwendung von gar keinem Pinsel |
| Hatched | `2` | Ein Pinsel, der ein vordefiniertes einfaches Muster oder eine „Schraffur“ auf einen einfarbigen Hintergrund malt |
| Pattern | `3` | Ein Pinsel, der ein Muster malt, das durch eine Bitmap definiert ist, die ein Bitmap16-Objekt oder ein DeviceIndependentBitmap (DIB)-Objekt sein KANN. |
| Indexed | `4` | Nicht unterstützt. |
| Dibpattern | `5` | Ein von einer DIB spezifizierter Musterpinsel. |
| Dibpatternpt | `6` | Ein von einer DIB spezifizierter Musterpinsel. |
| Pattern8X8 | `7` | Nicht unterstützt. |
| Dibpattern8X8 | `8` | Nicht unterstützt. |
| Monopattern | `9` | Nicht unterstützt. |
| Gradient | `10` | Der Farbverlauf Wird in wmf nicht unterstützt, unterstützt in odg. |
