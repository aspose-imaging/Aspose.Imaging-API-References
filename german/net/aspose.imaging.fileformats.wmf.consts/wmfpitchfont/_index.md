## WmfPitchFont enumeration

Die PitchFont-Enumeration definiert Werte, die zum Spezifizieren von Merkmalen einer Schriftart verwendet werden. Die Werte werden verwendet, um anzugeben, ob die Zeichen in einer Schriftart eine feste oder variable Breite oder einen festen Abstand haben.

```csharp
public enum WmfPitchFont : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| DEFAULT_PITCH | `0` | Die Standardtonhöhe, die implementierungsabhängig ist. |
| FIXED_PITCH | `1` | Eine feste Teilung, was bedeutet, dass alle Zeichen in der Schriftart die gleiche Breite einnehmen, wenn sie in einem String ausgegeben werden. |
| VARIABLE_PITCH | `2` | Ein variabler Abstand, was bedeutet, dass die Zeichen in der Schriftart Breiten einnehmen, die proportional zu den tatsächlichen Breiten der Glyphen sind, wenn sie in einer Zeichenfolge ausgegeben werden. Beispielsweise haben die Zeichen „i“ und Leerzeichen normalerweise viel kleinere Breiten als ein „W“- oder „O“-Zeichen. |
