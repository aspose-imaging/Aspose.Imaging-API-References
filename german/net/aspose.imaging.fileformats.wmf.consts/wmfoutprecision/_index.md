## WmfOutPrecision enumeration

Die OutPrecision-Enumeration definiert Werte für die Ausgabepräzision, die für die Schriftartzuordnung erforderlich ist, um bestimmte Schriftartparameter abzugleichen, einschließlich Höhe, Breite, Zeichenausrichtung, Escapezeichen, Zeichenabstand und Schriftart.

```csharp
public enum WmfOutPrecision : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | `0` | Ein Wert, der das Standardverhalten angibt. |
| String | `1` | Ein Wert, der zurückgegeben wird, wenn gerasterte Schriftarten aufgelistet werden. |
| Stroke | `3` | Ein Wert, der zurückgegeben wird, wenn TrueType- und andere Outline-Schriftarten und Vektorschriftarten aufgelistet werden. |
| Tt | `4` | Ein Wert, der die Auswahl einer TrueType-Schriftart angibt, wenn das System mehrere Schriftarten mit demselben Namen enthält. |
| Device | `5` | Ein Wert, der die Auswahl einer Geräteschriftart angibt, wenn das System mehrere Schriftarten mit demselben Namen enthält. |
| Raster | `6` | Ein Wert, der die Auswahl einer gerasterten Schriftart angibt, wenn das System mehrere Schriftarten mit demselben Namen enthält. |
| TtOnly | `7` | Ein Wert, der die Anforderung nur für TrueType-Schriftarten angibt. Wenn keine TrueType-Schriftarten im System installiert sind, wird das Standardverhalten angegeben. |
| Outline | `8` | Ein Wert, der die Anforderung für TrueType- und andere Outline-Schriftarten angibt. |
| ScreenOutline | `9` | Ein Wert, der eine Präferenz für TrueType und andere Outline-Schriftarten angibt. |
| PsOnly | `10` | Ein Wert, der eine Anforderung nur für PostScript-Schriftarten angibt. Wenn keine PostScript-Schriftarten im System installiert sind, wird das Standardverhalten angegeben. |
