## WmfPostScriptClipping enumeration

Die PostScriptClipping-Aufzählung definiert Funktionen, die auf den Beschneidungspfad angewendet werden können, der für die PostScript-Ausgabe verwendet wird.

```csharp
public enum WmfPostScriptClipping
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| CLIP_SAVE | `0` | Speichert den aktuellen PostScript-Beschneidungspfad. |
| CLIP_RESTORE | `1` | Stellt den PostScript-Beschneidungspfad auf den letzten Beschneidungspfad wieder her, der von einer vorherigen CLIP_SAVE-Funktion gespeichert wurde , die von einem CLIP_TO_PATH-Datensatz angewendet wurde (Abschnitt 2.3.6.6). |
| CLIP_INCLUSIVE | `2` | Schneidet den aktuellen PostScript-Beschneidungspfad mit dem aktuellen Beschneidungspfad und speichert das Ergebnis als neuen PostScript-Beschneidungspfad. |
