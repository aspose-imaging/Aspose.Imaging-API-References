## DjvuImage.CacheData method

Speichert die Daten privat.

```csharp
public override void CacheData()
```

### Beispiele

Das folgende Beispiel zeigt, wie alle Seiten eines DJVU-Bildes zwischengespeichert werden.

```csharp
[C#]

string dir = "c:\\temp\\";

// Laden Sie ein Bild aus einer DJVU-Datei.
using (Aspose.Imaging.FileFormats.Djvu.DjvuImage image = (Aspose.Imaging.FileFormats.Djvu.DjvuImage)Aspose.Imaging.Image.Load(dir + "sample.djvu"))
{
    // Dieser Aufruf speichert alle Seiten im Cache, sodass kein zusätzliches Laden von Daten aus dem zugrunde liegenden Datenstrom durchgeführt wird.
    image.CacheData();

    // Oder Sie können die Seiten einzeln zwischenspeichern.
    foreach (Aspose.Imaging.FileFormats.Djvu.DjvuPage page in image.Pages)
    {
        page.CacheData();
    }
}
```
