## TransformBrush.IsTransformChanged property

Ruft einen Wert ab, der angibt, ob Transformationen auf irgendeine Weise geändert wurden. Zum Beispiel Setzen der Transformationsmatrix oder Aufrufen einer der Methoden, die die Transformationsmatrix ändern. Die Eigenschaft wird aus Gründen der Abwärtskompatibilität mit GDI+ eingeführt.

```csharp
public bool IsTransformChanged { get; }
```

### Eigentumswert

`WAHR` wenn Transformation geändert wurde; Andernfalls,`FALSCH` .
