## EmfUniversalFontId.Checksum property

Ruft eine 32-Bit-Ganzzahl ohne Vorzeichen ab, die die Prüfsumme der Schriftart ist, oder legt sie fest. Der Prüfsummenwert hat die folgenden Bedeutungen. 0x00000000 Das Objekt ist eine Geräteschriftart. 0x00000001 Das Objekt ist eine Schriftart vom Typ 1, die auf dem Clientcomputer installiert wurde und vom PostScript-Druckertreiber als Geräteschriftart aufgeführt wird. 0x00000002 Das Objekt ist keine Schriftart, sondern ein Typ-1-Rasterizer. 3 ≤ Wert Das Objekt ist eine Bitmap-, Vektor- oder TrueType-Schriftart oder eine gerasterte Type 1-Schriftart, die von einem Type 1-Rasterizer erstellt wurde.
