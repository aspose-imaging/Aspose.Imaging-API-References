## EmfLogPenEx.StyleEntry property

Ruft ein optionales Array von 32-Bit-Ganzzahlen ohne Vorzeichen ab oder legt es fest, das die Längen von Strichen und Lücken in der von diesem Stift gezeichneten Linie definiert, wenn der Wert von PenStyle der Linienstil PS_USERSTYLE für den Stift ist. Das Array enthält eine Anzahl von Einträgen, die von NumStyleEntries angegeben werden, aber es wird so verwendet, als würde es sich endlos wiederholen. Der erste Eintrag im Array gibt die Länge des ersten Bindestrichs an. Der zweite Eintrag gibt die Länge der ersten Lücke an. Danach wechseln sich Längen von Strichen und Lücken ab. Wenn der Stifttyp im Feld PenStyle PS_GEOMETRIC ist, werden die Längen in logischen Einheiten angegeben; ansonsten werden die Längen in Geräteeinheiten angegeben.
