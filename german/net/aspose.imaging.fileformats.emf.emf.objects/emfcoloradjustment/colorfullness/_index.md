## EmfColorAdjustment.Colorfullness property

Ruft eine vorzeichenbehaftete 16-Bit-Ganzzahl ab oder legt diese fest, die die auf das Quellobjekt anzuwendende Farbigkeit angibt. Dieser Wert SOLLTE im Bereich von –100 bis 100 liegen. Ein Wert von Null bedeutet, dass eine Farbanpassung NICHT durchgeführt werden darf
