## EmfColorAdjustment.Brightness property

Ruft eine 16-Bit-Ganzzahl mit Vorzeichen ab oder legt diese fest, die die auf das Quellobjekt anzuwendende Helligkeitsmenge angibt. Dieser Wert SOLLTE im Bereich von –100 bis 100 liegen. Ein Wert von Null bedeutet, dass keine Helligkeitsanpassung durchgeführt werden darf.
