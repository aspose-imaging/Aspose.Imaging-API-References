## EmfLogFontEx.Style property

Ruft eine Zeichenfolge von 32 Unicode-Zeichen ab oder legt diese fest, die den Stil der Schriftart definiert. Wenn die Länge von dieser Zeichenfolge weniger als 32 Zeichen beträgt, MUSS eine abschließende NULL vorhanden sein, danach MUSS der Rest dieses Felds ignoriert werden.
