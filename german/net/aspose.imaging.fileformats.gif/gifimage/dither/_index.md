## GifImage.Dither method

Führt Dithering auf dem aktuellen Bild durch.

```csharp
public override void Dither(DitheringMethod ditheringMethod, int bitsCount, 
    IColorPalette customPalette)
```

| bitsCount | Int32 | Die letzten Bits zählen für das Dithering. |
| customPalette | IColorPalette | Die benutzerdefinierte Palette für Dithering. |

### Beispiele

Das folgende Beispiel lädt ein GIF-Bild und führt Schwellwert- und Floyd-Dithering mit unterschiedlicher Palettentiefe durch.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.FileFormats.Gif.GifImage gifImage = (Aspose.Imaging.FileFormats.Gif.GifImage)image;

    // Schwellwert-Dithering mit 4-Bit-Farbpalette durchführen, die 16 Farben enthält.
    // Je mehr Bits angegeben werden, desto höher die Qualität und die Größe des Ausgabebildes.
    // Beachten Sie, dass derzeit nur 1-Bit-, 4-Bit- und 8-Bit-Paletten unterstützt werden.
    gifImage.Dither(Aspose.Imaging.DitheringMethod.ThresholdDithering, 4, null);

    gifImage.Save(dir + "sample.ThresholdDithering4.png", new Aspose.Imaging.ImageOptions.PngOptions());
}

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.FileFormats.Gif.GifImage gifImage = (Aspose.Imaging.FileFormats.Gif.GifImage)image;

    // Führen Sie Floyd-Dithering mit einer 1-Bit-Farbpalette durch, die nur 2 Farben enthält - Schwarz und Weiß.
    // Je mehr Bits angegeben werden, desto höher die Qualität und die Größe des Ausgabebildes.
    // Beachten Sie, dass derzeit nur 1-Bit-, 4-Bit- und 8-Bit-Paletten unterstützt werden.
    gifImage.Dither(Aspose.Imaging.DitheringMethod.FloydSteinbergDithering, 1, null);

    gifImage.Save(dir + "sample.FloydSteinbergDithering1.png", new Aspose.Imaging.ImageOptions.PngOptions());
}
```
