## WmfLogColorSpaceW.Filename property

Ruft eine optionale, nullterminierte Unicode-UTF16-LE-zeichenfolge ab oder legt diese fest, die den Namen einer Datei angibt, die ein color -Profil enthält. Wenn ein Dateiname angegeben ist und die [`ColorSpaceType`](../colorspacetype) Feld auf LCS_CALIBRATED_RGB gesetzt ist, sollten die anderen Felder dieser Struktur ignoriert werden.

public string Filename { get; set; }
