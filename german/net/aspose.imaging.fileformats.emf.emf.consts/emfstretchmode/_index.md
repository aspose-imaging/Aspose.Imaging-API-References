## EmfStretchMode enumeration

Die StretchMode-Enumeration wird verwendet, um anzugeben, wie Farbdaten zu gestreckten oder komprimierten Bitmaps hinzugefügt oder daraus entfernt werden.

```csharp
public enum EmfStretchMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| STRETCH_ANDSCANS | `1` | Führt eine boolesche UND-Operation unter Verwendung der Farbwerte für die eliminierten und vorhandenen Pixel durch. Wenn die Bitmap eine monochrome Bitmap ist, behält dieser Modus schwarze Pixel auf Kosten von weißen Pixeln bei |
| STRETCH_ORSCANS | `2` | Führt eine boolesche ODER-Operation unter Verwendung der Farbwerte für die eliminierten und vorhandenen Pixel aus. Wenn es sich bei der Bitmap um eine monochrome Bitmap handelt, behält dieser Modus weiße Pixel auf Kosten der schwarzen Pixel bei. |
| STRETCH_DELETESCANS | `3` | Löscht die Pixel. Dieser Modus löscht alle eliminierten Pixelzeilen, ohne zu versuchen, ihre Informationen zu erhalten. |
| STRETCH_HALFTONE | `4` | Ordnet Pixel aus dem Quellrechteck Pixelblöcken im Zielrechteck zu. Die durchschnittliche Farbe über den Zielpixelblock entspricht in etwa der Farbe der Quellpixel. |
