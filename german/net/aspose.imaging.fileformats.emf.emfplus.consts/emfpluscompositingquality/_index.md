## EmfPlusCompositingQuality enumeration

Die Aufzählung CompositingQuality definiert Qualitätsstufen zum Erstellen zusammengesetzter Bilder

```csharp
public enum EmfPlusCompositingQuality : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| CompositingQualityDefault | `1` | Es wird keine Gammakorrektur durchgeführt. Die Gammakorrektur steuert die Gesamthelligkeit und den Kontrast eines Bildes. Ohne Gammakorrektur können zusammengesetzte Bilder zu hell oder zu dunkel erscheinen. |
| CompositingQualityHighSpeed | `2` | Es wird keine Gammakorrektur durchgeführt. Die Compositing-Geschwindigkeit wird auf Kosten der Qualität bevorzugt. Im Ergebnis gibt es keinen Unterschied zwischen diesem Wert und CompositingQualityDefault. |
| CompositingQualityHighQuality | `3` | Gammakorrektur wird durchgeführt. Die Compositing-Qualität wird auf Kosten der Geschwindigkeit bevorzugt. |
| CompositingQualityGammaCorrected | `4` | Aktivieren Sie die Gammakorrektur für qualitativ hochwertigeres Compositing mit geringerer Geschwindigkeit. Im Ergebnis gibt es keinen Unterschied zwischen diesem Wert und CompositingQualityHighQuality. |
| CompositingQualityAssumeLinear | `5` | Es wird keine Gammakorrektur durchgeführt; Die Verwendung linearer Werte führt jedoch zu einer besseren Qualität als der Standard bei einer etwas niedrigeren Geschwindigkeit. |
