## EmfPlusFilterType enumeration

Die FilterType-Enumeration definiert Typen von Filteralgorithmen, die zur Verbesserung der Text- und Grafikqualität und zum Rendern von Bildern verwendet werden können.

```csharp
public enum EmfPlusFilterType : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| FilterTypeNone | `0` | Gibt an, dass keine Filterung durchgeführt wird. |
| FilterTypePoint | `1` | Gibt an, dass jedes Zielpixel berechnet wird, indem das nächstgelegene Pixel aus dem Quellbild abgetastet wird. |
| FilterTypeLinear | `2` | Gibt an, dass die lineare Interpolation unter Verwendung des gewichteten Durchschnitts eines 2x2 Pixelbereichs, der das Quellpixel umgibt, durchgeführt wird. |
| FilterTypeTriangle | `3` | Gibt an, dass jedes Pixel im Quellbild gleichermaßen zum Zielbild beiträgt. Dies ist der langsamste Filteralgorithmus. |
| FilterTypeBox | `4` | Gibt einen Boxfilteralgorithmus an, bei dem jedes Zielpixel berechnet wird, indem ein Rechteck aus Quellpixeln gemittelt wird. Dieser Algorithmus ist nur nützlich, wenn die Größe eines Bildes reduziert wird. |
| FilterTypePyramidalQuad | `6` | Gibt an, dass ein Zeltfilter mit 4 Stichproben verwendet wird. |
| FilterTypeGaussianQuad | `7` | Gibt an, dass ein Gaußscher Filter mit 4 Stichproben verwendet wird, der einen Weichzeichnungseffekt auf einem Bild erzeugt. |
