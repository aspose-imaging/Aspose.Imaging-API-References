## EmfPlusCompositingMode enumeration

Die CompositingMode-Enumeration definiert Modi zum Kombinieren von Quellfarben mit Hintergrundfarben. Der Compositing-Modus stellt den Aktivierungszustand von Alpha-Blending dar.

```csharp
public enum EmfPlusCompositingMode : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| CompositingModeSourceOver | `0` | Aktiviert Alpha-Blending, das angibt, dass eine Farbe beim Rendern mit der Hintergrundfarbe gemischt wird. Das Ausmaß der Mischung wird durch den Wert der Alpha-Komponente der gerenderten Farbe bestimmt. |
| CompositingModeSourceCopy | `1` | Deaktiviert Alpha-Blending, was bedeutet, dass beim Rendern einer Quellfarbe die Hintergrundfarbe überschrieben wird. |
