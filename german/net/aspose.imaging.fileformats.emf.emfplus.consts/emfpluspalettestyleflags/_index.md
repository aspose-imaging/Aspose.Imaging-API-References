## EmfPlusPaletteStyleFlags enumeration

Die PaletteStyle-Flags geben Eigenschaften von Grafikpaletten an. Diese Flags können kombiniert werden, um mehrere Optionen anzugeben.

```csharp
[Flags]
public enum EmfPlusPaletteStyleFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| PaletteStyleHasAlpha | `1` | Falls gesetzt, MÜSSEN einer oder mehrere der Paletteneinträge Alpha-Transparenzinformationen enthalten. |
| PaletteStyleGrayScale | `2` | Falls gesetzt, MUSS die Palette nur Graustufeneinträge enthalten. |
| PaletteStyleHalftone | `4` | Falls gesetzt, MUSS die Palette diskrete Farbwerte enthalten, die für die Halbtonung verwendet werden können. |
