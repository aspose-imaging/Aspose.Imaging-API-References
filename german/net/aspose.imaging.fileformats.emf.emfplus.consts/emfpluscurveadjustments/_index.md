## EmfPlusCurveAdjustments enumeration

Die Aufzählung CurveAdjustments definiert Anpassungen, die auf die Farbkurve eines Bildes angewendet werden können.

```csharp
public enum EmfPlusCurveAdjustments
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| AdjustExposure | `0` | Gibt die Simulation zum Erhöhen oder Verringern der Belichtung eines Bildes an. |
| AdjustDensity | `1` | Gibt die Simulation zum Erhöhen oder Verringern der Dichte eines Bildes an. |
| AdjustContrast | `2` | Gibt eine Erhöhung oder Verringerung des Kontrasts eines Bildes an. |
| AdjustHighlight | `3` | Gibt eine Erhöhung oder Verringerung des Wertes eines Farbkanals eines Bildes an, wenn dieser Kanal bereits einen Wert hat, der über der halben Intensität liegt. Diese Anpassung kann verwendet werden, um die Definition in den hellen Bereichen eines Bildes zu verbessern, ohne die dunklen Bereiche zu beeinträchtigen. |
| AdjustShadow | `4` | Gibt eine Erhöhung oder Verringerung des Wertes eines Farbkanals eines Bildes an, wenn dieser Kanal bereits einen Wert hat, der unter der halben Intensität liegt. Diese Anpassung kann verwendet werden, um die Auflösung in den dunklen Bereichen eines Bildes zu verbessern, ohne die hellen Bereiche zu beeinträchtigen. |
| AdjustMidtone | `5` | Gibt eine Anpassung an, die ein Bild aufhellt oder abdunkelt. Farbkanalwerte in der Mitte des Intensitätsbereichs werden stärker verändert als Farbkanalwerte in der Nähe der minimalen oder maximalen Extremwerte der Intensität. Diese Anpassung kann verwendet werden, um ein Bild aufzuhellen oder abzudunkeln, ohne den Kontrast zwischen den dunkelsten und hellsten Teilen des Bildes zu verlieren. |
| AdjustWhiteSaturation | `6` | Gibt eine Anpassung der Weißsättigung eines Bildes an, definiert als Maximalwert im Intensitätsbereich für einen bestimmten Farbkanal, dessen Bereich normalerweise 0 bis 255 beträgt. |
| AdjustBlackSaturation | `7` | Gibt eine Anpassung der Schwarzsättigung eines Bildes an, das ist der Mindestwert im Intensitätsbereich für einen bestimmten Farbkanal, der normalerweise zwischen 0 und 255 liegt. |
