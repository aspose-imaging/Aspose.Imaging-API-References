## EmfPlusCustomLineCapDataFlags enumeration

Die CustomLineCapData-Flags geben Daten für benutzerdefinierte Zeilenbegrenzungen an. Diese Flags können kombiniert werden, um mehrere Optionen anzugeben.

```csharp
[Flags]
public enum EmfPlusCustomLineCapDataFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| CustomLineCapDataFillPath | `1` | Falls gesetzt, an[`EmfPlusFillPath`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfplusfillpath) Objekt MUSS im OptionalData-Feld des angegeben werden[`EmfPlusCustomLineCapData`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluscustomlinecapdata) Objekt zum Füllen der benutzerdefinierten Zeilenkappe. |
| CustomLineCapDataLinePath | `2` | Falls gesetzt, an[`EmfPlusLinePath`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluslinepath)-Objekt MUSS im OptionalData-Feld des EmfPlusCustomLineCapData-Objekts angegeben werden, um die benutzerdefinierte Linienobergrenze zu umreißen. |

### Bemerkungen

Benutzerdefinierte Grafiklinienkappen werden angegeben durch[`EmfPlusCustomLineCap`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluscustomlinecap) Objekte.
