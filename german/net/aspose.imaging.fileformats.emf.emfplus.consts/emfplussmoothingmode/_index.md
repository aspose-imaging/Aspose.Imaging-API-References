## EmfPlusSmoothingMode enumeration

Die SmoothingMode-Enumeration definiert Glättungsmodi, die auf Linien, Kurven und die Kanten gefüllter Bereiche angewendet werden, damit sie kontinuierlicher oder scharf definierter erscheinen.

```csharp
public enum EmfPlusSmoothingMode : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| SmoothingModeDefault | `0` | Gibt die standardmäßige Kurvenglättung ohne Anti-Aliasing an. |
| SmoothingModeHighSpeed | `1` | Gibt die beste Leistung ohne Anti-Aliasing an. |
| SmoothingModeHighQuality | `2` | Gibt die beste Qualität mit Anti-Aliasing an. |
