## EmfPlusPenAlignment enumeration

Die Aufzählung PenAlignment definiert die Verteilung der Stiftbreite in Bezug auf die gezeichnete Linie.

```csharp
public enum EmfPlusPenAlignment
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| PenAlignmentCenter | `0` | Gibt an, dass das EmfPlusPen-Objekt über der theoretischen Linie zentriert wird. |
| PenAlignmentInset | `1` | Gibt an, dass der Stift auf der Innenseite der theoretischen Linie positioniert wird. |
| PenAlignmentLeft | `2` | Gibt an, dass der Stift links von der theoretischen Linie positioniert wird. |
| PenAlignmentOutset | `3` | Gibt an, dass der Stift außerhalb der theoretischen Linie positioniert wird. |
| PenAlignmentRight | `4` | Gibt an, dass der Stift rechts von der theoretischen Linie positioniert wird. |

### Bemerkungen

Grafikstifte werden angegeben durch[`EmfPlusPen`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluspen) Objekte (Abschnitt 2.2.1.7). Die Stiftausrichtung kann visualisiert werden, indem man eine theoretische eindimensionale Linie betrachtet, die zwischen zwei festgelegten Punkten gezogen wird. Die Stiftausrichtung bestimmt den Anteil der Stiftbreite, der orthogonal zur theoretischen Linie ist.
