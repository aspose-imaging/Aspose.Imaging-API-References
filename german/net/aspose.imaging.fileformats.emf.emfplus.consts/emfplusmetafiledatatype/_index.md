## EmfPlusMetafileDataType enumeration

Die MetafileDataType-Enumeration definiert Typen von Metadateidaten, die in eine EMF+-Metadatei eingebettet werden können.

```csharp
public enum EmfPlusMetafileDataType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| MetafileDataTypeWmf | `1` | Gibt an, dass die Metadatei eine WMF-Metadatei ist, die Grafikoperationen mit WMF-Einträgen angibt, wie in [MS-WMF] angegeben. |
| MetafileDataTypeWmfPlaceable | `2` | Gibt an, dass die Metadatei eine WMF-Metadatei ist, die Grafikoperationen mit WMF-Datensätzen angibt und zusätzliche Header-Informationen enthält, die die WMF-Metadatei geräteunabhängig machen, wie in [MS-WMF]. angegeben. |
| MetafileDataTypeEmf | `3` | Gibt an, dass die Metadatei eine EMF-Metadatei ist, die Grafikoperationen mit EMF-Einträgen angibt, wie in [MS-EMF]. angegeben |
| MetafileDataTypeEmfPlusOnly | `4` | Gibt an, dass die Metadatei eine EMF+-Metadatei ist, die Grafikoperationen nur mit EMF+-Datensätzen angibt. |
| MetafileDataTypeEmfPlusDual | `5` | Gibt an, dass die Metadatei eine EMF+-Metadatei ist, die Grafikoperationen mit EMF- und EMF+-Datensätzen angibt. |

### Bemerkungen

Eingebettete Metadateidaten werden durch EmfPlusMetafileData-Objekte (Abschnitt 2.2.2.27) angegeben.
