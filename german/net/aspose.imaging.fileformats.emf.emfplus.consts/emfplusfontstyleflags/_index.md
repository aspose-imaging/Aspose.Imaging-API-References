## EmfPlusFontStyleFlags enumeration

Die FontStyle-Flags geben Stile von Grafikschriftarten an. Diese Flags können kombiniert werden, um mehrere Optionen anzugeben.

```csharp
[Flags]
public enum EmfPlusFontStyleFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| FontStyleBold | `1` | Wenn gesetzt, MUSS die Schriftart mit einer stärkeren Stärke oder Dicke gerendert werden. Wenn klar, MUSS die Schriftart mit einer normalen Dicke gerendert werden. |
| FontStyleItalic | `2` | Wenn gesetzt, MUSS die Schriftart mit den vertikalen Stämmen der Zeichen in einem größeren Winkel oder einer Neigung relativ zur Grundlinie gerendert werden. Wenn klar, MUSS die Schriftart mit den vertikalen Stämmen der Zeichen in einem normalen Winkel gerendert werden. |
| FontStyleUnderline | `4` | Wenn gesetzt, MUSS die Schriftart mit einer Linie unter der Grundlinie der Zeichen gerendert werden. Wenn klar, MUSS die Schriftart ohne Linie unter der Grundlinie gerendert werden. |
| FontStyleStrikeout | `8` | Wenn gesetzt, MUSS die Schriftart mit einer Linie gerendert werden, die parallel zur Grundlinie durch die Mitte der Zeichen gezogen wird. Wenn klar, MUSS die Schriftart ohne Linie durch die Zeichen gerendert werden. |
