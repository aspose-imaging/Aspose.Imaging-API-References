## EmfPlusPathPointTypeFlags enumeration

Die PathPointType-Flags geben Typeigenschaften von Punkten auf Grafikpfaden an. Diese Flags können kombiniert werden, um mehrere Optionen anzugeben.

```csharp
[Flags]
public enum EmfPlusPathPointTypeFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| PathPointTypeDashMode | `1` | Gibt an, dass ein Liniensegment, das durch den Punkt verläuft, gestrichelt dargestellt wird. |
| PathPointTypePathMarker | `2` | Gibt an, dass der Punkt eine Positionsmarkierung ist. |
| PathPointTypeCloseSubpath | `8` | Gibt an, dass der Punkt der Endpunkt eines Unterpfads ist. |

### Bemerkungen

Grafikpfade werden angegeben durch[`EmfPlusPath`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluspath) Objekte.
