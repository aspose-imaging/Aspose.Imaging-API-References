## EmfPlusLineJoinType enumeration

Die LineJoinType-Enumeration definiert Möglichkeiten zum Verbinden zweier Linien, die mit demselben Grafikstift gezeichnet wurden und deren Enden sich treffen.

```csharp
public enum EmfPlusLineJoinType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| LineJoinTypeMiter | `0` | Gibt eine gegehrte Linienverbindung an. |
| LineJoinTypeBevel | `1` | Gibt eine abgeschrägte Linienverbindung an. |
| LineJoinTypeRound | `2` | Gibt eine abgerundete Linienverbindung an. |
| LineJoinTypeMiterClipped | `3` | Gibt eine abgeschnittene Gehrungslinienverbindung an. |

### Bemerkungen

Grafikzeilen werden angegeben durch[`EmfPlusPen`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluspen) Objekte (Abschnitt 2.2.1.7). Durch eine Linienverbindung sieht der Schnittpunkt der beiden Linienenden kontinuierlicher aus.
