## EmfPlusPaletteStyleFlags enumeration

I flag PaletteStyle specificano le proprietà delle tavolozze grafiche. Questi flag possono essere combinati per specificare più opzioni.

```csharp
[Flags]
public enum EmfPlusPaletteStyleFlags
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| PaletteStyleHasAlpha | `1` | Se impostato, una o più voci della tavolozza DEVONO contenere informazioni sulla trasparenza alfa. |
| PaletteStyleGrayScale | `2` | Se impostata, la tavolozza DEVE contenere solo voci in scala di grigi. |
| PaletteStyleHalftone | `4` | Se impostata, la tavolozza DEVE contenere valori di colore discreti che possono essere utilizzati per i mezzitoni. |
