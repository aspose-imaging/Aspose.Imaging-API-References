## EmfPlusFontStyleFlags enumeration

I flag FontStyle specificano gli stili dei caratteri tipografici dei caratteri grafici. Questi flag possono essere combinati per specificare più opzioni.

```csharp
[Flags]
public enum EmfPlusFontStyleFlags
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| FontStyleBold | `1` | Se impostato, il carattere tipografico DEVE essere visualizzato con un peso o uno spessore maggiore. Se chiaro, il carattere tipografico DEVE essere visualizzato con uno spessore normale. |
| FontStyleItalic | `2` | Se impostato, il carattere tipografico DEVE essere reso con i gambi verticali dei caratteri ad un angolo maggiore o inclinato rispetto alla linea di base. Se chiaro, il carattere tipografico DEVE essere reso con i gambi verticali dei caratteri ad un angolo normale. |
| FontStyleUnderline | `4` | Se impostato, il carattere tipografico DEVE essere visualizzato con una linea sotto la linea di base dei caratteri. Se è chiaro, il carattere tipografico DEVE essere visualizzato senza una linea sotto la linea di base. |
| FontStyleStrikeout | `8` | Se impostato, il carattere tipografico DEVE essere visualizzato con una linea parallela alla linea di base tracciata al centro dei caratteri. Se è chiaro, il carattere tipografico DEVE essere visualizzato senza una linea tra i caratteri. |
