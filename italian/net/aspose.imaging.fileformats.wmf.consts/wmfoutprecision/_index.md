## WmfOutPrecision enumeration

L'enumerazione OutPrecision definisce i valori per la precisione dell'output, che è il requisito per che il mappatore di caratteri corrisponda a parametri di carattere specifici, inclusi altezza, larghezza, orientamento del carattere, scappamento, altezza e tipo di carattere.

```csharp
public enum WmfOutPrecision : byte
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Default | `0` | Un valore che specifica il comportamento predefinito. |
| String | `1` | Un valore che viene restituito quando vengono enumerati i caratteri rasterizzati. |
| Stroke | `3` | Un valore che viene restituito quando vengono enumerati TrueType e altri caratteri di struttura e caratteri vettoriali. |
| Tt | `4` | Un valore che specifica la scelta di un font TrueType quando il sistema contiene più font con lo stesso nome. |
| Device | `5` | Un valore che specifica la scelta di un font del dispositivo quando il sistema contiene più font con lo stesso nome. |
| Raster | `6` | Un valore che specifica la scelta di un font rasterizzato quando il sistema contiene più font con lo stesso nome. |
| TtOnly | `7` | Un valore che specifica il requisito solo per i caratteri TrueType. Se non sono installati font TrueType nel sistema, viene specificato il comportamento predefinito. |
| Outline | `8` | Un valore che specifica i requisiti per TrueType e altri caratteri di struttura. |
| ScreenOutline | `9` | Un valore che specifica una preferenza per TrueType e altri caratteri di contorno . |
