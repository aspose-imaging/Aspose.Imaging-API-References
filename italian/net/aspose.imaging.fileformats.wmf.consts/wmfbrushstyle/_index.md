## WmfBrushStyle enumeration

L'enumerazione BrushStyle specifica i diversi tipi di pennello possibili che possono essere utilizzati nelle operazioni grafiche. Per ulteriori informazioni, vedere la specifica dell'oggetto Brush (sezione 2.2.1.1).

```csharp
public enum WmfBrushStyle
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Solid | `0` | Un pennello che dipinge un unico colore costante, solido o retinato. |
| Null | `1` | Un pennello che non fa nulla. L'utilizzo di un pennello NULL in un'operazione grafica DEVE avere lo stesso effetto dell'assenza di pennello |
| Hatched | `2` | Un pennello che dipinge un motivo semplice predefinito, o "tratteggio", su uno sfondo a tinta unita |
| Pattern | `3` | Un pennello che dipinge un motivo definito da una bitmap, che PUÒ essere un oggetto Bitmap16 o un oggetto DeviceIndependentBitmap (DIB). |
| Indexed | `4` | Non supportato. |
| Dibpattern | `5` | Un pennello pattern specificato da un DIB. |
| Dibpatternpt | `6` | Un pennello pattern specificato da un DIB. |
| Pattern8X8 | `7` | Non supportato. |
| Dibpattern8X8 | `8` | Non supportato. |
| Monopattern | `9` | Non supportato. |
| Gradient | `10` | Il gradiente Non supportato in wmf, supporto in odg. |
