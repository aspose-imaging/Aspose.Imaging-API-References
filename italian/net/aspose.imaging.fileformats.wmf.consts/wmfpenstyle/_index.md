## WmfPenStyle enumeration

L'enumerazione PenStyle a 16 bit viene utilizzata per specificare diversi tipi di penne che possono essere utilizzate nelle operazioni grafiche.

```csharp
public enum WmfPenStyle : short
```

| Nome | Valore | Descrizione |

| Cosmetic | `0` | Il cosmetico |

| Solid | `0` | La penna è solida. |
