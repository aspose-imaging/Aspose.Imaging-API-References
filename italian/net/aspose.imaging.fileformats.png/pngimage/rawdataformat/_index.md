## PngImage.RawDataFormat property

Ottiene il formato dei dati grezzi.


### Esempi

L'esempio seguente carica le immagini PNG e stampa le informazioni sul formato dei dati grezzi e sul canale alfa.

```csharp
[C#]

// Le immagini PNG da caricare.
string[] fileNames = new string[]
{
    @"c:\temp\sample.png",
    @"c:\temp\alpha.png",
};

foreach (string fileName in fileNames)
{
    using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(fileName))
    {
        Aspose.Imaging.FileFormats.Png.PngImage pngImage = (Aspose.Imaging.FileFormats.Png.PngImage)image;
        System.Console.WriteLine("ImageFile={0}, FileFormat={1}, HasAlpha={2}", fileName, pngImage.RawDataFormat, pngImage.HasAlpha);
    }
}

// L'output potrebbe essere simile a questo:
// ImageFile=c:\temp\sample.png, FileFormat=Rgb24Bpp, canali utilizzati: 8,8,8, HasAlpha=False
// ImageFile=c:\temp\alpha.png, FileFormat=RGBA32Bpp, canali utilizzati: 8,8,8,8, HasAlpha=True
```
