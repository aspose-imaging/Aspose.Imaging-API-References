## EmfLogFont.CharSet property

Ottiene o imposta un intero senza segno a 8 bit che specifica il set di caratteri glifi. DEVE essere un valore nell'enumerazione WMF CharacterSet ([MS-WMF] sezione 2.1.1.5). Se il set di caratteri è sconosciuto, l'elaborazione del metafile NON DOVREBBE tentare di tradurre o interpretare stringhe renderizzate con quel font.
