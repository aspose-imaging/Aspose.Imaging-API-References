## EmfPlusDrawClosedCurve.Relative property

Ottiene o imposta un valore che indica se questo[`EmfPlusDrawClosedCurve`](../../emfplusdrawclosedcurve)è relativo. Questo bit indica se il campo PointData specifica posizioni relative o assolute. Se impostato, ogni elemento in PointData specifica una posizione nello spazio delle coordinate che è relativa alla posizione specificata dall'elemento precedente nell'array. Nel caso del primo elemento in PointData, si assume una posizione precedente alle coordinate (0,0). Se deselezionato, PointData specifica posizioni assolute in base al flag C. Nota Se questo flag è impostato, il flag Compressed (sopra) non è definito e DEVE essere ignorato

```csharp
public bool Relative { get; set; }
```

### Valore della proprietà

`VERO` se relativo; altrimenti,`falso` .
