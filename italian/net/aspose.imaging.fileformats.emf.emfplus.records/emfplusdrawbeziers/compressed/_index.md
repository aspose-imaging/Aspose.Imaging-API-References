## EmfPlusDrawBeziers.Compressed property

Ottiene o imposta un valore che indica se PointData è compresso. Se impostato, PointData specifica posizioni assolute nello spazio delle coordinate con coordinate intere a 16 bit. Se chiaro, PointData specifica posizioni assolute nello spazio delle coordinate con coordinate a virgola mobile a 32 bit. Nota Se è impostato il flag Relativo (sotto), questo flag non è definito e DEVE essere ignorato.

```csharp
public bool Compressed { get; set; }
```

### Valore della proprietà

`VERO` se compresso; altrimenti,`falso` .
