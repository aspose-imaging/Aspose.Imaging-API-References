## EmfPlusDrawImagePoints.ApplyingAnEffect property

Ottiene o imposta un valore che indica se [applicando un effetto]. Questo bit indica che il rendering dell'immagine include l'applicazione di un effetto. Se impostato, un oggetto della classe Effect DEVE essere stato specificato in un record EmfPlusSerializableObject precedente (sezione 2.3.5.2).

```csharp
public bool ApplyingAnEffect { get; set; }
```

### Valore della proprietà

`VERO` se [applicando un effetto]; altrimenti,`falso` .
