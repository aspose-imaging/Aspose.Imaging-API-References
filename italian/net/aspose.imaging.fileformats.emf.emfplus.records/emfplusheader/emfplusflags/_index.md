## EmfPlusHeader.EmfPlusFlags property

Ottiene o imposta i flag EMF plus. Un numero intero senza segno a 32 bit che contiene informazioni su come è stato registrato questo metafile. se è impostato il 31° bit del campo, questo flag indica che il metafile è stato registrato con un riferimento contesto del dispositivo per una visualizzazione video. Se deselezionato, il metafile è stato registrato con un contesto di dispositivo di riferimento per una stampante.

```csharp
public int EmfPlusFlags { get; set; }
```

### Valore della proprietà

I flag EMF plus.
