## EmfPlusDrawArc.DataSize property

Ottiene o imposta la dimensione dei dati. Un intero senza segno a 32 bit che specifica il numero allineato a 32 bit di byte di dati specifici del record che seguono. Per questo tipo di record, il valore DEVE essere uno dei seguenti : 0x00000010 Se il bit C è impostato nel campo Flag. 0x00000018 Se il bit C è azzerato nel campo Flag.
