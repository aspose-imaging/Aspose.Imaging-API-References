## EmfPlusBitmap.PixelFormat property

Ottiene o imposta pixel format PixelFormat (4 byte): un intero senza segno a 32 bit che specifica il formato dei pixel che compongono l'immagine bitmap . I formati pixel supportati sono specificati in[`EmfPlusPixelFormat`](../../../aspose.imaging.fileformats.emf.emfplus.consts/emfpluspixelformat) enumerazione (sezione 2.1.1.25). Se l'immagine è compressa, in base al campo Tipo, questo valore non è definito e DEVE essere ignorato.

```csharp
public EmfPlusPixelFormat PixelFormat { get; set; }
```

### Valore della proprietà

Il formato pixel.
