## EmfPlusPenOptionalData.CustomEndCapData property

Ottiene o imposta l'oggetto EmfPlusCustomEndCapData facoltativo (sezione 2.2.2.11) che definisce la forma dell'estremità personalizzata, ovvero la forma da utilizzare alla fine di una linea disegnata con questa penna. Può essere una qualsiasi delle varie forme, come un quadrato, un cerchio o un diamante. Questo campo DEVE essere presente se il flag PenDataCustomEndCap è impostato nel campo PenDataFlags dell'oggetto EmfPlusPenData

public EmfPlusCustomEndCapData CustomEndCapData { get; set; }
