## EmfPlusMetafile.MetafileData property

Ottiene o imposta dati a lunghezza variabile che specificano il metafile incorporato. Il contenuto e il formato dei dati possono essere diversi per ogni tipo di metafile.

```csharp
public byte[] MetafileData { get; set; }
```

### Osservazioni

Le immagini grafiche sono specificate dagli oggetti EmfPlusImage (sezione 2.2.1.4). Un oggetto EmfPlusMetafile DEVE essere presente nel campo ImageData di un oggetto EmfPlusImage se ImageTypeMetafile è specificato nel relativo campo Tipo. Questo oggetto è generico e viene utilizzato per diversi tipi di dati, tra cui: Un metafile WMF [MS-WMF]; Metafile WMF che può essere posizionato; Un metafile EMF [MS-EMF]; Un metafile EMF+ che specifica le operazioni grafiche solo con record EMF+; e Un metafile EMF+ che specifica le operazioni grafiche con record EMF+ e EMF. Vedere la sezione 2.2.2 per la specifica di oggetti struttura aggiuntivi.
