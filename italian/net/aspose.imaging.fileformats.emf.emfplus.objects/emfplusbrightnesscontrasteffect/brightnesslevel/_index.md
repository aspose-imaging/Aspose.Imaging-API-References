## EmfPlusBrightnessContrastEffect.BrightnessLevel property

Ottiene o imposta un intero con segno a 32 bit che specifica il livello di luminosità. Questo valore di DEVE essere compreso tra -255 e 255, con effetti come segue: -255 ≤ valore &lt; 0 Quando il valore diminuisce, la luminosità dell'immagine DEVE diminuire. 0 Un valore di 0 specifica che la luminosità NON DEVE cambiare . 0 &lt; valore ≤ 255 All'aumentare del valore, la luminosità dell'immagine DOVREBBE aumentare.
