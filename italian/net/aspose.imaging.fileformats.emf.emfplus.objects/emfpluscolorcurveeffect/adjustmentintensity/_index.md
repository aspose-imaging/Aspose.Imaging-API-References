## EmfPlusColorCurveEffect.AdjustmentIntensity property

Ottiene o imposta un numero intero con segno a 32 bit che specifica l'intensità della regolazione della curva al canale del colore specificato da CurveChannel. Gli intervalli di valori significativi per questo campo variano in base al valore CurveAdjustment, come segue: Intervallo di regolazione dell'esposizione: -255 ≤ valore &lt; 0 Quando il valore diminuisce, l'esposizione dell'immagine DEVE diminuire. 0 Un valore di 0 specifica che l'esposizione NON DEVE cambiare. 0 &lt; valore ≤ 255 All'aumentare del valore, l'esposizione dell'immagine DEVE aumentare. Intervallo di regolazione della densità: -255 ≤ valore &lt; 0 All'aumentare del valore, la densità dell'immagine DEVE diminuire , risultando in un'immagine più scura. 0 Un valore di 0 specifica che la densità NON DEVE cambiare. 0 &lt; valore ≤ 255 All'aumentare del valore, la densità dell'immagine DOVREBBE aumentare. Intervallo di regolazione del contrasto: -100 ≤ valore &lt; 0 Al diminuire del valore, il contrasto dell'immagine DOVREBBE diminuire. 0 Un valore di 0 specifica che il contrasto NON DEVE cambiare. 0 &lt; valore ≤ 100 All'aumentare del valore, il contrasto dell'immagine DOVREBBE aumentare e. Intervallo di regolazione dell'evidenziazione: -100 ≤ valore &lt; 0 Quando il valore diminuisce, le aree chiare dell'immagine DEVONO apparire più scure. 0 Un valore di 0 specifica che l'evidenziazione NON DEVE cambiare. 0 &lt; valore ≤ 100 As il valore aumenta, le aree chiare dell'immagine DEVONO apparire più chiare. Intervallo di regolazione dell'ombra: -100 ≤ valore &lt; 0 Al diminuire del valore, le aree scure dell'immagine DEVONO apparire più scure. 0 Un valore di 0 specifica che il l'ombra NON DEVE cambiare. 0 &lt; valore ≤ 100 All'aumentare del valore, le aree scure dell'immagine DEVONO apparire più chiare. Intervallo di regolazione della saturazione del bianco: 0 — 255 All'aumentare del valore, il limite superiore della gamma del canale del colore intensità aumenta. Intervallo di regolazione della saturazione del nero: 0 — 255 All'aumentare del valore, aumenta il limite inferiore dell'intervallo delle intensità del canale colore.
