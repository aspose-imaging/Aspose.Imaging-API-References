## EmfPlusColorBalanceEffect.YellowBlue property

Ottiene o imposta un intero con segno a 32 bit che specifica una modifica nella quantità di blu in nell'immagine. Questo valore DEVE essere compreso tra -100 e 100, con i seguenti effetti: -100 ≤ valore &lt; 0 Man mano che il valore diminuisce, la quantità di blu nell'immagine DEVE diminuire e la quantità di giallo DEVE aumentare. 0 Un valore di 0 specifica che le quantità di blu e giallo NON DEVONO cambiare. 0 &lt; valore ≤ 100 All'aumentare del valore, la quantità di blu nell'immagine DOVREBBE aumentare e la quantità di di giallo DOVREBBE diminuire.
