## PsdOptions.RemoveGlobalTextEngineResource property

Ottiene o imposta un valore che indica se - Rimuovi la risorsa del motore di testo globale - Utilizzato per alcuni file PSD con livelli di testo, nel solo caso in cui non possono essere aperti in Adobe Photoshop dopo l'elaborazione (principalmente per i livelli di testo dei caratteri assenti correlati). Dopo aver utilizzato questa opzione, l'utente deve aprire il prossimo file di Photoshop: Menu "Testo" -&gt; "Elabora caratteri assenti". Dopo quell'operazione tutto il testo apparirà di nuovo. Tieni presente che questa operazione potrebbe causare alcune modifiche finali al layout.

```csharp
public bool RemoveGlobalTextEngineResource { get; set; }
```

`VERO` if [rimuovere la risorsa del motore di testo globale]; altrimenti,`falso` .
