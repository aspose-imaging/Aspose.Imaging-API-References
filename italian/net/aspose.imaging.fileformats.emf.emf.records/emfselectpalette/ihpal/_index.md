## EmfSelectPalette.IhPal property

Ottiene o imposta un intero senza segno a 32 bit che specifica l'indice di un oggetto LogPalette (sezione 2.2.17) nella tabella oggetti EMF o il valore DEFAULT_PALETTE, che è l'indice di una tavolozza oggetti stock dall'enumerazione StockObject (sezione 2.1.31).

```csharp
public int IhPal { get; set; }
```

### Osservazioni

Questo valore NON DEVE essere zero o l'indice di qualsiasi altro oggetto stock.
