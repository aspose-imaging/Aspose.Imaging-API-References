## EmfVertexData.VertexPadding property

Ottiene o imposta un array di lunghezza variabile opzionale di nTri per quattro byte che DEVE essere presente se il valore del campo ulMode indica oggetti GradientRectangle (sezione 2.2.7). Se il valore del campo ulMode indica oggetti GradientTriangle (sezione 2.2.8), non è presente alcun VertexPadding. Questo campo DEVE essere ignorato.
