## PdfCoreOptions.PdfCompliance property

Ottiene o imposta la conformità PDF.

```csharp
public PdfComplianceVersion PdfCompliance { get; set; }
```

### Valore della proprietà

La conformità PDF.

### Esempi

Converti l'immagine EPS in PDF utilizzando il rendering PostScript.

```csharp
[C#]

using (var image = (EpsImage)Image.Load("Sample.eps"))
{
    var options = new PdfOptions
    {
        PdfCoreOptions = new PdfCoreOptions
        {
            PdfCompliance = PdfComplianceVersion.PdfA1b // Imposta la conformità PDF richiesta
        }
    };
  
    image.Save("Sample.pdf", options);
}
```
