## StringFormat.DigitSubstitutionLanguage property

Ottiene o imposta la lingua utilizzata quando le cifre locali vengono sostituite con quelle occidentali.

```csharp
public int DigitSubstitutionLanguage { get; set; }
```

### Valore della proprietà

Un identificatore della lingua del supporto linguistico nazionale (NLS) che identifica la lingua che verrà utilizzata quando le cifre locali vengono sostituite con quelle occidentali. È possibile passare ilLCID proprietà di aCultureInfo oggetto come identificatore di lingua NLS. Ad esempio, supponiamo di creare unCultureInfo oggetto passando la stringa "ar-EG" ad aCultureInfo costruttore. Se si passa ilLCID proprietà di quelloCultureInfo oggetto insieme a. Traditional alStringDigitSubstitute) metodo, le cifre arabo-indiane verranno sostituite con le cifre occidentali al momento della visualizzazione.
