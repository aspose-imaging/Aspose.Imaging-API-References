## TransparencySupporter.Opacity property

Ottiene o imposta l'opacità dell'oggetto. Il valore deve essere compreso tra 0 e 1. Il valore 0 significa che l'oggetto è completamente visibile, il valore 1 significa che l'oggetto è completamente opaco.

public float Opacity { get; set; }
