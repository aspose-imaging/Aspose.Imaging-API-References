## EmfPointEnum enumeration

L'enumerazione Punto viene utilizzata per specificare come utilizzare un punto in una chiamata di disegno.

```csharp
[Flags]
public enum EmfPointEnum : byte
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| PT_CLOSEFIGURE | `1` | Un tipo PT_LINETO o PT_BEZIERTO può essere combinato con questo valore utilizzando l'operatore bit per bit OR per indicare che il punto corrispondente è l'ultimo punto in una figura e la figura è chiusa |
| PT_LINETO | `2` | Specifica che deve essere tracciata una linea dalla posizione corrente a questo punto, che poi diventa la nuova posizione corrente |
| PT_BEZIERTO | `4` | Specifica che questo punto è un punto di controllo o un punto finale per una curva di Bézier. |
| PT_MOVETO | `6` | Specifica che questo punto inizia una figura disgiunta. Questo punto diventa la nuova posizione corrente. |
