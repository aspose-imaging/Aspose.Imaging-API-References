## WmfLogColorSpace.Filename property

Se un nome file è specificato, e il[`ColorSpaceType`](../colorspacetype) il campo è impostato su LCS_CALIBRATED_RGB, gli altri campi di questa struttura DOVREBBE essere ignorati.
