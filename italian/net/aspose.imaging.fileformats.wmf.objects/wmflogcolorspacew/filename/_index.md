## WmfLogColorSpaceW.Filename property

Ottiene o imposta una stringa carattere Unicode UTF16-LE con terminazione null, che specifica il nome di un file che contiene un profilo color . Se viene specificato un nome file e il [`ColorSpaceType`](../colorspacetype) campo è impostato su LCS_CALIBRATED_RGB, il altri campi di questa struttura DOVREBBE essere ignorato.
