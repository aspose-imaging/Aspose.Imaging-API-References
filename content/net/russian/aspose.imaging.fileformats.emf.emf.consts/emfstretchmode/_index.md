## EmfStretchMode enumeration

Перечисление StretchMode используется для указания того, как данные о цвете добавляются или удаляются из растровых изображений, которые растягиваются или сжимаются.

```csharp
public enum EmfStretchMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| STRETCH_ANDSCANS | `1` | Выполняет логическую операцию И, используя значения цвета для устраненных и существующих пикселей. Если растровое изображение является монохромным растровым изображением, этот режим сохраняет черные пиксели за счет белых пикселей |
| STRETCH_ORSCANS | `2` | Выполняет логическую операцию ИЛИ, используя значения цвета для исключенных и существующих пикселей. Если растровое изображение является монохромным, этот режим сохраняет белые пиксели за счет черных. |
| STRETCH_DELETESCANS | `3` | Удаляет пиксели. Этот режим удаляет все удаленные строки пикселей, не пытаясь сохранить информацию о них. |
| STRETCH_HALFTONE | `4` | Сопоставляет пиксели из исходного прямоугольника с блоками пикселей в целевом прямоугольнике. Средний цвет целевого блока пикселей приблизительно соответствует цвету исходных пикселей. |
