## WmfBrushStyle enumeration

Перечисление BrushStyle определяет различные возможные типы кистей , которые можно использовать в графических операциях. Для получения дополнительной информации см. спецификацию объекта Brush (раздел 2.2.1.1).

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Solid | `0` | Кисть, рисующая одним постоянным цветом, сплошным или размытым. |
| Null | `1` | Кисть, которая ничего не делает. Использование кисти NULL в графической операции ДОЛЖНО иметь тот же эффект, что и использование кисти вообще без использования |
| Hatched | `2` | Кисть, рисующая предопределенный простой узор или "штрих" на сплошном фоне |
| Pattern | `3` | Кисть, рисующая узор, определяемый растровым изображением, которым МОЖЕТ быть объект Bitmap16 Object или объект DeviceIndependentBitmap (DIB). |
| Indexed | `4` | Не поддерживается. |
| Dibpattern | `5` | Узорная кисть, заданная DIB. |
