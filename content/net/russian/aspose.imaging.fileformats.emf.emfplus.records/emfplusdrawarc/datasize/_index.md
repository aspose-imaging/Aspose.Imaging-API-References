## EmfPlusDrawArc.DataSize property

Получает или задает размер данных. 32-разрядное целое число без знака, указывающее 32-разрядное выровненное число байтов данных, относящихся к записи, которые следуют за ним. Для этого типа записи значение ДОЛЖНО быть одним из следующих: : 0x00000010 Если бит C установлен в поле Flags. 0x00000018 Если бит C очищен в поле Flags.
