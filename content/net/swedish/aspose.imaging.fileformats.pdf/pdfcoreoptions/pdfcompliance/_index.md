## PdfCoreOptions.PdfCompliance property

Hämtar eller ställer in PDF-kompatibilitet.

```csharp
public PdfComplianceVersion PdfCompliance { get; set; }
```

### Fastighetsvärde

PDF-kompatibiliteten.

### Exempel

Konvertera EPS-bild till PDF med PostScript-rendering.

```csharp
[C#]

using (var image = (EpsImage)Image.Load("Sample.eps"))
{
    var options = new PdfOptions
    {
        PdfCoreOptions = new PdfCoreOptions
        {
            PdfCompliance = PdfComplianceVersion.PdfA1b // Ställ in obligatorisk PDF-kompatibilitet
        }
    };
  
    image.Save("Sample.pdf", options);
}
```
