## RasterImage.GetPixel method

Får en bildpixel.

```csharp
public Color GetPixel(int x, int y)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Int32 | Pixel x platsen. |
| y | Int32 | Pixel y-platsen. |

### Returvärde

Pixelfärgen för den angivna platsen.

### Exempel

Följande exempel laddar en rasterbild och erhåller färgen på en godtycklig pixel.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Få färgen på den övre vänstra pixeln i bilden.
    Color color = rasterImage.GetPixel(0, 0);

    // Hämta värdena för de individuella färgkomponenterna
    byte alpha = color.A;
    byte red = color.R;
    int green = color.G;
    int blue = color.B;

    System.Console.WriteLine("The color of the pixel(0,0) is A={0},R={1},G={2},B={3}", alpha, red, green, blue);
}
```
