## EmfSaveDc class

Sparar det aktuella tillståndet för uppspelningsenhetskontext på en hög med tillstånd som sparats av föregående EMR_SAVEDC -poster, om några. Tillståndet består av grafikegenskaper och objekt, inklusive den för närvarande valda bitmappen, pensel, palett, teckensnitt, penna och region. An EMR_RESTOREDC-post används för att återställa tillståndet. Denna EMF-post specificerar inga parametrar.

public sealed class EmfSaveDc : EmfStateRecordType
