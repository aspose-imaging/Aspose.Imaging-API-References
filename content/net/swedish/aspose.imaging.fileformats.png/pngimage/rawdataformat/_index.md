### Exempel

Följande exempel laddar PNG-bilder och skriver ut information om rådataformat och alfakanal.

```csharp
[C#]

// PNG-bilderna som ska laddas.
string[] fileNames = new string[]
{
    @"c:\temp\sample.png",
    @"c:\temp\alpha.png",
};

foreach (string fileName in fileNames)
{
    using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(fileName))
    {
        Aspose.Imaging.FileFormats.Png.PngImage pngImage = (Aspose.Imaging.FileFormats.Png.PngImage)image;
        System.Console.WriteLine("ImageFile={0}, FileFormat={1}, HasAlpha={2}", fileName, pngImage.RawDataFormat, pngImage.HasAlpha);
    }
}

// Utdata kan se ut så här:
// ImageFile=c:\temp\sample.png, FileFormat=Rgb24Bpp, använda kanaler: 8,8,8, HasAlpha=False
// ImageFile=c:\temp\alpha.png, FileFormat=RGBA32Bpp, använda kanaler: 8,8,8,8, HasAlpha=True
```
