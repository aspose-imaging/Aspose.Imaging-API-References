## EmfPlusStringFormatData.CharRange property

Ruft ein optionales Array von RangeCount EmfPlusCharacterRange -Objekten ab oder legt es fest, die den Bereich von Zeichenpositionen innerhalb einer Textzeichenfolge angeben. Der Begrenzungsbereich wird definiert durch den Bereich der Anzeige, der von einer Gruppe von Zeichen belegt wird, die durch den Zeichenbereich spezifiziert sind. Dieses Feld MUSS vorhanden sein, wenn der Wert des Felds RangeCount im EmfPlusStringFormat-Objekt größer als 0 ist.

public EmfPlusCharacterRange[] CharRange { get; set; }
