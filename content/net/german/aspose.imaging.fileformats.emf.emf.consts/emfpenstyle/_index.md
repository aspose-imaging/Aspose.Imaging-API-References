## EmfPenStyle enumeration

Die Aufzählung PenStyle definiert die Attribute von Stiften, die in Grafikoperationen verwendet werden können. Ein Stiftstil ist eine Kombination aus Stifttyp, Linienstil, Linienende und Linienverbindung.

```csharp
[Flags]
public enum EmfPenStyle
```

| PS_ENDCAP_ROUND | `0` | Ein Zeilenende, das runde Enden angibt. |
| PS_JOIN_ROUND | `0` | Eine Linienverbindung, die Rundungsverbindungen angibt |
