## EmfPointEnum enumeration

Die Point-Enumeration wird verwendet, um anzugeben, wie ein Punkt in einem Zeichnungsaufruf verwendet werden soll.

```csharp
[Flags]
public enum EmfPointEnum : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| PT_CLOSEFIGURE | `1` | Ein PT_LINETO- oder PT_BEZIERTO-Typ kann mit diesem Wert kombiniert werden, indem der bitweise -Operator ODER verwendet wird, um anzugeben, dass der entsprechende Punkt der letzte Punkt in einer Figur ist und die Figur geschlossen ist |
| PT_LINETO | `2` | Gibt an, dass eine Linie von der aktuellen Position zu diesem Punkt gezeichnet werden soll, der dann zur neuen aktuellen Position wird |
| PT_BEZIERTO | `4` | Gibt an, dass dieser Punkt ein Kontrollpunkt oder Endpunkt für eine Bezier-Kurve ist. |
| PT_MOVETO | `6` | Gibt an, dass dieser Punkt eine disjunkte Figur beginnt. Dieser Punkt wird zur neuen aktuellen Position. |
