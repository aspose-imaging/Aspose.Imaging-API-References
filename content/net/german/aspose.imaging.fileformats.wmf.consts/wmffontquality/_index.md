## WmfFontQuality enumeration

Die FontQuality-Enumeration gibt an, wie genau die Attribute der logischen Schriftart mit denen der physischen Schriftart übereinstimmen sollten, wenn Text wiedergegeben wird.

```csharp
public enum WmfFontQuality : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | `0` | Gibt an, dass die Zeichenqualität der Schriftart keine Rolle spielt, sodass ENTWURF verwendet werden kann. |
| Draft | `1` | Gibt an, dass die Zeichenqualität der Schriftart weniger wichtig ist als der Abgleich logischer Attribute. Für gerasterte Schriftarten SOLLTE die Skalierung aktiviert werden, was bedeutet, dass mehr Schriftgrößen verfügbar sind. |
| Proof | `2` | Gibt an, dass die Zeichenqualität der Schriftart wichtiger ist als die Übereinstimmung logischer Attribute. Für gerasterte Schriftarten SOLLTE die Skalierung deaktiviert und die Schriftart gewählt werden, die der Größe am nächsten kommt. |
| Nonantialiased | `3` | Gibt an, dass Anti-Aliasing NICHT verwendet werden SOLLTE, wenn text wiedergibt |
| Antialiased | `4` | Gibt an, dass beim Rendern von Text Anti-Aliasing verwendet werden SOLLTE, wenn die Schriftart dies unterstützt. |
| Cleartype | `5` | Gibt an, dass ClearType-Anti-Aliasing verwendet werden SOLLTE, wenn Text wiedergegeben wird, sofern die Schriftart dies unterstützt. |
