## EmfLogFontEx.Script property

Ruft eine Zeichenfolge von 32 Unicode-Zeichen ab oder legt diese fest, die den Zeichensatz der Schriftart definiert. Wenn die Länge dieser Zeichenfolge weniger als 32 Zeichen beträgt, MUSS eine abschließende NULL vorhanden sein, danach MUSS der Rest dieses Felds ignoriert werden.
