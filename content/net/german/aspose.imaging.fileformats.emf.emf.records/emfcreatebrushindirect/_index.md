## EmfCreateBrushIndirect class

Der Datensatz EMR_CREATEBRUSHINDIRECT definiert einen logischen Pinsel für Grafikoperationen.

Das durch diesen Datensatz definierte logische Pinselobjekt kann im Kontext des Wiedergabegeräts durch einen EMR_SELECTOBJECT-Datensatz (Abschnitt 2.3.8.5) ausgewählt werden, der den logischen Pinsel angibt, der in nachfolgenden Grafikoperationen verwendet werden soll.

public sealed class EmfCreateBrushIndirect : EmfObjectCreationRecordType
