## EmfPlusHeader.VideoDisplay property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob ein Video angezeigt wird. Wenn es gesetzt ist, zeigt dieses Flag an, dass die Metadatei mit einem Referenzgerät -Kontext für eine Videoanzeige aufgezeichnet wurde. Wenn klar, wurde die Metadatei mit einem Referenzgerät -Kontext für einen Drucker aufgezeichnet.

```csharp
public bool VideoDisplay { get; set; }
```

### Eigentumswert

`Stimmt` wenn [Videoanzeige]; Andernfalls,`FALSCH` .
