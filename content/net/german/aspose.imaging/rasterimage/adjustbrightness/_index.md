## RasterImage.AdjustBrightness method

Anpassen einer Helligkeit für das Bild.

```csharp
public virtual void AdjustBrightness(int brightness)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| brightness | Int32 | Helligkeitswert. |

### Beispiele

Das folgende Beispiel führt eine Helligkeitskorrektur eines Bildes durch.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Stellen Sie den Helligkeitswert ein. Die akzeptierten Helligkeitswerte liegen im Bereich [-255, 255].
    rasterImage.AdjustBrightness(50);
    rasterImage.Save(dir + "sample.AdjustBrightness.png");
}
```
