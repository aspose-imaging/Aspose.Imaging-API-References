## RasterImage.SetPixel method

Setzt ein Bildpixel für die angegebene Position.

```csharp
public void SetPixel(int x, int y, Color color)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | Int32 | Die Pixel-x-Position. |
| y | Int32 | Die Pixel-y-Position. |
| color | Color | Die Pixelfarbe für die angegebene Position. |

### Beispiele

Das folgende Beispiel lädt ein Rasterbild und legt die Farbe eines beliebigen Pixels fest.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Legt die Farbe des oberen linken Pixels fest.
    rasterImage.SetArgb32Pixel(0, 0, Aspose.Imaging.Color.Aqua.ToArgb());

    // Eine andere Möglichkeit besteht darin, eine Instanz von Aspose.Imaging.Color direkt zu übergeben
    rasterImage.SetPixel(0, 0, Aspose.Imaging.Color.Aqua);
}
```
