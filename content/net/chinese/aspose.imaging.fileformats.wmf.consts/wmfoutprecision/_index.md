## WmfOutPrecision enumeration

OutPrecision 枚举定义输出精度的值，这是 字体映射器匹配特定字体参数的要求，包括高度、宽度、字符方向、 转义、间距和字体类型。

```csharp
public enum WmfOutPrecision : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Default | `0` | 一个指定默认行为的值。 |
| String | `1` | 枚举光栅化字体时返回的值。 |
| Stroke | `3` | 枚举TrueType等轮廓字体， 矢量字体时返回的值。 |
| Tt | `4` | 当 system 包含多个同名字体时，指定 TrueType 字体选择的值。 |
| Device | `5` | 当system 包含多个同名字体时，指定设备字体选择的值。 |
| Raster | `6` | 当系统 包含多个同名字体时，指定光栅化字体选择的值。 |
| TtOnly | `7` | 指定仅 TrueType 字体要求的值。如果 系统中没有安装TrueType字体，则指定默认行为。 |
| Outline | `8` | 指定 TrueType 和其他轮廓字体要求的值。 |
| ScreenOutline | `9` | 指定 TrueType 和其他 轮廓字体的首选项的值。 |
| PsOnly | `10` | 指定仅 PostScript 字体要求的值。如果系统中没有安装 PostScript 字体，则指定默认行为。 |
