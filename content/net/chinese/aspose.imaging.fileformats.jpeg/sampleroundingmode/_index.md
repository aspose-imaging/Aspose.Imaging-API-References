## SampleRoundingMode enumeration

定义将 n 位值转换为 8 位值的方式。

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Extrapolate | `0` | 外推一个 8 位值以适合 n 位，其中 1 &lt; n &lt; 8. 所有可能的 8 位值的数量为 1 &lt;&lt; 8 = 256，从 0 到 255。 所有可能的数量n 位值是 1 &lt;&lt; n，从 0 到 (1 &lt;&lt; n) - 1. 对应于某个 8 位值 V8 的最合理的 n 位值 Vn 等于 Vn = V8 &gt;&gt; (8 - n)。 |
| Truncate | `1` | 截断一个 8 位值以适合 n 位，其中 1 &lt; n &lt; 8. 所有可能的 n 位值的数量为 1 &lt;&lt; n，从 0 到 (1 &lt;&lt; n) - 1. 与某个 8 位值 V8 对应的最合理的 n 位值 Vn 等于 Vn = V8 &amp; ((1 &lt;&lt; n) - 1). |
