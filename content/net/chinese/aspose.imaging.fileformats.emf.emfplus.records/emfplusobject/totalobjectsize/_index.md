## EmfPlusObject.TotalObjectSize property

获取或设置对象的总大小。 如果记录是可连续的，则当设置了继续位时，将出现此字段 。连续对象有多个以 EmfPlusContieudObjectRecord 开头的 EMF+ 记录。每个 EmfPlusContinuedObjectRecord 将包含 a TotalObjectSize。一旦读取了 TotalObjectSize 字节数，下一个 EMF+ 记录将不会被视为连续对象的一部分。

```csharp
public int TotalObjectSize { get; set; }
```

### 适当的价值

对象的总大小。
