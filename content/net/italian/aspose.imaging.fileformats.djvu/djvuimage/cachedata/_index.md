## DjvuImage.CacheData method

Memorizza nella cache i dati privati.

```csharp
public override void CacheData()
```

### Esempi

L'esempio seguente mostra come memorizzare nella cache tutte le pagine di un'immagine DJVU.

```csharp
[C#]

string dir = "c:\\temp\\";

// Carica un'immagine da un file DJVU.
using (Aspose.Imaging.FileFormats.Djvu.DjvuImage image = (Aspose.Imaging.FileFormats.Djvu.DjvuImage)Aspose.Imaging.Image.Load(dir + "sample.djvu"))
{
    // Questa chiamata memorizza nella cache tutte le pagine in modo che non venga eseguito alcun caricamento di dati aggiuntivo dal flusso di dati sottostante.
    image.CacheData();

    // Oppure puoi memorizzare nella cache le pagine singolarmente.
    foreach (Aspose.Imaging.FileFormats.Djvu.DjvuPage page in image.Pages)
    {
        page.CacheData();
    }
}
```
