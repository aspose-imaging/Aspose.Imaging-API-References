## DicomImage.CacheData method

Memorizza nella cache i dati privati.

```csharp
public override void CacheData()
```

### Esempi

L'esempio seguente mostra come memorizzare nella cache tutte le pagine di un'immagine DICOM.

```csharp
[C#]

string dir = "c:\\temp\\";

// Carica un'immagine da un file DICOM.
using (Aspose.Imaging.FileFormats.Dicom.DicomImage image = (Aspose.Imaging.FileFormats.Dicom.DicomImage)Aspose.Imaging.Image.Load(dir + "sample.dicom"))
{
    // Questa chiamata memorizza nella cache tutte le pagine in modo che non venga eseguito alcun caricamento di dati aggiuntivo dal flusso di dati sottostante.
    image.CacheData();

    // Oppure puoi memorizzare nella cache le pagine singolarmente.
    foreach (Aspose.Imaging.FileFormats.Dicom.DicomPage page in image.DicomPages)
    {
        page.CacheData();
    }
}
```
