## EmfPlusCurveAdjustments enumeration

L'enumerazione CurveAdjustments definisce le regolazioni che possono essere applicate alla curva colore di un'immagine.

```csharp
public enum EmfPlusCurveAdjustments
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| AdjustExposure | `0` | Specifica la simulazione per aumentare o diminuire l'esposizione di un'immagine. |
| AdjustDensity | `1` | Specifica la simulazione per aumentare o diminuire la densità di un'immagine. |
| AdjustContrast | `2` | Specifica un aumento o una diminuzione del contrasto di un'immagine. |
| AdjustHighlight | `3` | Specifica un aumento o una diminuzione del valore di un canale colore di un'immagine, se quel canale ha già un valore superiore alla metà dell'intensità. Questa regolazione può essere utilizzata per aumentare la definizione nelle aree chiare di un'immagine senza influire sulle aree scure. |
| AdjustShadow | `4` | Specifica un aumento o una diminuzione del valore di un canale colore di un'immagine, se quel canale ha già un valore inferiore alla metà dell'intensità. Questa regolazione può essere utilizzata per aumentare la definizione nelle aree scure di un'immagine senza influire sulle aree chiare. |
| AdjustMidtone | `5` | Specifica una regolazione che schiarisce o scurisce un'immagine. I valori del canale colore al centro dell'intervallo di intensità vengono alterati più dei valori del canale colore vicino agli estremi minimo o massimo di intensità. Questa regolazione può essere utilizzata per schiarire o scurire un'immagine senza perdere il contrasto tra le parti più scure e più chiare dell'immagine. |
| AdjustWhiteSaturation | `6` | Specifica una regolazione della saturazione del bianco di un'immagine, definita come il valore massimo nell'intervallo di intensità per un determinato canale di colore, il cui intervallo è generalmente compreso tra 0 e 255. |
| AdjustBlackSaturation | `7` | Specifica una regolazione della saturazione del nero di un'immagine, che è il valore minimo nell'intervallo di intensità per un determinato canale di colore, che in genere va da 0 a 255. |
