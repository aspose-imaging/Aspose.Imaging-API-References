## Image.CanSave method

Determina se l'immagine può essere salvata nel formato file specificato rappresentato dalle opzioni di salvataggio passate.

```csharp
public bool CanSave(ImageOptionsBase options)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| options | ImageOptionsBase | Le opzioni di salvataggio da utilizzare. |

### Valore di ritorno

`VERO` se l'immagine può essere salvata nel formato file specificato rappresentato dalle opzioni di salvataggio passate; altrimenti,`falso` .

### Esempi

Questo esempio mostra come determinare se l'immagine può essere salvata nel formato file specificato rappresentato dalle opzioni di salvataggio passate.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.ImageOptions.JpegOptions saveOptions = new Aspose.Imaging.ImageOptions.JpegOptions();
    saveOptions.Quality = 50;

    // Determina se l'immagine può essere salvata su JPEG
    bool canSave = image.CanSave(saveOptions);
}
```
