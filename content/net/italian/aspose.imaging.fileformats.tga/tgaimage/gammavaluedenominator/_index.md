## TgaImage.GammaValueDenominator property

Ottiene la parte Gamma Value Denominator. Un'immagine non corretta (un'immagine senza gamma) dovrebbe avere il valore 1.0 come risultato.

```csharp
public ushort GammaValueDenominator { get; }
```
