## EmfPlusBitmap.Height property

Ottiene o imposta bitmap height Height (4 byte): un intero con segno a 32 bit che specifica l'altezza in pixel dell'area occupata dalla bitmap. Se l'immagine è compressa, in base al campo Tipo, questo valore non è definito e DEVE essere ignorato.

```csharp
public int Height { get; set; }
```

### Valore della proprietà
