## EmfPlusColorCurveEffect.AdjustmentIntensity property

Yoğunluk ayar aralığı: -255 ≤ değer &lt; 0 Değer azaldıkça, görüntünün yoğunluğu AZALMALIDIR , sonuç olarak daha koyu bir görüntü verir. 0 0 değeri, yoğunluğun DEĞİŞMEMESİ gerektiğini belirtir. 0 &lt; değer ≤ 255 Değer arttıkça görüntünün yoğunluğu da ARTIRILMALIDIR. Kontrast ayar aralığı: -100 ≤ değer &lt; 0 Değer azaldıkça, görüntünün kontrastı DÜŞÜRMELİDİR. 0 0 değeri, kontrastın DEĞİŞMEYECEĞİNİ belirtir. 0 &lt; değer ≤ 100 Değer arttıkça görüntünün kontrastı da ARTIRILMALIDIR. e. Gölge ayar aralığı: -100 ≤ değer &lt; 0 Değer azaldıkça görüntünün karanlık alanları daha koyu görünmelidir. 0 0 değeri, gölge DEĞİŞMEMELİDİR. 0 &lt; değer ≤ 100 Değer arttıkça, görüntünün karanlık alanları daha açık görünmelidir.
