## EmfColorSpace enumeration

L'énumération ColorSpace est utilisée pour spécifier quand activer et désactiver l'épreuvage couleur et quand supprimer les transformations.

```csharp
public enum EmfColorSpace
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| CS_ENABLE | `1` | Mappe les couleurs sur la gamme de couleurs de l'appareil cible. Cela permet l'épreuvage couleur. Toutes les commandes de dessin ultérieures dans le contexte de l'appareil de lecture restitueront les couleurs telles qu'elles apparaîtraient sur l'appareil cible. |
| CS_DISABLE | `2` | Désactive l'épreuvage couleur. |
| CS_DELETE_TRANSFORM | `3` | Si la gestion des couleurs est activée pour le profil cible, la désactive et supprime la transformation concaténée. |
