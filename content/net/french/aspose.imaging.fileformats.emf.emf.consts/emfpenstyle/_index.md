## EmfPenStyle enumeration

L'énumération PenStyle définit les attributs des stylos qui peuvent être utilisés dans les opérations graphiques. Un style de stylo est une combinaison de type de stylo, de style de ligne, d'extrémité de ligne et de jointure de ligne.

```csharp
[Flags]
public enum EmfPenStyle
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| PS_COSMETIC | `0` | Un type de stylo qui spécifie une ligne avec une largeur d'une unité logique et un style qui est une couleur unie |
| PS_ENDCAP_ROUND | `0` | Un bouchon de ligne qui spécifie les extrémités rondes. |
| PS_JOIN_ROUND | `0` | Une jointure de ligne qui spécifie des jointures rondes |
| PS_SOLID | `0` | Un style de trait de couleur unie |
