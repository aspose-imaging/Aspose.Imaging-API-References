## WmfSetMapperFlags.MapperValues property

Obtient ou définit les valeurs du mappeur.

```csharp
public int MapperValues { get; set; }
```

### Valeur de la propriété

Le mappeur de polices tente de faire correspondre un rapport d'aspect de police au rapport d'aspect actuel de l'appareil. Si le bit zéro est défini, le mappeur sélectionne uniquement les polices correspondantes.
