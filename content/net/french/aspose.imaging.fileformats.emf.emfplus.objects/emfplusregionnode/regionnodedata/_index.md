## EmfPlusRegionNode.RegionNodeData property

Obtient ou définit des données facultatives de longueur variable qui définissent l'objet de données de région node spécifié dans le champ Type. Le contenu et le format des données peuvent être différents pour chaque type de nœud region . Ce champ NE DOIT PAS être présent si le type node est RegionNodeDataTypeEmpty ou RegionNodeDataTypeInfinite Cet objet est générique et est utilisé pour spécifier différents types de données de nœud de région, y compris : Un objet EmfPlusRegionNodePath (section 2.2.2.42), pour un nœud terminal ; Un Objet EmfPlusRectF (section 2.2.2.39), pour un nœud terminal ; and Un objet EmfPlusRegionNodeChildNodes (section 2.2.2.41), pour un nœud non terminal.
