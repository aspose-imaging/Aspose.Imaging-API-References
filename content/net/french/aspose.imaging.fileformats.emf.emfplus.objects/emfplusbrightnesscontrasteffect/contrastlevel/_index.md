## EmfPlusBrightnessContrastEffect.ContrastLevel property

Obtient ou définit un entier signé 32 bits qui spécifie le niveau de contraste. Cette valeur DOIT être comprise entre -100 et 100, avec les effets suivants : -100 ≤ valeur &lt; 0 Lorsque la valeur diminue, le contraste de l'image DEVRAIT diminuer. 0 Une valeur de 0 indique que le contraste NE DOIT PAS changer . 0 &lt; valeur ≤ 100 Lorsque la valeur augmente, le contraste de l'image DEVRAIT augmenter.
