## EmfPlusBitmap.Stride property

Obtient ou définit la foulée de l'image Stride (4 octets) : un entier signé de 32 bits qui spécifie le décalage d'octet entre le début d'une ligne de balayage et la suivante. Cette valeur est le nombre d'octets par pixel, qui est spécifié dans le champ PixelFormat, multiplié par la largeur en pixels, qui est spécifiée dans le champ Width. La valeur de ce champ DOIT être un multiple de quatre. Si l'image est compressée, selon le champ Type, cette valeur est indéfinie et DOIT être ignorée.

```csharp
public int Stride { get; set; }
```

### Valeur de la propriété

La foulée.
