## EmfLogBrushEx.BrushStyle property

Obtient ou définit un entier non signé 32 bits qui spécifie le style de pinceau. La valeur DOIT être une énumération de l'énumération WMF BrushStyle ([MS-WMF] section 2.1.1.4). Les valeurs de style prises en charge dans cette structure sont répertoriées plus loin dans cette section. Le style BS_NULL DEVRAIT être utilisé pour spécifier un pinceau qui n'a aucun effet.
