## EmfLogFont.CharSet property

Obtient ou définit un entier non signé 8 bits qui spécifie le jeu de glyphes de caractères. Il DOIT être une valeur dans l'énumération WMF CharacterSet ([MS-WMF] section 2.1.1.5). Si le jeu de caractères est inconnu, le traitement du métafichier NE DOIT PAS tenter de traduire ou d'interpréter les chaînes rendues avec cette police.
