## EmfPlusDrawDriverString.IsColor property

Obtient ou définit une valeur indiquant si cette instance est de couleur. Ce bit indique le type de données dans le champ BrushId. S'il est défini, BrushId spécifie la valeur de couleur dans un objet EmfPlusARGB (section 2.2.2.1). Si clair, BrushId contient l'index EMF+ Object Table d'un objet EmfPlusBrush (section 2.2.1.1).
