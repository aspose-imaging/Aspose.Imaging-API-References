## EmfPlusDrawDriverString.Glyphs property

Obtient ou définit les glyphes array Un tableau de valeurs 16 bits qui définissent la chaîne de texte à dessiner. Si l'indicateur DriverStringOptionsCmapLookup dans le champ DriverStringOptionsFlags est défini, chaque valeur dans this array spécifie un caractère Unicode. Sinon, chaque valeur spécifie un index vers un glyphe de caractère a dans l'objet EmfPlusFont spécifié par la valeur ObjectId dans le champ Flags.
