## PdfCoreOptions.PdfCompliance property

Obtient ou définit la conformité PDF.

```csharp
public PdfComplianceVersion PdfCompliance { get; set; }
```

### Valeur de la propriété

La conformité PDF.

### Exemples

Convertissez une image EPS en PDF à l'aide du rendu PostScript.

```csharp
[C#]

using (var image = (EpsImage)Image.Load("Sample.eps"))
{
    var options = new PdfOptions
    {
        PdfCoreOptions = new PdfCoreOptions
        {
            PdfCompliance = PdfComplianceVersion.PdfA1b // Définir la conformité PDF requise
        }
    };
  
    image.Save("Sample.pdf", options);
}
```
