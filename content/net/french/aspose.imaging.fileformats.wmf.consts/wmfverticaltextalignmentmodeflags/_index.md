## WmfVerticalTextAlignmentModeFlags enumeration

VerticalTextAlignmentMode Les indicateurs spécifient la relation entre un point de référence et un rectangle de délimitation, pour l'alignement du texte. Ces drapeaux peuvent être combinés pour spécifier plusieurs options, avec la restriction qu'un seul drapeau peut être choisi qui modifie la position du dessin dans le contexte de périphérique de lecture . L'alignement vertical du texte est effectué lorsque la police a une ligne de base verticale par défaut, telle que Kanji .

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| VTA_TOP | `0` | Le point de référence DOIT être sur le bord supérieur du rectangle englobant. |
| VTA_RIGHT | `0` | Le point de référence DOIT être sur le bord droit du rectangle englobant. |
| VTA_BOTTOM | `2` | Le point de référence DOIT être sur le bord inférieur du rectangle englobant. |
| VTA_CENTER | `6` | Le point de référence DOIT être aligné verticalement avec le centre du rectangle englobant. |
| VTA_LEFT | `8` | Le point de référence DOIT être sur le bord gauche du rectangle englobant. |
| VTA_BASELINE | `24` | Le point de référence DOIT être sur la ligne de base du texte. |
