## StretchMode enumeration

Le[`StretchMode`](../stretchmode)L'énumération spécifie le mode d'étirement bitmap , qui définit la manière dont le système combine les lignes ou les colonnes d'un bitmap avec les pixels existants.

```csharp
public enum StretchMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| BlackOnWhite | `1` | Effectue une opération booléenne ET en utilisant les valeurs de couleur pour les pixels éliminés et existants . Si le bitmap est un bitmap monochrome , ce mode préserve les pixels noirs au détriment des blancs pixels |
| WhiteOnBlack | `2` | Effectue une opération booléenne OU en utilisant les valeurs de couleur pour les pixels éliminés et existants . Si le bitmap est un bitmap monochrome , ce mode préserve les pixels blancs au détriment des noirs pixels |
| ColorOnColor | `3` | Supprime les pixels. Ce mode supprime toutes les lignes éliminées de pixels sans essayer de conserver leurs informations. |
| HalfTone | `4` | Mappe les pixels du rectangle source en blocs de pixels dans le rectangle de destination . La couleur moyenne sur le bloc de destination de pixels se rapproche de la couleur des pixels source. |
