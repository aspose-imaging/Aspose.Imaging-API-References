## WmfBrushStyle enumeration

L'énumération BrushStyle spécifie les différents types de pinceaux possibles qui peuvent être utilisés dans les opérations graphiques. Pour plus d'informations, consultez la_spécification de l'objet Brush (section 2.2.1.1).

```csharp
public enum WmfBrushStyle
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Solid | `0` | Un pinceau qui peint une seule couleur constante, solide ou tramée. |
| Null | `1` | Une brosse qui ne fait rien. L'utilisation d'un pinceau NULL dans une opération graphique DOIT avoir le même effet que l'utilisation d'aucun pinceau du tout |
| Hatched | `2` | Un pinceau qui peint un motif simple prédéfini, ou "hachure", sur un fond uni |
| Pattern | `3` | Un pinceau qui peint un motif défini par un bitmap, qui PEUT être un objet Bitmap16 ou un objet DeviceIndependentBitmap (DIB). |
| Indexed | `4` | Non pris en charge. |
| Dibpattern | `5` | Un pinceau à motif spécifié par un DIB. |
| Dibpatternpt | `6` | Un pinceau à motif spécifié par un DIB. |
| Pattern8X8 | `7` | Non pris en charge. |
| Dibpattern8X8 | `8` | Non pris en charge. |
| Monopattern | `9` | Non pris en charge. |
| Gradient | `10` | Le dégradé Non pris en charge dans wmf, pris en charge dans odg. |
