## DxfOptions.TextAsLines property

Indique si le texte doit être exporté sous forme de contours constitués de polylignes (par défaut) ou sous forme d'entités TEXTE Autocad modifiables. Si cette option est définie

```csharp
public bool TextAsLines { get; set; }
```

### Valeur de la propriété

`vrai` pour exporter le texte sous forme de contours (par défaut), sinon,`faux` pour exporter du texte en tant qu'entités TEXT.
