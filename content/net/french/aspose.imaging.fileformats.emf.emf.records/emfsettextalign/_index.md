### Remarques

Les enregistrements EMR_SMALLTEXTOUT, EMR_EXTTEXTOUTA et EMR_EXTTEXTOUTW utilisent des valeurs d'alignement text pour positionner une chaîne de texte sur le support de sortie. Les valeurs spécifient la relation entre un point de référence et un rectangle qui délimite le texte. Le point de référence est soit la position actuelle, soit un point transmis à un enregistrement de sortie de texte. Le rectangle qui délimite le texte est formé par les cellules de caractère dans la chaîne de texte.
