## EmfCreateBrushIndirect class

L'enregistrement EMR_CREATEBRUSHINDIRECT définit un pinceau logique pour les opérations graphiques.

L'objet pinceau logique défini par cet enregistrement peut être sélectionné dans le contexte du périphérique de lecture par un enregistrement EMR_SELECTOBJECT (section 2.3.8.5), qui spécifie le pinceau logique à utiliser dans les opérations graphiques suivantes.

public sealed class EmfCreateBrushIndirect : EmfObjectCreationRecordType
