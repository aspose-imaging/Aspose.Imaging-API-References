## EmfRop4.BackgroundRop3 property

Obtient l'arrière-plan ROP3. Les 8 bits non signés les plus significatifs d'une valeur d'opération raster ternaire 24 bits à partir de l'énumération WMF Ternary Raster Operation ([MS-WMF] section 2.1.1.31). Ce code définit comment combiner les données de couleur d'arrière-plan de les bitmaps source et de destination et le motif de pinceau.

```csharp
public byte BackgroundRop3 { get; }
```

### Valeur de la propriété

Le fond ROP3.
