## EmfSetTextAlign.TextAlignmentMode property

Obtiene o establece un entero sin signo de 32 bits que especifica la alineación del texto por mediante una máscara de marcas de alineación del texto. Estos son[`WmfTextAlignmentModeFlags`](../../../aspose.imaging.fileformats.wmf.consts/wmftextalignmentmodeflags) ([MS-WMF] sección 2.1.2.3) para texto con una línea base horizontal, o[`WmfVerticalTextAlignmentModeFlags`](../../../aspose.imaging.fileformats.wmf.consts/wmfverticaltextalignmentmodeflags) ([MS-WMF] sección 2.1.2.4) para texto con una línea de base vertical . Solo se puede elegir un valor de los que afectan a la alineación horizontal y vertical .

public int TextAlignmentMode { get; set; }
