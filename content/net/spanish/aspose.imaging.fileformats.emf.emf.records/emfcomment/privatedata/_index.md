## EmfComment.PrivateData property

Obtiene o establece una matriz opcional de bytes que especifica los datos privados. El primer DWORD de estos datos NO DEBE ser uno de los valores de identificador de comentarios predefinidos especificados en la sección 2.3.3. Los datos privados son desconocidos para EMF; solo es significativo para las aplicaciones que conocen el formato de los datos y cómo usarlos. EMR_COMMENT los registros de datos privados PUEDEN ignorarse.
