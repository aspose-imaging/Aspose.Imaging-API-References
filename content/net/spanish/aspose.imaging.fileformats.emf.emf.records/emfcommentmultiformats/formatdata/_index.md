## EmfCommentMultiFormats.FormatData property

Obtiene o establece una matriz de longitud variable de bytes de datos de imagen para todos los formatos gráficos contenidos en este registro. El tamaño de los datos de cada imagen lo proporciona el campo DataSize en el objeto EmrFormat correspondiente. Por lo tanto, el tamaño total de este campo es la suma de los valores de DataSize en todos los objetos EmrFormat. El formato gráfico de los datos de cada imagen se especifica en el campo Firma del objeto correspondiente EmrFormat.

public byte[][] FormatData { get; set; }
