## EmfSelectPalette.IhPal property

Obtiene o establece un entero sin signo de 32 bits que especifica el índice de un objeto LogPalette (sección 2.2.17) en la tabla de objetos EMF o el valor DEFAULT_PALETTE, que es el índice de una paleta de objetos estándar de la enumeración StockObject (sección 2.1.31).

```csharp
public int IhPal { get; set; }
```

### Observaciones

Este valor NO DEBE ser cero ni el índice de ningún otro objeto de stock.
