## OdGradientStyle enumeration

El estilo degradado

```csharp
public enum OdGradientStyle
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Axial | `0` | El axial define un degradado bilineal que también se conoce como degradado reflejado o degradado lineal reflejado. Se crea como un degradado lineal que se refleja (o refleja) a lo largo de su eje. |
| Ellipsoid | `1` | El elipsoide define un degradado donde los colores se mezclan a lo largo del radio desde el centro de un elipsoide como se define con los atributos draw:cx y draw:cy. La longitud del semieje mayor es el ancho del área rellena y la longitud del semi-menor |
| Linear | `2` | linear define un degradado donde los colores se mezclan a lo largo del eje lineal del degradado. El eje del degradado se especifica con el atributo draw:angle en el sentido de las agujas del reloj hasta el eje vertical. |
| Radial | `3` | El radial define un degradado en el que los colores se mezclan a lo largo del radio desde el centro de un círculo definido con los atributos draw:cx y draw:cy. El exterior del círculo se rellena con el color final. |
| Rectangle | `4` | El rectángulo define un degradado que produce una mezcla rectangular desde el centro del rectángulo hasta el más corto de los 4 bordes. El centro del rectángulo se define con los atributos draw:cx y draw:cy. El ancho del rectángulo es el ancho del área rellena, el alto del rectángulo es el alto del área rellenada. El exterior de el cuadrado se rellena con el color final. |
| Square | `5` | El cuadrado define un degradado que produce una mezcla de cuadrados, imitando la perspectiva visual en un corredor o la vista aérea de una pirámide. También conocido como "gradiente de caja" y "gradiente piramidal". El centro del cuadrado se define con los atributos draw:cx y draw:cy. El ancho y alto del cuadrado es el valor mínimo del ancho o el alto del área rellenada. El exterior del cuadrado se rellena con el color final. |
| None | `6` | El estilo de degradado es none |
