## WmfPostScriptClipping enumeration

La enumeración de recorte de PostScript define funciones que se pueden aplicar al trazado de recorte utilizado para la salida de PostScript.

```csharp
public enum WmfPostScriptClipping
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| CLIP_SAVE | `0` | Guarda el trazado de recorte PostScript actual. |
| CLIP_RESTORE | `1` | Restaura el trazado de recorte de PostScript al último trazado de recorte que fue guardado por una función CLIP_SAVE anterior aplicada por un registro CLIP_TO_PATH (sección 2.3.6.6). |
| CLIP_INCLUSIVE | `2` | Intersecta el trayecto de recorte PostScript actual con el trayecto de recorte actual y guarda el resultado como el nuevo trayecto de recorte PostScript. |
