## EmfExtTextOutOptions enumeration

La enumeración ExtTextOutOptions especifica parámetros que controlan varios aspectos de la salida de texto por registros EMR_SMALLTEXTOUT(sección 2.3.5.37) y en objetos EmrText.

```csharp
[Flags]
public enum EmfExtTextOutOptions
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| ETO_OPAQUE | `2` | Este bit indica que el color de fondo actual DEBE utilizarse para rellenar el rectángulo |
| ETO_CLIPPED | `4` | Este bit indica que el texto DEBE recortarse al rectángulo. |
| ETO_GLYPH_INDEX | `10` | Este bit indica que los códigos de caracteres en una cadena de texto de salida son en realidad índices de los glifos de caracteres en una fuente TrueType. Los índices de glifos son específicos de la fuente, , por lo que para mostrar los caracteres correctos en la reproducción, la fuente que se utiliza DEBE ser idéntica a la fuente utilizada para generar los índices. |
| ETO_RTLREADING | `80` | Este bit indica que el texto DEBE estar dispuesto en orden de lectura de derecha a izquierda, en lugar del orden predeterminado de izquierda a derecha. Esto DEBE aplicarse solo cuando la fuente seleccionada en el contexto del dispositivo de reproducción es hebrea o árabe |
| ETO_NO_RECT | `100` | Este bit indica que el registro no especifica un rectángulo delimitador para la salida de texto. |
| ETO_SMALL_CHARS | `200` | Este bit indica que los códigos de caracteres en una cadena de texto de salida son 8 bits, derivados de los bytes bajos de los códigos de caracteres Unicode UTF16-LE de 16 bits, en los que se supone que el byte alto es 0. |
| ETO_NUMERICSLOCAL | `400` | Este bit indica que para mostrar números, DEBERÍAN usarse dígitos apropiados para la configuración regional |
| ETO_NUMERICSLATIN | `800` | Este bit indica que para mostrar números, DEBERÍAN utilizarse dígitos europeos |
| ETO_IGNORELANGUAGE | `1000` | Este bit indica que no se debe realizar ningún procesamiento especial del sistema operativo para la colocación de glifos en cadenas de derecha a izquierda; es decir, todo el posicionamiento de los glifos DEBE ser atendido por los registros de dibujo y estado en el metarchivo |
| ETO_PDY | `2000` | Este bit indica que se DEBEN proporcionar valores de desplazamiento de caracteres tanto horizontal como vertical |
| ETO_REVERSE_INDEX_MAP | `10000` | Este bit está reservado y NO DEBE usarse |
