## EmfPlusSmoothingMode enumeration

La enumeración SmoothingMode define modos de suavizado para aplicar a líneas, curvas y los bordes de las áreas rellenas para que parezcan más continuos o definidos con nitidez.

```csharp
public enum EmfPlusSmoothingMode : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| SmoothingModeDefault | `0` | Especifica el suavizado de curva predeterminado sin suavizado. |
| SmoothingModeHighSpeed | `1` | Especifica el mejor rendimiento sin suavizado. |
| SmoothingModeHighQuality | `2` | Especifica la mejor calidad con suavizado. |
| SmoothingModeNone | `3` | No realiza suavizado de curvas ni suavizado. |
| SmoothingModeAntiAlias8x4 | `4` | Especifica buena calidad utilizando un filtro de caja de 8x4. |
| SmoothingModeAntiAlias8x8 | `5` | Especifica una mejor calidad utilizando un filtro de cuadro de 8x8. |
