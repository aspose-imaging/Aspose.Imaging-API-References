## EmfPlusLineStyle enumeration

La enumeración LineStyle define estilos de líneas que se dibujan con plumas gráficas.

```csharp
public enum EmfPlusLineStyle
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| LineStyleSolid | `0` | Especifica una línea continua. |
| LineStyleDash | `1` | Especifica una línea discontinua. |
| LineStyleDot | `2` | Especifica una línea de puntos. |
| LineStyleDashDot | `3` | Especifica una línea alterna de puntos y guiones. |
| LineStyleDashDotDot | `4` | Especifica una línea alterna de guión-punto-punto. |
| LineStyleCustom | `5` | Especifica una línea discontinua personalizada y definida por el usuario. |

### Observaciones

Las líneas de gráficos se especifican mediante[`EmfPlusPen`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluspen) objetos (sección 2.2.1.7).
