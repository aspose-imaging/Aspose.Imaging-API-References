## EmfPlusCompositingQuality enumeration

La enumeración CompositingQuality define niveles de calidad para crear imágenes compuestas

```csharp
public enum EmfPlusCompositingQuality : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| CompositingQualityDefault | `1` | No se realiza ninguna corrección gamma. La corrección gamma controla el brillo y el contraste generales de una imagen. Sin la corrección gamma, las imágenes compuestas pueden aparecer demasiado claras o demasiado oscuras. |
| CompositingQualityHighSpeed | `2` | No se realiza ninguna corrección gamma. Se favorece la velocidad de composición a expensas de la calidad. En cuanto al resultado, no hay diferencia entre este valor y CompositingQualityDefault. |
| CompositingQualityHighQuality | `3` | Se realiza la corrección gamma. Se favorece la calidad de composición a expensas de la velocidad. |
| CompositingQualityGammaCorrected | `4` | Habilite la corrección gamma para composiciones de mayor calidad con menor velocidad. En cuanto al resultado, no hay diferencia entre este valor y CompositingQualityHighQuality. |
| CompositingQualityAssumeLinear | `5` | No se realiza ninguna corrección gamma; sin embargo, el uso de valores lineales da como resultado una mejor calidad que la predeterminada a una velocidad ligeramente inferior. |
