## Matrix.TypeGeneralTransform field

Esta constante indica que la transformación definida por este objeto realiza una conversión arbitraria de las coordenadas de entrada. Si esta transformación se puede clasificar por cualquiera de las constantes anteriores, el tipo será la constante TypeIdentity o una combinación de la bandera apropiada bits para las distintas conversiones de coordenadas que realiza esta transformación.
