## FileStreamContainer.IsTemporal property

Obtiene o establece un valor que indica si el flujo es temporal.

```csharp
public bool IsTemporal { get; set; }
```

### El valor de la propiedad

`verdadero` si el flujo es temporal; de lo contrario,`falso` .

### Observaciones

Una secuencia temporal se eliminará cuando se elimine. Si la transmisión se basa en la memoria, esta propiedad no tiene ningún efecto. La transmisión se puede marcar como temporal o persistente en caso de que se haya creado explícitamente; de lo contrario, se genera la excepción correspondiente.
