## EmfPlusBitmap.PixelFormat property

Obtiene o establece el formato de píxeles PixelFormat (4 bytes): un entero sin signo de 32 bits que especifica el formato de los píxeles que componen la imagen de mapa de bits . Los formatos de píxeles admitidos se especifican en el[`EmfPlusPixelFormat`](../../../aspose.imaging.fileformats.emf.emfplus.consts/emfpluspixelformat) enumeración (sección 2.1.1.25). Si la imagen está comprimida, según el campo Tipo, este valor no está definido y DEBE ignorarse.

```csharp
public EmfPlusPixelFormat PixelFormat { get; set; }
```

### El valor de la propiedad

El formato de píxel.
