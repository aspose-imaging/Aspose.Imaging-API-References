## WmfSetTextCharExtra class

El registro META_SETTEXTCHAREXTRA define el espaciado entre caracteres para la justificación de texto en el contexto del dispositivo de reproducción. El espacio se agrega a el espacio en blanco entre cada carácter, incluido `break` caracteres, cuando una línea de texto justificado es salida.
