## JpegLsPresetCodingParameters class

Define los parámetros de codificación preestablecidos de JPEG-LS como se define en ISO/IEC 14495-1, C.2.4.1.1. JPEG-LS define un conjunto predeterminado de parámetros, pero se pueden usar parámetros personalizados. Cuando se usan, estos parámetros se escriben en el flujo de bits codificados, ya que son necesarios para el proceso de decodificación.
