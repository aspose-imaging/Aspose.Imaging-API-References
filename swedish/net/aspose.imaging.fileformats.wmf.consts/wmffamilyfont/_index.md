## WmfFamilyFont enumeration

FamilyFont-uppräkningen anger teckensnittsfamiljen. Teckensnittsfamiljer beskriver utseendet på ett teckensnitt i på ett allmänt sätt. De är avsedda för att ange typsnitt när det exakta typsnittet som önskas inte är tillgängligt.

```csharp
public enum WmfFamilyFont : byte
```

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| Dontcare | `0` | Standardteckensnittet är angivet, vilket är implementeringsberoende. |
| Roman | `1` | Teckensnitt med variabla streckbredder, som är proportionella mot de faktiska bredderna på glyferna, och som har seriffer. "MS Serif" är ett exempel. |
| Swiss | `2` | Teckensnitt med variabla streckbredder, som är proportionella mot de faktiska bredderna på -glyferna, och som inte har seriffer. "MS Sans Serif" är ett exempel. |
| Modern | `3` | Teckensnitt med konstant streckbredd, med eller utan seriffer. Typsnitt med fast bredd är vanligtvis moderna. "Pica", "Elite" och "Courier New" är exempel. |
| Script | `4` | Teckensnitt designade för att se ut som handstil. "Script" och "Cursive" är exempel. |
| Decorative | `5` | Nyhetsteckensnitt. "Old English" är ett exempel. |
