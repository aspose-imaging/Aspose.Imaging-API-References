## WmfOutPrecision enumeration

OutPrecision-uppräkningen definierar värden för utdataprecision, vilket är kravet för att teckensnittsmapparen ska matcha specifika teckensnittsparametrar, inklusive höjd, bredd, teckenorientering, escapement, tonhöjd och teckensnittstyp.

```csharp
public enum WmfOutPrecision : byte
```

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| Default | `0` | Ett värde som anger standardbeteende. |
| String | `1` | Ett värde som returneras när rastrerade teckensnitt räknas upp. |
| Stroke | `3` | Ett värde som returneras när TrueType och andra konturteckensnitt och vektorteckensnitt räknas upp. |
| Tt | `4` | Ett värde som anger valet av ett TrueType-teckensnitt när system innehåller flera teckensnitt med samma namn. |
| Device | `5` | Ett värde som anger valet av ett enhetsteckensnitt när system innehåller flera teckensnitt med samma namn. |
| Raster | `6` | Ett värde som anger valet av ett rastrerat teckensnitt när system innehåller flera teckensnitt med samma namn. |
| TtOnly | `7` | Ett värde som anger kravet för endast TrueType-teckensnitt. Om det inte finns några TrueType-teckensnitt installerade i systemet, anges standardbeteende. |
| Outline | `8` | Ett värde som anger kravet för TrueType och andra konturteckensnitt. |
