## BinarizeBradley(double, int) {#binarizebradley_1}

Binarisering av en bild med Bradleys adaptiva tröskelalgoritm med hjälp av integralbildströskelvärdet

```csharp
public virtual void BinarizeBradley(double brightnessDifference, int windowSize)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| brightnessDifference | Double | Skillnaden i ljusstyrka mellan pixel och genomsnittet av ett sxs-fönster med pixlar centrerat runt denna pixel. |
| windowSize | Int32 | Storleken på sxs-fönstret med pixlar centrerat runt denna pixel |

### Exempel

Följande exempel binariserar en rasterbild med Bradleys adaptiva tröskelalgoritm med den angivna fönsterstorleken. Binariserade bilder innehåller endast 2 färger - svart och vitt.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Binarisera bilden med en ljusstyrkeskillnad på 5. Ljusstyrkan är skillnaden mellan en pixel och genomsnittet av ett 10 x 10 fönster med pixlar centrerat runt denna pixel.
    rasterImage.BinarizeBradley(5, 10);
    rasterImage.Save(dir + "sample.BinarizeBradley5_10x10.png");
}
```
