## RasterImage.SetPixel method

Ställer in en bildpixel för den angivna positionen.

```csharp
public void SetPixel(int x, int y, Color color)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Int32 | Pixel x platsen. |
| y | Int32 | Pixel y-platsen. |
| color | Color | Pixelfärgen för den angivna positionen. |

### Exempel

Följande exempel laddar en rasterbild och ställer in färgen på en godtycklig pixel.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Ställer in färgen på den övre vänstra pixeln.
    rasterImage.SetArgb32Pixel(0, 0, Aspose.Imaging.Color.Aqua.ToArgb());

    // Ett annat sätt är att skicka en instans av Aspose.Imaging.Color direkt
    rasterImage.SetPixel(0, 0, Aspose.Imaging.Color.Aqua);
}
```
