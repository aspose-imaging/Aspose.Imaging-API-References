## Image.UsePalette property

Får ett värde som indikerar om bildpaletten används.

```csharp
public virtual bool UsePalette { get; }
```

### Fastighetsvärde

`Sann` om paletten används i bilden; annat,`falsk` .

### Exempel

Bestäm om paletten används av bilden.

```csharp
[C#]

using (var image = Image.Load(folder + "Sample.bmp"))
{
    if (image.UsePalette)

        Console.WriteLine("The palette is used by the image");
    }
}
```
