## FileStreamContainer.IsTemporal property

Hämtar eller ställer in ett värde som anger om strömmen är temporär.

```csharp
public bool IsTemporal { get; set; }
```

### Fastighetsvärde

`Sann` om strömmen är tidsmässig; annat,`falsk` .

### Anmärkningar

En temporär ström tar bort sig själv när den kasseras. Om strömmen är minnesbaserad har den här egenskapen ingen effekt. Strömmen kan markeras som temporär eller beständig om den skapades uttryckligen, annars skapas det lämpliga undantaget.
