## DjvuImage.CacheData method

Cachelagrar data privat.

```csharp
public override void CacheData()
```

### Exempel

Följande exempel visar hur man cachelagrar alla sidor i en DJVU-bild.

```csharp
[C#]

string dir = "c:\\temp\\";

// Ladda en bild från en DJVU-fil.
using (Aspose.Imaging.FileFormats.Djvu.DjvuImage image = (Aspose.Imaging.FileFormats.Djvu.DjvuImage)Aspose.Imaging.Image.Load(dir + "sample.djvu"))
{
    // Detta anrop cachar alla sidor så att ingen ytterligare dataladdning kommer att utföras från den underliggande dataströmmen.
    image.CacheData();

    // Eller så kan du cachelagra sidorna individuellt.
    foreach (Aspose.Imaging.FileFormats.Djvu.DjvuPage page in image.Pages)
    {
        page.CacheData();
    }
}
```
