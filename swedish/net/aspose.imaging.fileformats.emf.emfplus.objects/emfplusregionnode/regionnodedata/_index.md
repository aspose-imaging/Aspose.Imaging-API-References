## EmfPlusRegionNode.RegionNodeData property

Hämtar eller ställer in en valfri data med variabel längd som definierar dataobjektet region node som anges i fältet Typ. Datans innehåll och format kan vara olika för varje region nodtyp. Det här fältet FÅR INTE finnas om typen node är RegionNodeDataTypeEmpty eller RegionNodeDataTypeInfinite Detta objekt är generiskt och används för att specificera olika typer av regionnoddata, inklusive: Ett EmfPlusRegionNodePath-objekt, för.2 terminal.0;x2; EmfPlusRectF-objekt (avsnitt 2.2.2.39), för en terminalnod; and Ett EmfPlusRegionNodeChildNodes-objekt (avsnitt 2.2.2.41), för en icke-terminal nod.
