## EmfPlusBrightnessContrastEffect.ContrastLevel property

Hämtar eller ställer in ett 32-bitars signerat heltal som anger kontrastnivån. Detta värde MÅSTE vara i intervallet -100 till 100, med effekter enligt följande: -100 ≤ värde &lt; 0 När värdet minskar SKA bildens kontrast minska. 0 Ett värde på 0 anger att kontrasten INTE FÅR ändras . 0 &lt; värde ≤ 100 När värdet ökar, SKA bildens kontrast öka.
