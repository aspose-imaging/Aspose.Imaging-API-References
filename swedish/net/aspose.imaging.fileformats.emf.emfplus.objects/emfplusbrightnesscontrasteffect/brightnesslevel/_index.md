## EmfPlusBrightnessContrastEffect.BrightnessLevel property

Hämtar eller ställer in ett 32-bitars signerat heltal som anger ljusstyrka. Detta värde MÅSTE vara i intervallet -255 till 255, med effekter enligt följande: -255 ≤ värde &lt; 0 När värdet minskar SKA bildens ljusstyrka minska. 0 Ett värde på 0 anger att ljusstyrkan INTE FÅR ändras . 0 &lt; värde ≤ 255 När värdet ökar SKA bildens ljusstyrka öka.
