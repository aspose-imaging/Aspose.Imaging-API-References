## EmfPlusSmoothingMode enumeration

Uppräkningen SmoothingMode definierar utjämningslägen som ska tillämpas på linjer, kurvor och kanterna på fyllda områden för att få dem att framstå som mer kontinuerliga eller skarpt definierade.

```csharp
public enum EmfPlusSmoothingMode : byte
```

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| SmoothingModeDefault | `0` | Anger standardkurvutjämning utan kantutjämning. |
| SmoothingModeHighSpeed | `1` | Anger bästa prestanda utan kantutjämning. |
| SmoothingModeHighQuality | `2` | Anger bästa kvalitet med kantutjämning. |
| SmoothingModeNone | `3` | Utför ingen kurvutjämning och ingen kantutjämning. |
