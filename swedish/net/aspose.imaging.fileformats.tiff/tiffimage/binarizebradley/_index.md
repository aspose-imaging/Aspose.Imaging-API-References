## TiffImage.BinarizeBradley method

Binarisering av en bild med Bradleys adaptiva tröskelalgoritm med hjälp av integralbildströskelvärdet

```csharp
public override void BinarizeBradley(double brightnessDifference, int windowSize)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| brightnessDifference | Double | Skillnaden i ljusstyrka mellan pixel och genomsnittet av ett sxs-fönster av pixels centrerat runt denna pixel. |
| windowSize | Int32 | Storleken på sxs-fönstret med pixlar centrerat runt denna pixel |

### Exempel

Följande exempel binariserar en TIFF-bild med Bradleys adaptiva tröskelalgoritm med den angivna fönsterstorleken. Binariserade bilder innehåller endast 2 färger - svart och vitt.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.tif"))
{
    Aspose.Imaging.FileFormats.Tiff.TiffImage tiffImage = (Aspose.Imaging.FileFormats.Tiff.TiffImage)image;

    // Binarisera bilden med en ljusstyrkeskillnad på 5. Ljusstyrkan är skillnaden mellan en pixel och genomsnittet av ett 10 x 10 fönster med pixlar centrerat runt denna pixel.
    tiffImage.BinarizeBradley(5, 10);
    tiffImage.Save(dir + "sample.BinarizeBradley5_10x10.png", new Aspose.Imaging.ImageOptions.PngOptions());
}
```
