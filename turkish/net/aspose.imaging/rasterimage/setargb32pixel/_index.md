## RasterImage.SetArgb32Pixel method

Belirtilen konum için bir 32-bit ARGB pikseli ayarlar.

```csharp
public void SetArgb32Pixel(int x, int y, int argb32Color)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| x | Int32 | Piksel x konumu. |
| y | Int32 | Piksel y konumu. |
| argb32Color | Int32 | Belirtilen konum için 32 bit ARGB pikseli. |

### Örnekler

Aşağıdaki örnek, bir raster görüntü yükler ve rastgele bir pikselin rengini ayarlar.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Sol üst pikselin rengini ayarlar.
    rasterImage.SetArgb32Pixel(0, 0, Aspose.Imaging.Color.Aqua.ToArgb());

    // Başka bir yol da Aspose.Imaging.Color'ın bir örneğini doğrudan iletmektir.
    rasterImage.SetPixel(0, 0, Aspose.Imaging.Color.Aqua);
}
```
