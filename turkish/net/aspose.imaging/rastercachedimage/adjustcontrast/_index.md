## RasterCachedImage.AdjustContrast method

Görüntü kontrastı

```csharp
public override void AdjustContrast(float contrast)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| contrast | Single | Kontrast değeri ([-100; 100] aralığında) |

### Örnekler

Aşağıdaki örnek, raster önbelleğe alınmış bir görüntünün kontrast düzeltmesini gerçekleştirir.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterCachedImage rasterImage = (Aspose.Imaging.RasterCachedImage)image;

    // Kontrast değerini ayarlayın. Kabul edilen kontrast değerleri [-100f, 100f] aralığındadır.
    rasterImage.AdjustContrast(50);
    rasterImage.Save(dir + "sample.AdjustContrast.png");
}
```
