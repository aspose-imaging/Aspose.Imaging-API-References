| Parametre | Tip | Tanım |

### Geri dönüş değeri

`doğru` görüntü, geçirilen kaydetme seçenekleriyle temsil edilen belirtilen dosya biçiminde kaydedilebilirse; aksi halde,`yanlış` .

### Örnekler

Bu örnek, görüntünün, geçirilen kaydetme seçenekleriyle temsil edilen belirtilen dosya biçimine kaydedilip kaydedilemeyeceğini belirlemeyi gösterir.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.ImageOptions.JpegOptions saveOptions = new Aspose.Imaging.ImageOptions.JpegOptions();
    saveOptions.Quality = 50;

    // Resmin Jpeg'e kaydedilip kaydedilemeyeceğini belirleyin
    bool canSave = image.CanSave(saveOptions);
}
```
