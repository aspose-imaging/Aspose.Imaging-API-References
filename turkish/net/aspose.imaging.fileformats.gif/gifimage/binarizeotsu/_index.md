## GifImage.BinarizeOtsu method

Otsu eşik değeri ile bir görüntünün ikilileştirilmesi

```csharp
public override void BinarizeOtsu()
```

### Örnekler

Aşağıdaki örnek, bir GIF görüntüsünü Otsu eşikleme ile ikili hale getirir. İkilileştirilmiş görüntüler yalnızca 2 renk içerir - siyah ve beyaz.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.FileFormats.Gif.GifImage gifImage = (Aspose.Imaging.FileFormats.Gif.GifImage)image;

    // Görüntüyü Otsu eşikleme ile ikili hale getirin.
    gifImage.BinarizeOtsu();
    gifImage.Save(dir + "sample.BinarizeOtsu.png", new Aspose.Imaging.ImageOptions.PngOptions());
}
```
