## DjvuImage.CacheData method

Özel verileri önbelleğe alır.

```csharp
public override void CacheData()
```

### Örnekler

Aşağıdaki örnek, bir DJVU görüntüsünün tüm sayfalarının nasıl önbelleğe alınacağını gösterir.

```csharp
[C#]

string dir = "c:\\temp\\";

// Bir DJVU dosyasından bir görüntü yükleyin.
using (Aspose.Imaging.FileFormats.Djvu.DjvuImage image = (Aspose.Imaging.FileFormats.Djvu.DjvuImage)Aspose.Imaging.Image.Load(dir + "sample.djvu"))
{
    // Bu çağrı, tüm sayfaları önbelleğe alır, böylece temel alınan veri akışından ek veri yüklemesi yapılmaz.
    image.CacheData();

    // Veya sayfaları ayrı ayrı önbelleğe alabilirsiniz.
    foreach (Aspose.Imaging.FileFormats.Djvu.DjvuPage page in image.Pages)
    {
        page.CacheData();
    }
}
```
