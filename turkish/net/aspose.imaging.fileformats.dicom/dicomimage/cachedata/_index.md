## DicomImage.CacheData method

Özel verileri önbelleğe alır.

```csharp
public override void CacheData()
```

### Örnekler

Aşağıdaki örnek, bir DICOM görüntüsünün tüm sayfalarının nasıl önbelleğe alınacağını gösterir.

```csharp
[C#]

string dir = "c:\\temp\\";

// DICOM dosyasından bir görüntü yükleyin.
using (Aspose.Imaging.FileFormats.Dicom.DicomImage image = (Aspose.Imaging.FileFormats.Dicom.DicomImage)Aspose.Imaging.Image.Load(dir + "sample.dicom"))
{
    // Bu çağrı, tüm sayfaları önbelleğe alır, böylece temel alınan veri akışından ek veri yüklemesi yapılmaz.
    image.CacheData();

    // Veya sayfaları ayrı ayrı önbelleğe alabilirsiniz.
    foreach (Aspose.Imaging.FileFormats.Dicom.DicomPage page in image.DicomPages)
    {
        page.CacheData();
    }
}
```
