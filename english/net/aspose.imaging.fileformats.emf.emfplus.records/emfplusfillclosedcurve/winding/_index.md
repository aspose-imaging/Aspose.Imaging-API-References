## EmfPlusFillClosedCurve.Winding property

Gets or sets a value indicating whether this [`EmfPlusFillClosedCurve`](../../emfplusfillclosedcurve) is winding. This bit indicates how to perform the fill operation. If set, the fill is a "winding" fill. If clear, the fill is an "alternate" fill.

```csharp
public bool Winding { get; set; }
```

### Property Value

`true` if winding; otherwise, `false`.
