## EmfPlusStringAlignment enumeration

The StringAlignment enumeration defines ways to align strings with respect to a text layout rectangle.

```csharp
public enum EmfPlusStringAlignment
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| StringAlignmentNear | `0` | Specifies that string alignment is toward the origin of the layout rectangle. This can be used to align characters along a line or to align text within a rectangle. For a right-to-left layout rectangle, the origin SHOULD be at the upper right. |
| StringAlignmentCenter | `1` | Specifies that alignment is centered between the origin and extent of the layout rectangle. |
| StringAlignmentFar | `2` | Specifies that alignment is to the right side of the layout rectangle. |
