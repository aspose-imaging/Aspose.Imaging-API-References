## EmfPlusBrushType enumeration

The BrushType enumeration defines types of graphics brushes, which are used to fill graphics regions.

```csharp
public enum EmfPlusBrushType
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| BrushTypeSolidColor | `0` | Specifies a solid-color brush, which is characterized by an EmfPlusArgb value. |
| BrushTypeHatchFill | `1` | Specifies a hatch brush, which is characterized by a predefined pattern. |
| BrushTypeTextureFill | `2` | Specifies a texture brush, which is characterized by an image. |
| BrushTypePathGradient | `3` | Specifies a path gradient brush, which is characterized by a color gradient path gradient brush data. |
| BrushTypeLinearGradient | `4` | contains linear gradient brush data. |
