## EmfDibColors enumeration

The DIBColors enumeration defines how to interpret the values in the color table of a DIB.

```csharp
public enum EmfDibColors
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| DIB_RGB_COLORS | `0` | The color table contains literal RGB values |
| DIB_PAL_COLORS | `1` | The color table consists of an array of 16-bit indexes into the LogPalette object (section 2.2.17) that is currently defined in the playback device context. |
| DIB_PAL_INDICES | `2` | No color table exists. The pixels in the DIB are indices into the current logical palette in the playback device context. |
