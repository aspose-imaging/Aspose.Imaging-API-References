## Rectangle.FromPoints method

Creates a new [`Rectangle`](../../rectangle) from two points specified. Two verticales of the created [`Rectangle`](../../rectangle) will be equal to the passed *point1* and *point2*. These would be typically the opposite vertices.

```csharp
public static Rectangle FromPoints(Point point1, Point point2)
```

| Parameter | Type | Description |
| --- | --- | --- |
| point1 | Point | The first [`Point`](../../point) for the new rectangle. |
| point2 | Point | The second [`Point`](../../point) for the new rectangle. |

## Return Value

A newly created [`Rectangle`](../../rectangle).
