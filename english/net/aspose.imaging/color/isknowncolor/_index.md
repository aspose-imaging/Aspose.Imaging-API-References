## Color.IsKnownColor property

Gets a value indicating whether this [`Color`](../../color) structure is a predefined color. Predefined colors are represented by the elements of the [`KnownColor`](../../knowncolor) enumeration.

```csharp
public bool IsKnownColor { get; }
```

### Property Value

True if this [`Color`](../../color) was created from a predefined color by using either the [`FromName`](../fromname) method or the [`FromKnownColor`](../fromknowncolor) method; otherwise, false.
