## Point Inequality operator

Compares two [`Point`](../../point) objects. The result specifies whether the values of the [`X`](../x) or [`Y`](../y) properties of the two [`Point`](../../point) objects are unequal.

| point2 | Point | A second [`Point`](../../point) to compare. |

### Return Value

True if the values of either the [`X`](../x) properties or the [`Y`](../y) properties of *point1* and *point2* differ; otherwise, false.
