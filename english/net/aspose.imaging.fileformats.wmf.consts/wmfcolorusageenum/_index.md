## WmfColorUsageEnum enumeration

The ColorUsage Enumeration specifies whether a color table exists in a device-independent bitmap(DIB) and how to interpret its values.

```csharp
public enum WmfColorUsageEnum
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| DIB_RGB_COLORS | `0` | The color table contains RGB values specified by RGBQuad Objects (section 2.2.2.20). |
| DIB_PAL_COLORS | `1` | The color table contains 16-bit indices into the current logical palette in the playback device context. |
