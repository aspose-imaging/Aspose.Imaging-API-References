## EmfPlusTintEffect.Amount property

Gets or sets A 32-bit signed integer that specifies how much the hue is strengthened or weakened. -100 ≤ value &lt; 0 Negative values specify how much the hue is weakened, which equates to the addition of black. 0 A value of 0 specifies that the tint MUST NOT change. 0 &lt; value ≤ 100 Positive values specify how much the hue is strengthened, which equates to the addition of white.
