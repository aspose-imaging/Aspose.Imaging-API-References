## WmfLogColorSpace.GammaRed property

Gets or sets a 32-bit fixed point value that defines the toned response curve for red. If the [`ColorSpaceType`](../colorspacetype/) field does not specify LCS_CALIBRATED_RGB, this field MUST be ignored.
