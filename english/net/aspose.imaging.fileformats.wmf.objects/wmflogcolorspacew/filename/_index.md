## WmfLogColorSpaceW.Filename property

Gets or sets an optional, null-terminated Unicode UTF16-LE character string, which specifies the name of a file that contains a color profile. If a file name is specified, and the [`ColorSpaceType`](../colorspacetype/) field is set to LCS_CALIBRATED_RGB, the other fields of this structure SHOULD be ignored.
