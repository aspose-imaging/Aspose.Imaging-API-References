| **Name** | **Description** |
| :- | :- |
| abort() | Aborts this task.<br/>            The task is completed immediately, with the risk of not freeing internal unmanaged resources. |

| run_async() | Runs this task. |
| wait_on_done() | Waits until task is finished. |
