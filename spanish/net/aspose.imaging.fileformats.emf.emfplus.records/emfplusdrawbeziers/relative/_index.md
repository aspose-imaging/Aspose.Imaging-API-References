## EmfPlusDrawBeziers.Relative property

Obtiene o establece un valor que indica si PointData es relativo. Si se establece, cada elemento de PointData especifica una ubicación en el espacio de coordenadas que es relativa a la ubicación especificada por el elemento anterior en la matriz. En el caso del primer elemento en PointData, se asume una ubicación previa en las coordenadas (0,0). Si está claro, PointData especifica ubicaciones absolutas según al indicador C. Nota Si se establece este indicador, el indicador C (arriba) no está definido y DEBE ignorarse.

```csharp
public bool Relative { get; set; }
```

### El valor de la propiedad

`verdadero` si es relativo; de lo contrario,`falso` .
