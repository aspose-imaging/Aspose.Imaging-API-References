## EmfPlusDrawArc.Size property

Obtiene o establece el tamaño. Un entero sin signo de 32 bits que especifica el número alineado de 32 bits de bytes en todo el registro, incluido el encabezado del registro de 12 bytes y datos específicos del registro. Para este tipo de registro, el valor DEBE ser uno de los siguientes: 0x0000001C Si el bit C está establecido en el campo Banderas. 0x00000024 Si el bit C está limpio en el campo Banderas
