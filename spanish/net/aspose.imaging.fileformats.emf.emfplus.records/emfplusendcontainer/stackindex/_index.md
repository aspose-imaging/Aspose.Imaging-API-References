## EmfPlusEndContainer.StackIndex property

Obtiene o establece un entero sin signo de 32 bits que especifica el índice de un contenedor state de gráficos. El índice DEBE coincidir con el valor asociado con un estado de gráficos container abierto por un EmfPlusBeginContainer anterior (sección 2.3.7.1) o registro EmfPlusBeginContainerNoParams (sección 2.3.7.2).

```csharp
public int StackIndex { get; set; }
```

### El valor de la propiedad

El índice de la pila.
