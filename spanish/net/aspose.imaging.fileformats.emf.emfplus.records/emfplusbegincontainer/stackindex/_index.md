## EmfPlusBeginContainer.StackIndex property

Obtiene o establece un entero sin signo de 32 bits que especifica un índice para asociarlo con el contenedor de estado de gráficos . El índice DEBE ser referenciado por un registro EmfPlusEndContainer subsiguiente (sección 2.3.7.3) para cerrar el contenedor de estado de gráficos.

```csharp
public int StackIndex { get; set; }
```

### El valor de la propiedad

El índice de la pila.
