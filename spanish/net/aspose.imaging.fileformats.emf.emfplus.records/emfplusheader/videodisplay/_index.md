## EmfPlusHeader.VideoDisplay property

Obtiene o establece un valor que indica si se muestra video. si se establece, esta marca indica que el metarchivo se grabó con un contexto de dispositivo de referencia para una pantalla de video. Si está claro, el metarchivo se registró con un contexto de dispositivo de referencia para una impresora.

```csharp
public bool VideoDisplay { get; set; }
```

### El valor de la propiedad

`verdadero` si [visualización de video]; de lo contrario,`falso` .
