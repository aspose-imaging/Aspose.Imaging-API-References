### Ejemplos

El siguiente ejemplo carga imágenes PNG e imprime información sobre el formato de datos sin procesar y el canal alfa.

```csharp
[C#]

// Las imágenes PNG a cargar.
string[] fileNames = new string[]
{
    @"c:\temp\sample.png",
    @"c:\temp\alpha.png",
};

foreach (string fileName in fileNames)
{
    using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(fileName))
    {
        Aspose.Imaging.FileFormats.Png.PngImage pngImage = (Aspose.Imaging.FileFormats.Png.PngImage)image;
        System.Console.WriteLine("ImageFile={0}, FileFormat={1}, HasAlpha={2}", fileName, pngImage.RawDataFormat, pngImage.HasAlpha);
    }
}

// La salida puede verse así:
// ImageFile=c:\temp\sample.png, FileFormat=Rgb24Bpp, canales usados: 8,8,8, HasAlpha=False
// ImageFile=c:\temp\alpha.png, FileFormat=RGBA32Bpp, canales usados: 8,8,8,8, HasAlpha=True
```
