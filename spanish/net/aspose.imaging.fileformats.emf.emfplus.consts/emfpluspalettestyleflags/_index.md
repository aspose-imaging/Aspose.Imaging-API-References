## EmfPlusPaletteStyleFlags enumeration

Los indicadores PaletteStyle especifican las propiedades de las paletas de gráficos. Estas banderas se pueden combinar para especificar múltiples opciones.

```csharp
[Flags]
public enum EmfPlusPaletteStyleFlags
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| PaletteStyleHasAlpha | `1` | Si se establece, una o más de las entradas de la paleta DEBEN contener información de transparencia alfa. |
| PaletteStyleGrayScale | `2` | Si se establece, la paleta DEBE contener solo entradas en escala de grises. |
| PaletteStyleHalftone | `4` | Si se establece, la paleta DEBE contener valores de color discretos que se pueden usar para medios tonos. |

### Observaciones

Las paletas de gráficos se especifican mediante[`EmfPlusPalette`](../../aspose.imaging.fileformats.emf.emfplus.objects/emfpluspalette) objetos.
