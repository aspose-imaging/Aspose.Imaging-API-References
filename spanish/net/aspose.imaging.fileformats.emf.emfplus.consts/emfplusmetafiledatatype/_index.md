## EmfPlusMetafileDataType enumeration

La enumeración MetafileDataType define tipos de datos de metarchivos que se pueden incrustar en un metarchivo EMF+.

```csharp
public enum EmfPlusMetafileDataType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| MetafileDataTypeWmf | `1` | Especifica que el metarchivo es un metarchivo WMF que especifica operaciones gráficas con registros WMF, como se especifica en [MS-WMF]. |
| MetafileDataTypeWmfPlaceable | `2` | Especifica que el metarchivo es un metarchivo WMF que especifica operaciones gráficas con registros WMF y que contiene información de encabezado adicional que hace que el metarchivo WMF sea independiente del dispositivo, como se especifica en [MS-WMF]. |
| MetafileDataTypeEmf | `3` | Especifica que el metarchivo es un metarchivo EMF que especifica operaciones gráficas con registros EMF, como se especifica en [MS-EMF]. |
| MetafileDataTypeEmfPlusOnly | `4` | Especifica que el metarchivo es un metarchivo EMF+ que especifica operaciones gráficas solo con registros EMF+. |
| MetafileDataTypeEmfPlusDual | `5` | Especifica que el metarchivo es un metarchivo EMF+ que especifica operaciones gráficas con registros EMF y EMF+. |

### Observaciones

Los datos del metarchivo incrustado se especifican mediante objetos EmfPlusMetafileData (sección 2.2.2.27).
