## EmfPlusImageEffectsIdentifiers class

Los identificadores de ImageEffects definen GUID estándar para especificar efectos de imagen de gráficos. Los controladores de dispositivos utilizan estos identificadores para publicar sus niveles de compatibilidad con estos efectos. Las constantes del identificador se definen utilizando la representación de cadena entre llaves GUID ([MS-DTYP] sección 2.3.4.3).
