## EmfPlusCurveAdjustments enumeration

La enumeración CurveAdjustments define los ajustes que se pueden aplicar a la curva de color de una imagen.

```csharp
public enum EmfPlusCurveAdjustments
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| AdjustExposure | `0` | Especifica la simulación de aumentar o disminuir la exposición de una imagen. |
| AdjustDensity | `1` | Especifica la simulación de aumentar o disminuir la densidad de una imagen. |
| AdjustContrast | `2` | Especifica un aumento o disminución del contraste de una imagen. |
| AdjustHighlight | `3` | Especifica un aumento o disminución del valor de un canal de color de una imagen, si ese canal ya tiene un valor superior a la mitad de la intensidad. Este ajuste se puede usar para aumentar la definición en las áreas claras de una imagen sin afectar las áreas oscuras. |
| AdjustShadow | `4` | Especifica un aumento o disminución del valor de un canal de color de una imagen, si ese canal ya tiene un valor que está por debajo de la mitad de la intensidad. Este ajuste se puede usar para aumentar la definición en las áreas oscuras de una imagen sin afectar las áreas claras. |
| AdjustMidtone | `5` | Especifica un ajuste que aclara u oscurece una imagen. Los valores del canal de color en el medio del rango de intensidad se alteran más que los valores del canal de color cerca de los extremos mínimo o máximo de intensidad. Este ajuste se puede utilizar para aclarar u oscurecer una imagen sin perder el contraste entre las partes más oscuras y más claras de la imagen. |
| AdjustWhiteSaturation | `6` | Especifica un ajuste a la saturación de blanco de una imagen, definida como el valor máximo en el rango de intensidades para un canal de color determinado, cuyo rango suele ser de 0 a 255. |
| AdjustBlackSaturation | `7` | Especifica un ajuste a la saturación de negro de una imagen, que es el valor mínimo en el rango de intensidades para un canal de color determinado, que suele ser de 0 a 255. |
