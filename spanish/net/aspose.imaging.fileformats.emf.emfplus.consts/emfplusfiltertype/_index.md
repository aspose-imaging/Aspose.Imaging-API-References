## EmfPlusFilterType enumeration

La enumeración FilterType define los tipos de algoritmos de filtrado que se pueden usar para mejorar la calidad de texto y gráficos y representar imágenes.

```csharp
public enum EmfPlusFilterType : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| FilterTypeNone | `0` | Especifica que no se realiza el filtrado. |
| FilterTypePoint | `1` | Especifica que cada píxel de destino se calcula muestreando el píxel más cercano de la imagen de origen. |
| FilterTypeLinear | `2` | Especifica que la interpolación lineal se realiza utilizando el promedio ponderado de un área de píxeles de 2x2 que rodea el píxel de origen. |
| FilterTypeTriangle | `3` | Especifica que cada píxel de la imagen de origen contribuye por igual a la imagen de destino. Este es el más lento de los algoritmos de filtrado. |
| FilterTypeBox | `4` | Especifica un algoritmo de filtro de cuadro, en el que cada píxel de destino se calcula promediando un rectángulo de píxeles de origen. Este algoritmo es útil solo cuando se reduce el tamaño de una imagen. |
| FilterTypePyramidalQuad | `6` | Especifica que se utiliza un filtro de tienda de 4 muestras. |
| FilterTypeGaussianQuad | `7` | Especifica que se utiliza un filtro gaussiano de 4 muestras, que crea un efecto de desenfoque en una imagen. |
