## EmfPlusFontStyleFlags enumeration

Las banderas FontStyle especifican estilos de tipografías de fuentes gráficas. Estas banderas se pueden combinar para especificar múltiples opciones.

```csharp
[Flags]
public enum EmfPlusFontStyleFlags
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| FontStyleBold | `1` | Si se establece, el tipo de letra DEBE representarse con un peso o grosor mayor. Si está claro, el tipo de letra DEBE representarse con un grosor normal. |
| FontStyleItalic | `2` | Si se establece, el tipo de letra DEBE representarse con los tallos verticales de los caracteres en un ángulo aumentado o inclinado en relación con la línea de base. Si está claro, el tipo de letra DEBE representarse con los tallos verticales de los caracteres en un ángulo normal. |
| FontStyleUnderline | `4` | Si se establece, el tipo de letra DEBE representarse con una línea debajo de la línea base de los caracteres. Si está claro, el tipo de letra DEBE representarse sin una línea debajo de la línea base. |
| FontStyleStrikeout | `8` | Si se establece, el tipo de letra DEBE representarse con una línea paralela a la línea base dibujada a través del medio de los caracteres. Si está claro, el tipo de letra DEBE representarse sin una línea a través de los caracteres. |
