## EmfPlusDriverStringOptionsFlags enumeration

Los indicadores de DriverStringOptions especifican las propiedades del posicionamiento y la representación del texto de los gráficos. Estas banderas se pueden combinar para especificar múltiples opciones.

```csharp
[Flags]
public enum EmfPlusDriverStringOptionsFlags
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| DriverStringOptionsCmapLookup | `1` | Si se establece, las posiciones de los glifos de caracteres DEBERÍAN especificarse en una tabla de búsqueda de mapa de caracteres. Si está desactivada, las posiciones de los glifos DEBERÍAN obtenerse de una matriz de coordenadas. |
| DriverStringOptionsVertical | `2` | Si se establece, la cadena DEBE representarse verticalmente. Si está desactivada, la cadena DEBE representarse horizontalmente. |
| DriverStringOptionsRealizedAdvance | `4` | Si se establece, las posiciones de los glifos de los caracteres DEBERÍAN calcularse en relación con la posición del primer glifo. Si está desactivada, las posiciones de los glifos DEBERÍAN obtenerse a partir de una matriz de coordenadas. |
| DriverStringOptionsLimitSubpixel | `8` | Si se establece, se DEBE usar menos memoria para almacenar en caché los glifos suavizados, lo que produce una representación de texto de menor calidad. Si se establece, se DEBE usar más memoria, lo que produce una representación de texto de mayor calidad. |
