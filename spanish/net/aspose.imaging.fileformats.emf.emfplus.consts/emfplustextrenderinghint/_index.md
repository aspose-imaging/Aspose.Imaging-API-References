## EmfPlusTextRenderingHint enumeration

La enumeración TextRenderingHint define tipos de sugerencias de texto y suavizado, lo que afecta la calidad de la representación del texto.

```csharp
public enum EmfPlusTextRenderingHint : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| TextRenderingHintSystemDefault | `0` | Especifica que cada carácter de texto DEBE dibujarse utilizando cualquier configuración de suavizado de fuentes que se haya configurado en el sistema operativo. |
| TextRenderingHintSingleBitPerPixelGridFit | `1` | Especifica que cada carácter de texto DEBE dibujarse utilizando su mapa de bits de glifo. El suavizado PUEDE usarse para mejorar la apariencia de los tallos y la curvatura de los glifos de los caracteres. |
| TextRenderingHintSingleBitPerPixel | `2` | Especifica que cada carácter de texto DEBE dibujarse utilizando su mapa de bits de glifo. No se utiliza suavizado. |
| TextRenderingHintAntialiasGridFit | `3` | Especifica que cada carácter de texto DEBE dibujarse utilizando su mapa de bits de glifo suavizado con suavizado. El renderizado es de alta calidad debido al anti-aliasing, pero a un mayor costo de rendimiento. |
| TextRenderingHintAntialias | `4` | Especifica que cada carácter de texto se dibuja utilizando su mapa de bits de glifo suavizado sin sugerencias. Se obtienen mejores resultados de calidad con el suavizado, pero las diferencias en el ancho de la plica PUEDEN ser notables porque las sugerencias están desactivadas. |
| TextRenderingHintClearTypeGridFit | `5` | Especifica que cada carácter de texto DEBE dibujarse utilizando su mapa de bits de glifo ClearType con suavizado. Esta es la configuración de sugerencias de texto de la más alta calidad, que se utiliza para aprovechar las características de la fuente ClearType. |
