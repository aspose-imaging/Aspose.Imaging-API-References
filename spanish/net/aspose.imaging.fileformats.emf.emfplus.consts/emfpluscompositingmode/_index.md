## EmfPlusCompositingMode enumeration

La enumeración CompositingMode define modos para combinar colores de origen con colores de fondo. El modo de composición representa el estado habilitado de mezcla alfa.

```csharp
public enum EmfPlusCompositingMode : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| CompositingModeSourceOver | `0` | Habilita la combinación alfa, que especifica que cuando se representa un color, se combina con el color de fondo. La extensión de la fusión está determinada por el valor del componente alfa del color que se está renderizando. |
| CompositingModeSourceCopy | `1` | Deshabilita la combinación alfa, lo que significa que cuando se procesa un color de origen, sobrescribe el color de fondo. |
