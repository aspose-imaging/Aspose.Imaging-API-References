## WmfExtTextOut.Dx property

Obtiene o establece el dx.

```csharp
public short[] Dx { get; set; }
```

### El valor de la propiedad

El dx. Indique la distancia entre los orígenes de las celdas character adyacentes. Por ejemplo, las unidades lógicas Dx[i] separan los orígenes de la celda de caracteres i y la celda de caracteres i + 1. Si este campo está presente, DEBE haber el mismo número de valores que caracteres hay en la cadena.
