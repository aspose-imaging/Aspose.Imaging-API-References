## WmfBitmapInfoHeader.Width property

Obtiene o establece un entero con signo de 32 bits que define el ancho del DIB, en píxeles. Este valor DEBE ser positivo. Este campo DEBE especificar el ancho del archivo de imagen descomprimido, si el valor de Compresión especifica formato JPEG o PNG .
