## Image.CanSave method

Determina si la imagen se puede guardar en el formato de archivo especificado representado por las opciones de guardado pasadas.

```csharp
public bool CanSave(ImageOptionsBase options)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| options | ImageOptionsBase | Las opciones de guardado a usar. |

### Valor_devuelto

`verdadero` si la imagen se puede guardar en el formato de archivo especificado representado por las opciones de guardado pasadas; de lo contrario,`falso` .

### Ejemplos

Este ejemplo muestra cómo determinar si la imagen se puede guardar en el formato de archivo especificado representado por las opciones de guardado pasadas.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.ImageOptions.JpegOptions saveOptions = new Aspose.Imaging.ImageOptions.JpegOptions();
    saveOptions.Quality = 50;

    // Determinar si la imagen se puede guardar en Jpeg
    bool canSave = image.CanSave(saveOptions);
}
```
