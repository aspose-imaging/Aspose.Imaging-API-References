## StringFormat.DigitSubstitutionLanguage property

Obtiene o establece el idioma que se utiliza cuando los dígitos locales se sustituyen por dígitos occidentales.

```csharp
public int DigitSubstitutionLanguage { get; set; }
```

### El valor de la propiedad

Un identificador de idioma compatible con el idioma nacional (NLS) que identifica el idioma que se usará cuando los dígitos locales se sustituyan por dígitos occidentales. Puede pasar elLCID propiedad de unCultureInfo objeto como el identificador de idioma NLS. Por ejemplo, suponga que crea unCultureInfo objeto pasando la cadena "ar-EG" a unCultureInfo constructor. Si pasas elLCID propiedad de esoCultureInfo objeto junto con. Traditional haciaStringDigitSubstitute) método, entonces los dígitos árabes-índicos se sustituirán por dígitos occidentales en el momento de la visualización.

### Observaciones

El setter se introduce para el método obsoleto SetDigitSubstitution.
