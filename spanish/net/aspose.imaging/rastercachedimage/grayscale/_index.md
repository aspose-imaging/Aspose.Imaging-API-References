## RasterCachedImage.Grayscale method

Transformación de una imagen a su representación en escala de grises

```csharp
public override void Grayscale()
```

### Ejemplos

El siguiente ejemplo transforma una imagen almacenada en caché raster coloreada a su representación en escala de grises. Las imágenes en escala de grises están compuestas exclusivamente por tonos de gris y solo contienen información de intensidad.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterCachedImage rasterImage = (Aspose.Imaging.RasterCachedImage)image;

    rasterImage.Grayscale();
    rasterImage.Save(dir + "sample.Grayscale.png");
}
```
