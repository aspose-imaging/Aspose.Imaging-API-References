## LineJoin enumeration

Especifica cómo unir segmentos de línea o curva consecutivos en una figura (subtrayectoria) contenida en un[`GraphicsPath`](../graphicspath) objeto.

```csharp
public enum LineJoin
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Miter | `0` | Especifica una unión en inglete. Esto produce una esquina aguda o una esquina recortada, dependiendo de si la longitud del inglete excede el límite del inglete. |
| Bevel | `1` | Especifica una unión biselada. Esto produce una esquina diagonal. |
| Round | `2` | Especifica una unión circular. Esto produce un arco circular suave entre las líneas. |
| MiterClipped | `3` | Especifica una unión en inglete. Esto produce una esquina afilada o una esquina biselada, dependiendo de si la longitud del inglete excede el límite del inglete. |
