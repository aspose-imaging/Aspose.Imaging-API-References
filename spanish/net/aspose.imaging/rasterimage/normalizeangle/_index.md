## NormalizeAngle(bool, Color) {#normalizeangle_1}

Normaliza el ángulo. Este método es aplicable a documentos de texto escaneados para eliminar el escaneo sesgado. Este método utiliza[`GetSkewAngle`](../getskewangle) y[`Rotate`](../rotate) métodos.

```csharp
public virtual void NormalizeAngle(bool resizeProportionally, Color backgroundColor)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| resizeProportionally | Boolean | si se establece en`verdadero` cambiará el tamaño de la imagen de acuerdo con las proyecciones del rectángulo rotado (puntos de las esquinas) en otro caso, eso deja las dimensiones intactas y solo se rotan los contenidos internos de la imagen. |
| backgroundColor | Color | Color del fondo. |

### Ejemplos

La inclinación es un artefacto que puede aparecer durante el proceso de escaneado de documentos cuando el texto o las imágenes del documento se giran en un ligero ángulo. Puede tener varias causas, pero la más común es que el papel se extravíe durante un escaneo. Por lo tanto, enderezar es el proceso de detectar y solucionar este problema en archivos escaneados (es decir, mapa de bits) para que los documentos enderezados tengan el texto/las imágenes correctamente y ajustados horizontalmente.

```csharp
[C#]

string dir = "c:\\aspose.imaging\\issues\\net\\3567\\";

string inputFilePath = dir + "skewed.png";
string outputFilePath = dir + "skewed.out.png";

// Deshágase del escaneo sesgado con parámetros predeterminados
using (Aspose.Imaging.RasterImage image = (Aspose.Imaging.RasterImage)Aspose.Imaging.Image.Load(inputFilePath))
{
    // Alinear
    image.NormalizeAngle(false /*do not resize*/, Aspose.Imaging.Color.LightGray /*background color*/);
    image.Save(outputFilePath);
}
```
