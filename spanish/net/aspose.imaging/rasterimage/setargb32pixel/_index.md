## RasterImage.SetArgb32Pixel method

Establece un píxel ARGB de 32 bits de imagen para la posición especificada.

```csharp
public void SetArgb32Pixel(int x, int y, int argb32Color)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| x | Int32 | La ubicación del píxel x. |
| y | Int32 | La ubicación del píxel y. |
| argb32Color | Int32 | El píxel ARGB de 32 bits para la posición especificada. |

### Ejemplos

El siguiente ejemplo carga una imagen rasterizada y establece el color de un píxel arbitrario.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Establece el color del píxel superior izquierdo.
    rasterImage.SetArgb32Pixel(0, 0, Aspose.Imaging.Color.Aqua.ToArgb());

    // Otra forma es pasar una instancia de Aspose.Imaging.Color directamente
    rasterImage.SetPixel(0, 0, Aspose.Imaging.Color.Aqua);
}
```
