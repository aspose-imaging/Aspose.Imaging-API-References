## RasterImage.GetPixel method

Obtiene un píxel de imagen.

```csharp
public Color GetPixel(int x, int y)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| x | Int32 | La ubicación del píxel x. |
| y | Int32 | La ubicación del píxel y. |

### Valor_devuelto

El color de píxel de la ubicación especificada.

### Ejemplos

El siguiente ejemplo carga una imagen rasterizada y obtiene el color de un píxel arbitrario.

```csharp
[C#]

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(@"c:\temp\sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Obtenga el color del píxel superior izquierdo de la imagen.
    Color color = rasterImage.GetPixel(0, 0);

    // Obtener los valores de los componentes de color individuales
    byte alpha = color.A;
    byte red = color.R;
    int green = color.G;
    int blue = color.B;

    System.Console.WriteLine("The color of the pixel(0,0) is A={0},R={1},G={2},B={3}", alpha, red, green, blue);
}
```
