## WmfBrushStyle enumeration

La enumeración BrushStyle especifica los diferentes tipos de pinceles posibles que se pueden usar en las operaciones gráficas. Para obtener más información, consulte la especificación del objeto Brush (sección 2.2.1.1).

```csharp
public enum WmfBrushStyle
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Solid | `0` | Un pincel que pinta un solo color constante, ya sea sólido o difuminado. |
| Null | `1` | Un pincel que no hace nada. Usar un pincel NULL en una operación de gráficos DEBE tener el mismo efecto que no usar ningún pincel |
| Hatched | `2` | Un pincel que pinta un patrón simple predefinido, o "sombreado", sobre un fondo sólido |
| Pattern | `3` | Un pincel que pinta un patrón definido por un mapa de bits, que PUEDE ser un objeto Bitmap16 o un objeto DeviceIndependentBitmap (DIB). |
| Indexed | `4` | No compatible. |
| Dibpattern | `5` | Un pincel de patrón especificado por un DIB. |
| Dibpatternpt | `6` | Un pincel de patrón especificado por un DIB. |
| Pattern8X8 | `7` | No compatible. |
| Dibpattern8X8 | `8` | No compatible. |
| Monopattern | `9` | No compatible. |
| Gradient | `10` | El gradiente No soportado en wmf, soportado en odg. |
