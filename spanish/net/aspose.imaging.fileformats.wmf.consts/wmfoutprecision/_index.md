## WmfOutPrecision enumeration

La enumeración OutPrecision define valores para la precisión de salida, que es el requisito para el asignador de fuentes para que coincida con parámetros de fuente específicos, incluidos el alto, el ancho, la orientación de los caracteres, el escape , el paso y el tipo de fuente.

```csharp
public enum WmfOutPrecision : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `0` | Un valor que especifica el comportamiento predeterminado. |
| String | `1` | Un valor que se devuelve cuando se enumeran las fuentes rasterizadas. |
| Stroke | `3` | Un valor que se devuelve cuando se enumeran TrueType y otras fuentes de contorno y fuentes vectoriales. |
| Tt | `4` | Un valor que especifica la elección de una fuente TrueType cuando el sistema contiene varias fuentes con el mismo nombre. |
| Device | `5` | Un valor que especifica la elección de una fuente de dispositivo cuando el sistema contiene varias fuentes con el mismo nombre. |
| Raster | `6` | Un valor que especifica la elección de una fuente rasterizada cuando el sistema contiene varias fuentes con el mismo nombre. |
| TtOnly | `7` | Un valor que especifica el requisito de solo fuentes TrueType. Si no hay fuentes TrueType instaladas en el sistema, se especifica el comportamiento predeterminado. |
| Outline | `8` | Un valor que especifica el requisito para TrueType y otras fuentes de contorno. |
| ScreenOutline | `9` | Un valor que especifica una preferencia por TrueType y otras fuentes de contorno . |
| PsOnly | `10` | Un valor que especifica un requisito solo para fuentes PostScript. Si no hay fuentes PostScript instaladas en el sistema, se especifica el comportamiento predeterminado. |
