## WmfPenStyle enumeration

La enumeración PenStyle de 16 bits se usa para especificar diferentes tipos de plumas que se pueden usar en operaciones gráficas.

```csharp
public enum WmfPenStyle : short
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Cosmetic | `0` | El cosmético |
| EndcapRound | `0` | Los extremos de línea son redondos. |
| JoinRound | `0` | Las uniones de línea son redondas. |
| Solid | `0` | La pluma es sólida. |
| Dash | `1` | La pluma tiene guiones. |
| Dot | `2` | La pluma tiene puntos. |
| Dashdot | `3` | La pluma tiene guiones y puntos alternados. |
| Dashdotdot | `4` | El bolígrafo tiene guiones y puntos dobles. |
| Null | `5` | El bolígrafo es invisible. |
| Insideframe | `6` | La pluma es sólida. Cuando esta pluma se usa en cualquier registro de dibujo que tenga un rectángulo delimitador, las dimensiones de la figura se reducen para que quepa completamente en el rectángulo delimitador, teniendo en cuenta el ancho de la pluma. |
| Userstyle | `7` | El lápiz utiliza una matriz de estilos proporcionada por el usuario. |
| Alternate | `8` | El lápiz establece cada otro píxel (este estilo solo se aplica a los lápices cosméticos). |
| EndcapSquare | `256` | Las tapas de los extremos de las líneas son cuadradas. |
| EndcapFlat | `512` | Las tapas de los extremos de las líneas son planas. |
| JoinBevel | `4096` | Las uniones de línea están biseladas. |
| JoinMiter | `8192` | Las uniones de línea se miden cuando están dentro del límite actual establecido por el registro SETMITERLIMIT META_ESCAPE. Una unión se bisela cuando excedería el límite. |
