## DicomImage.InsertPage method

Inserta una nueva página en la lista de páginas en el índice especificado.

```csharp
public DicomPage InsertPage(int pageIndex)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| pageIndex | Int32 | Índice de la página. |

### Valor_devuelto

El recién creado[`DicomPage`](../../dicompage).

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentOutOfRangeException | *pageIndex* - El índice de la página es inferior a`0` . o *pageIndex* - El índice de la página es mayor que[`PageCount`](../pagecount) . |

### Ejemplos

Cree una imagen Dicom de varias páginas.

```csharp
[C#]

using (DicomImage image = (DicomImage)Image.Create(
        new DicomOptions() { Source = new StreamSource(new MemoryStream()) },
        100,
        100))
{
    // Dibujar algo usando gráficos vectoriales
    Graphics graphics = new Graphics(image);
    graphics.FillRectangle(new SolidBrush(Color.BlueViolet), image.Bounds);
    graphics.FillRectangle(new SolidBrush(Color.Aqua), 10, 20, 50, 20);
    graphics.FillEllipse(new SolidBrush(Color.Orange), 30, 50, 70, 30);

    // Guarda los píxeles de la imagen dibujada. Ahora están en la primera página de la imagen de Dicom.
    int[] pixels = image.LoadArgb32Pixels(image.Bounds);

    // Agrega algunas páginas después, haciéndolas más oscuras
    for (int i = 1; i < 5; i++)
    {
        DicomPage page = image.AddPage();
        page.SaveArgb32Pixels(page.Bounds, pixels);
        page.AdjustBrightness(i * 30);
    }

    // Agregue algunas páginas al frente de la página principal, haciéndolas más brillantes
    for (int i = 1; i < 5; i++)
    {
        DicomPage page = image.InsertPage(0);
        page.SaveArgb32Pixels(page.Bounds, pixels);
        page.AdjustBrightness(-i * 30);
    }

    // Guardar la imagen de varias páginas creada en el archivo de salida
    image.Save("MultiPage.dcm");
}
```
