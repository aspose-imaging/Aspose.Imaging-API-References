## BmpImage.FileFormat property

Obtiene un valor de formato de archivo

```csharp
public override FileFormat FileFormat { get; }
```

### Ejemplos

El siguiente ejemplo muestra cómo extraer información sobre el formato de datos sin procesar y el canal alfa de una imagen BMP.

```csharp
[C#]

// Crea una imagen BMP de 32 bpp de 100 x 100 px.
using (Aspose.Imaging.FileFormats.Bmp.BmpImage bmpImage = new Aspose.Imaging.FileFormats.Bmp.BmpImage(100, 100, 32, null))
{
    System.Console.WriteLine("FileFormat={0}, RawDataFormat={1}, HasAlpha={2}", bmpImage.FileFormat, bmpImage.RawDataFormat, bmpImage.HasAlpha);
};

// Crea una imagen BMP de 24 bpp de 100 x 100 px.
using (Aspose.Imaging.FileFormats.Bmp.BmpImage bmpImage = new Aspose.Imaging.FileFormats.Bmp.BmpImage(100, 100, 24, null))
{
    System.Console.WriteLine("FileFormat={0}, RawDataFormat={1}, HasAlpha={2}", bmpImage.FileFormat, bmpImage.RawDataFormat, bmpImage.HasAlpha);
};

// En general, BMP no es compatible con el canal alfa, por lo que la salida se verá así:
// FileFormat = Bmp, RawDataFormat = Rgb32Bpp, canales usados: 8,8,8,8, HasAlpha = False
// FileFormat = Bmp, RawDataFormat = Rgb24Bpp, canales usados: 8,8,8, HasAlpha = False
```
