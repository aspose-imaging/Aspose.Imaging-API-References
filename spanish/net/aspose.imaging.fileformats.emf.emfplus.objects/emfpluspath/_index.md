## EmfPlusPath class

El objeto EmfPlusPath especifica una serie de segmentos de línea y curva que forman una ruta de gráficos. El orden para los puntos de datos Bezier es el punto de inicio, el punto de control 1, el punto de control 2 y el punto final. Para obtener más información sobre , consulte [MSDN - DrawBeziers].
