## EmfPlusStringFormatData.CharRange property

Obtiene o establece una matriz opcional de objetos RangeCount EmfPlusCharacterRange que especifican el rango de posiciones de caracteres dentro de una cadena de texto. La región límite está definida por el área de la pantalla que está ocupada por un grupo de caracteres especificados por el rango de caracteres. Este campo DEBE estar presente si el valor del campo RangeCount en el objeto EmfPlusStringFormat es mayor que 0.

public EmfPlusCharacterRange[] CharRange { get; set; }
