## EmfPlusStringFormatData.TabStops property

Obtiene o establece una matriz opcional de valores de coma flotante que especifican las ubicaciones de tabulación opcionales para este objeto. Cada valor de tabulación representa el número de espacios entre tabulaciones o, para la primera tabulación, el número de espacios entre el comienzo de una línea de texto y la primera tabulación. Este campo DEBE estar presente si el valor del campo TabStopCount en el objeto EmfPlusStringFormat es mayor que 0.

public float[] TabStops { get; set; }
