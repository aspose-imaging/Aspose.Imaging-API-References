## EmfPlusLinearGradientBrushData.WrapMode property

Obtiene o establece el modo de ajuste.

```csharp
public EmfPlusWrapMode WrapMode { get; set; }
```

### El valor de la propiedad

Un entero de 32 bits con signo de la enumeración WrapMode (sección 2.1.1.34) que especifica si pintará el área fuera del límite del pincel. Al pintar fuera del límite, el modo de ajuste especifica cómo se repite el degradado de color.
