## EmfPlusLevelsEffect.MidTone property

Obtiene o establece Especifica cuánto aclarar u oscurecer los medios tonos de una imagen. Los valores del canal Color en el medio del rango de intensidad se alteran más que los valores cerca de los extremos alto o bajo, lo que significa que una imagen se puede aclarar u oscurecer sin perder el contraste entre las partes más oscuras y más claras de la imagen. -100 ≤ valor &lt; 0 Especifica que los tonos medios se oscurecen. 0 Especifica que los tonos medios NO DEBEN cambiar. 0 &lt; valor ≤ 100 Especifica que los tonos medios se vuelven más claros.

public int MidTone { get; set; }

El tono medio.
