## EmfPlusPathGradientBrushData.BoundaryData property

Obtiene o establece el límite del pincel de degradado de la ruta, que se especifica mediante una ruta o una spline cardinal cerrada. Si el indicador BrushDataPath está establecido en el campo BrushDataFlags, este campo DEBE contener un objeto EmfPlusBoundaryPathData (sección 2.2.2.6); de lo contrario, este campo DEBE contener un objeto EmfPlusBoundaryPointData (sección 2.2.2.7).
