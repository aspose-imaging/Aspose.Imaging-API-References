## EmfPlusColorCurveEffect.AdjustmentIntensity property

Obtiene o establece un entero de 32 bits con signo que especifica la intensidad del ajuste de la curva en el canal de color especificado por CurveChannel. Los rangos de valores significativos para este campo varían según el valor de CurveAdjustment, de la siguiente manera: Rango de ajuste de exposición: -255 ≤ valor &lt; 0 A medida que el valor disminuye, la exposición de la imagen DEBE disminuir. 0 Un valor de 0 especifica que la exposición NO DEBE cambiar. 0 &lt; valor ≤ 255 A medida que aumenta el valor, la exposición de la imagen DEBE aumentar. Rango de ajuste de densidad: -255 ≤ valor &lt; 0 A medida que disminuye el valor, la densidad de la imagen DEBE disminuir , lo que resulta en una imagen más oscura. 0 Un valor de 0 especifica que la densidad NO DEBE cambiar. 0 &lt; valor ≤ 255 A medida que aumenta el valor, la densidad de la imagen DEBE aumentar. Rango de ajuste de contraste: -100 ≤ valor &lt; 0 A medida que disminuye el valor, el contraste de la imagen DEBE disminuir. 0 Un valor de 0 especifica que el contraste NO DEBE cambiar. 0 &lt; valor ≤ 100 A medida que aumenta el valor, el contraste de la imagen DEBE aumentar e. Rango de ajuste de resaltado: -100 ≤ valor &lt; 0 A medida que el valor disminuye, las áreas claras de la imagen DEBERÍAN aparecer más oscuras. 0 Un valor de 0 especifica que el resaltado NO DEBE cambiar. 0 &lt; valor ≤ 100 Como el valor aumenta, las áreas claras de la imagen DEBERÍAN aparecer más claras. Rango de ajuste de sombra: -100 ≤ valor &lt; 0 A medida que el valor disminuye, las áreas oscuras de la imagen DEBERÍAN aparecer más oscuras. 0 Un valor de 0 especifica que el la sombra NO DEBE cambiar. 0 &lt; valor ≤ 100 A medida que aumenta el valor, las áreas oscuras de la imagen DEBEN aparecer más claras. Rango de ajuste de saturación de blanco: 0 — 255 A medida que aumenta el valor, el límite superior del rango del canal de color intensidades aumenta. Rango de ajuste de saturación de negro: 0 — 255 A medida que aumenta el valor, aumenta el límite inferior del rango de intensidades del canal de color.
