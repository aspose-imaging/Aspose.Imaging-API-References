### Observaciones

Las imágenes de mapa de bits se especifican mediante objetos EmfPlusBitmap (sección 2.2.2.2). Se realiza un efecto de matriz de color multiplicando un vector de color por un objeto ColorMatrixEffect. Una matriz de color de 5x5 puede realizar una transformación lineal, incluida la reflexión, la rotación, el corte o la escala, seguida de una traslación.
