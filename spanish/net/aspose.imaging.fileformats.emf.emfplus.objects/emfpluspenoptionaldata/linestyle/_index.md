## EmfPlusPenOptionalData.LineStyle property

Obtiene o establece un entero opcional de 32 bits con signo que especifica el estilo utilizado para las líneas dibujadas con este objeto de pluma. Este campo DEBE estar presente si el indicador PenDataLineStyle está establecido en el campo PenDataFlags del objeto EmfPlusPenData, y el valor DEBE estar definido en la enumeración LineStyle (sección 2.1.1.20).

public EmfPlusLineStyle LineStyle { get; set; }
