## EmfPlusPenOptionalData.CustomEndCapData property

Obtiene o establece el objeto opcional EmfPlusCustomEndCapData (sección 2.2.2.11) que define la forma personalizada de la tapa final, que es la forma que usará al final de una línea dibujada con esta pluma. Puede ser cualquiera de varias formas, como un cuadrado, un círculo o un diamante. Este campo DEBE estar presente si el indicador PenDataCustomEndCap es establecido en el campo PenDataFlags del objeto EmfPlusPenData

public EmfPlusCustomEndCapData CustomEndCapData { get; set; }
