## EmfPlusPenOptionalData.DashOffset property

Obtiene o establece un valor de punto flotante de 32 bits opcional que especifica la distancia desde el comienzo de una línea hasta el comienzo del primer espacio en un patrón de línea discontinua. Este campo DEBE estar presente si el indicador PenDataDashedLineOffset está establecido en el campo PenDataFlags del objeto EmfPlusPenData.

public float DashOffset { get; set; }
