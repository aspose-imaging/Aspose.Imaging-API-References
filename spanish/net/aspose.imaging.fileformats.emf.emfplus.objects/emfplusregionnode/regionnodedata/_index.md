## EmfPlusRegionNode.RegionNodeData property

Obtiene o establece datos opcionales de longitud variable que definen el objeto de datos node de la región especificado en el campo Tipo. El contenido y el formato de los datos pueden ser diferentes para cada tipo de nodo region . Este campo NO DEBE estar presente si el tipo node es RegionNodeDataTypeEmpty o RegionNodeDataTypeInfinite Este objeto es genérico y se usa para especificar diferentes tipos de datos de nodo de región, incluidos: Un objeto EmfPlusRegionNodePath (sección 2.2.2.42), para un nodo terminal; Un objeto EmfPlusRectF (sección 2.2.2.39), para un nodo terminal; and Un objeto EmfPlusRegionNodeChildNodes (sección 2.2.2.41), para un nodo no terminal.
