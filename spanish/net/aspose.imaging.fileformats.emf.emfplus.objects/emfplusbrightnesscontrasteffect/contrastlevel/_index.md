## EmfPlusBrightnessContrastEffect.ContrastLevel property

Obtiene o establece un entero de 32 bits con signo que especifica el nivel de contraste. Este valor DEBE estar en el rango de -100 a 100, con los siguientes efectos: -100 ≤ valor &lt; 0 A medida que el valor disminuye, el contraste de la imagen DEBE disminuir. 0 Un valor de 0 especifica que el contraste NO DEBE cambiar . 0 &lt; valor ≤ 100 A medida que aumenta el valor, el contraste de la imagen DEBE aumentar.
