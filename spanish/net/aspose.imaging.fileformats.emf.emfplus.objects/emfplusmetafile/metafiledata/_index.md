## EmfPlusMetafile.MetafileData property

Obtiene o establece datos de longitud variable que especifican el metarchivo incrustado. El contenido y el formato de los datos pueden ser diferentes para cada tipo de metarchivo.

```csharp
public byte[] MetafileData { get; set; }
```

### Observaciones

Las imágenes gráficas se especifican mediante objetos EmfPlusImage (sección 2.2.1.4). Un objeto EmfPlusMetafile DEBE estar presente en el campo ImageData de un objeto EmfPlusImage si se especifica ImageTypeMetafile en su campo Tipo. Este objeto es genérico y se utiliza para diferentes tipos de datos, incluidos: Un metarchivo WMF [MS-WMF]; Metarchivo WMF que se puede colocar; Un metarchivo EMF [MS-EMF]; Un metarchivo EMF+ que especifica operaciones gráficas solo con registros EMF+; and Un metarchivo EMF+ que especifica operaciones gráficas con registros EMF+ y EMF. Consulte la sección 2.2.2 para la especificación de objetos de estructura adicionales.
