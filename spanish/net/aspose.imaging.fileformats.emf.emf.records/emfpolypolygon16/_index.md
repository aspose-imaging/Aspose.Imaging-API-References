## EmfPolyPolygon16 class

El registro EMR_POLYPOLYGON16 especifica una serie de polígonos cerrados. Cada polígono se delinea con la pluma actual y se rellena con el pincel actual y el modo de relleno de polígono. Los polígonos dibujados por este registro pueden superponerse.
