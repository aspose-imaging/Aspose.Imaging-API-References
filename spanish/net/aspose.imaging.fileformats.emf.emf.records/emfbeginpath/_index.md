## EmfBeginPath class

Este registro abre un corchete de ruta en el contexto del dispositivo de reproducción actual. Después de abrir un corchete de ruta, una aplicación puede comenzar a procesar registros para definir los puntos que se encuentran en la ruta. Una aplicación DEBE cerrar un corchete de ruta abierto procesando EMR_ENDPATH record. Cuando una aplicación procesa el registro EMR_BEGINPATH, todas las rutas anteriores DEBEN descartarse del contexto del dispositivo de reproducción.
