## EmfCreateBrushIndirect class

El registro EMR_CREATEBRUSHINDIRECT define un pincel lógico para operaciones gráficas.

El objeto de pincel lógico definido por este registro se puede seleccionar en el contexto del dispositivo de reproducción mediante un registro EMR_SELECTOBJECT (sección 2.3.8.5), que especifica el pincel lógico que se usará en operaciones gráficas posteriores.

public sealed class EmfCreateBrushIndirect : EmfObjectCreationRecordType
