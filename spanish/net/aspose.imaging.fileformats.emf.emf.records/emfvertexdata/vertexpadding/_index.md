## EmfVertexData.VertexPadding property

Obtiene o establece una matriz opcional de longitud variable de nTri por cuatro bytes que DEBE estar presente si el valor del campo ulMode indica objetos GradientRectangle (sección 2.2.7). Si el valor del campo ulMode indica objetos GradientTriangle (sección 2.2.8), no hay VertexPadding presente. Este campo DEBE ser ignorado.

public byte[] VertexPadding { get; set; }
