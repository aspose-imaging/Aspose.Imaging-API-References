## EmfCommentPublicRecordType.CommentIdentifier property

Obtiene o establece un entero sin signo de 32 bits que identifica este registro de comentario como datos públicos especificados. El valor 0x43494447, que es la cadena ASCII "CIDG", identifica como un registro EMR_COMMENT_PUBLIC.
