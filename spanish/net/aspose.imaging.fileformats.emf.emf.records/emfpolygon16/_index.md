## EmfPolygon16 class

El registro EMR_POLYGON16 especifica un polígono que consta de dos o más vértices conectados por líneas rectas. El polígono se delinea con el lápiz actual y se rellena con el pincel actual y el modo de relleno de polígono. El polígono se cierra automáticamente dibujando una línea desde el último vértice hasta el primero.
