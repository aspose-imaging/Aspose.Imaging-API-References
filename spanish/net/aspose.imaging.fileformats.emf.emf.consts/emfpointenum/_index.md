## EmfPointEnum enumeration

La enumeración Punto se utiliza para especificar cómo se utilizará un punto en una llamada de dibujo.

```csharp
[Flags]
public enum EmfPointEnum : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| PT_CLOSEFIGURE | `1` | Un tipo PT_LINETO o PT_BEZIERTO se puede combinar con este valor usando el operador bit a bit O para indicar que el punto correspondiente es el último punto en una figura y la figura está cerrada |
| PT_LINETO | `2` | Especifica que se dibujará una línea desde la posición actual hasta este punto, que luego se convierte en la nueva posición actual |
| PT_BEZIERTO | `4` | Especifica que este punto es un punto de control o un punto final para una curva Bezier. |
| PT_MOVETO | `6` | Especifica que este punto inicia una figura disjunta. Este punto se convierte en la nueva posición actual. |
