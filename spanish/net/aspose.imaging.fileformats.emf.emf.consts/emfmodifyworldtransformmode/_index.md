## EmfModifyWorldTransformMode enumeration

La enumeración ModifyWorldTransformMode define modos para usar datos de transformación especificados para modificar la transformación de espacio mundial a espacio de página que está definida actualmente en el contexto del dispositivo de reproducción.

```csharp
public enum EmfModifyWorldTransformMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| MWT_IDENTITY | `1` | Restablece la transformación actual usando la matriz de identidad. En este modo, los datos de transformación especificados se ignoran |
| MWT_LEFTMULTIPLY | `2` | Multiplica la transformada actual. En este modo, los datos de transformación especificados son el multiplicando izquierdo y la transformación que está definida actualmente en el contexto del dispositivo de reproducción es el multiplicando derecho |
| MWT_RIGHTMULTIPLY | `3` | Multiplica la transformada actual. En este modo, los datos de transformación especificados son el multiplicando derecho, y la transformación que está definida actualmente en el contexto del dispositivo de reproducción es el multiplicando izquierdo |
| MWT_SET | `4` | Realiza la función de un registro EMR_SETWORLDTRANSFORM (sección 2.3.12.2). |
