## EmfText.Reference property

Obtiene o establece un objeto WMF PointL ([MS-WMF] sección 2.2.2.15) que especifica las coordenadas del punto de referencia utilizado para colocar la cadena. El punto de referencia está definido por el último registro EMR_SETTEXTALIGN (sección 2.3.11.25). Si no se ha establecido dicho registro, la alineación predeterminada es TA_LEFT,TA_TOP.
