## EmfFormat.Version property

Obtiene o establece un entero sin signo de 32 bits que especifica el número de versión del formato. Si el campo Firma especifica PostScript encapsulado (EPS), este valor DEBE ser 0x00000001; de lo contrario, este valor DEBE ser ignorado
