Obtiene o establece las compensaciones.

```csharp
public float[] Offsets { get; set; }
```

### El valor de la propiedad

La matriz de compensaciones de color degradado. La longitud de la matriz debe ser mayor o igual a 2. El primer valor debe ser igual a 0, el último valor debe ser igual a 1, el otro valor debe estar en el rango (0,1)
