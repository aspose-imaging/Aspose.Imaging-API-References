## EmfExtTextOutOptions enumeration

Перечисление ExtTextOutOptions определяет параметры, управляющие различными аспектами вывода текста в записях EMR_SMALLTEXTOUT (раздел 2.3.5.37) и в объектах EmrText.

```csharp
[Flags]
public enum EmfExtTextOutOptions
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| ETO_OPAQUE | `2` | Этот бит указывает, что текущий цвет фона ДОЛЖЕН использоваться для заливки прямоугольника |
| ETO_CLIPPED | `4` | Этот бит указывает, что текст ДОЛЖЕН быть обрезан до прямоугольника. |
| ETO_GLYPH_INDEX | `10` | Этот бит указывает, что коды символов в выходной текстовой строке фактически являются индексами глифов символов в шрифте TrueType. Индексы глифов зависят от шрифта, поэтому для отображения правильных символов при воспроизведении используемый шрифт ДОЛЖЕН быть идентичным шрифту, используемому для создания индексов. |
| ETO_RTLREADING | `80` | Этот бит указывает, что текст ДОЛЖЕН располагаться в порядке чтения справа налево, вместо по умолчанию слева направо заказ. Это СЛЕДУЕТ применять только тогда, когда шрифт , выбранный в контексте устройства воспроизведения, является либо ивритом, либо арабским |
| ETO_NO_RECT | `100` | Этот бит указывает, что запись не определяет ограничивающий прямоугольник для вывода текста. |
| ETO_SMALL_CHARS | `200` | Этот бит указывает, что коды символов в выходной текстовой строке являются 8-битными, производными от младших байтов 16-битного Unicode UTF16. -LE коды символов, в которых старший байт считается равным 0. |
| ETO_NUMERICSLOCAL | `400` | Этот бит указывает, что для отображения чисел СЛЕДУЕТ использовать цифры, соответствующие локали |
| ETO_NUMERICSLATIN | `800` | Этот бит указывает, что для отображения чисел СЛЕДУЕТ использовать европейские цифры |
| ETO_IGNORELANGUAGE | `1000` | Этот бит указывает, что никакая специальная обработка операционной системой для размещения глифов не должна выполняться для строк справа налево; то есть, все позиционирование глифов ДОЛЖНО быть обеспечено записями рисования и состояния в метафайле |
| ETO_PDY | `2000` | Этот бит указывает, что ДОЛЖНЫ быть указаны как значения горизонтального, так и вертикального смещения символов |
| ETO_REVERSE_INDEX_MAP | `10000` | Этот бит зарезервирован и НЕ ДОЛЖЕН использоваться |
