## EmfExtTextOutOptions enumeration

Перечисление ExtTextOutOptions определяет параметры, управляющие различными аспектами вывода текста в записях EMR_SMALLTEXTOUT (раздел 2.3.5.37) и в объектах EmrText.

```csharp
[Flags]
public enum EmfExtTextOutOptions
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| ETO_OPAQUE | `2` | Этот бит указывает, что текущий цвет фона ДОЛЖЕН использоваться для заполнения прямоугольника |
| ETO_CLIPPED | `4` | Этот бит указывает, что текст ДОЛЖЕН быть обрезан до прямоугольника. |
| ETO_GLYPH_INDEX | `10` | Этот бит указывает, что коды символов в выходной текстовой строке на самом деле являются индексами глифов символов в шрифте TrueType. Индексы глифов зависят от шрифта, , поэтому для отображения правильных символов при воспроизведении используемый шрифт ДОЛЖЕН быть идентичен шрифту, используемому для создания индексов. |
| ETO_RTLREADING | `80` | Этот бит указывает, что текст ДОЛЖЕН быть расположен в порядке чтения справа налево, вместо порядка слева направо по умолчанию. Это СЛЕДУЕТ применять только тогда, когда font , выбранный в контексте устройства воспроизведения, является либо ивритом, либо арабским . |
| ETO_NO_RECT | `100` | Этот бит указывает, что в записи не указан ограничивающий прямоугольник для вывода текста. |
| ETO_SMALL_CHARS | `200` | Этот бит указывает, что коды символов в выходной текстовой строке являются 8-битными, получены из младших байтов 16-битных кодов символов Unicode UTF16-LE, , в которых предполагается, что старший байт равен 0. |
| ETO_NUMERICSLOCAL | `400` | Этот бит указывает, что для отображения чисел СЛЕДУЕТ использовать цифры, соответствующие локали |
| ETO_NUMERICSLATIN | `800` | Этот бит указывает, что для отображения чисел СЛЕДУЕТ использовать европейские цифры |
| ETO_IGNORELANGUAGE | `1000` | Этот бит указывает, что никакая специальная обработка операционной системой для размещения глифов не должна выполняться для строк с направлением справа налево; то есть все позиционирование глифов ДОЛЖНО быть обеспечено рисунком и записями состояния в метафайле |
| ETO_PDY | `2000` | Этот бит указывает, что ДОЛЖНЫ быть указаны как значения смещения символов по горизонтали, так и по вертикали |
| ETO_REVERSE_INDEX_MAP | `10000` | Этот бит зарезервирован и НЕ ДОЛЖЕН использоваться |
