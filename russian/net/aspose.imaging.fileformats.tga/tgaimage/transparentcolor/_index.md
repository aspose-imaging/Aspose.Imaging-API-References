## TgaImage.TransparentColor property

Получает или задает цвет ключа.

Ключевой цвет можно рассматривать как «фоновый цвет» или «прозрачный цвет». Это цвет "неизображения" области экрана, и тот же цвет, в который будет очищен экран , если его стереть в приложении.

```csharp
public override Color TransparentColor { get; set; }
```

    gammaValueNumerator = image.GammaValueNumerator;
    hasAlphaChannel = image.HasAlpha;

    height = image.Height;
