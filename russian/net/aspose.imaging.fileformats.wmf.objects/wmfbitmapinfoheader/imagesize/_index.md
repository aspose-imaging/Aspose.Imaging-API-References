## WmfBitmapInfoHeader.ImageSize property

Получает или задает 32-разрядное целое число без знака, определяющее размер изображения в байтах. Если значение Compression равно BI_RGB, это значение ДОЛЖНО быть равно нулю и ДОЛЖНО игнорироваться. Если значение Compression равно BI_JPEG или BI_PNG, это значение ДОЛЖНО указывать размер буфера изображения JPEG или PNG, соответственно.

public int ImageSize { get; set; }
