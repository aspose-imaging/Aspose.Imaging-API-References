## PdfCoreOptions.PdfCompliance property

Получает или задает соответствие PDF.

```csharp
public PdfComplianceVersion PdfCompliance { get; set; }
```

### Стоимость имущества

Соответствие PDF.

### Примеры

Преобразование изображения EPS в PDF с использованием рендеринга PostScript.

```csharp
[C#]

using (var image = (EpsImage)Image.Load("Sample.eps"))
{
    var options = new PdfOptions
    {
        PdfCoreOptions = new PdfCoreOptions
        {
            PdfCompliance = PdfComplianceVersion.PdfA1b  // Установить требуемое соответствие PDF
        }
    };
  
    image.Save("Sample.pdf", options);
}
```
