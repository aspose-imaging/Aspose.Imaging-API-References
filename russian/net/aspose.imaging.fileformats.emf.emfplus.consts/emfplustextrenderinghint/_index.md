## EmfPlusTextRenderingHint enumeration

Перечисление TextRenderingHint определяет типы хинтинга текста и сглаживания, влияющие на качество рендеринга текста.

```csharp
public enum EmfPlusTextRenderingHint : byte
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| TextRenderingHintSystemDefault | `0` | Указывает, что каждый текстовый символ ДОЛЖЕН быть нарисован с использованием любых параметров сглаживания шрифтов, настроенных в операционной системе. |
| TextRenderingHintSingleBitPerPixelGridFit | `1` | Указывает, что каждый текстовый символ ДОЛЖЕН быть нарисован с использованием его растрового изображения глифа. Сглаживание МОЖЕТ использоваться для улучшения внешнего вида основ и кривизны глифов символов. |
| TextRenderingHintSingleBitPerPixel | `2` | Указывает, что каждый текстовый символ ДОЛЖЕН быть нарисован с использованием его растрового изображения глифа. Сглаживание не используется. |
| TextRenderingHintAntialiasGridFit | `3` | Указывает, что каждый текстовый символ ДОЛЖЕН быть нарисован с использованием сглаженного растрового изображения глифа со сглаживанием. Рендеринг высокого качества из-за сглаживания, но за счет более высокой производительности. |
| TextRenderingHintAntialias | `4` | Указывает, что каждый текстовый символ рисуется с использованием своего сглаженного растрового изображения глифа без подсказок. Лучшее качество достигается за счет сглаживания, но различия в ширине основы МОГУТ быть заметны, поскольку хинтинг отключен. |
| TextRenderingHintClearTypeGridFit | `5` | Указывает, что каждый текстовый символ ДОЛЖЕН быть нарисован с использованием его растрового изображения глифа ClearType со сглаживанием. Это параметр текстовых подсказок самого высокого качества, который используется для использования возможностей шрифта ClearType. |
