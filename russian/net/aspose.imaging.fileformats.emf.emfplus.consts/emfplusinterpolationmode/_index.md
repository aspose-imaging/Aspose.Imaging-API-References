## EmfPlusInterpolationMode enumeration

Перечисление InterpolationMode определяет способы выполнения масштабирования, включая растяжение и сжатие.

```csharp
public enum EmfPlusInterpolationMode : byte
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| InterpolationModeDefault | `0` | Указывает режим интерполяции по умолчанию, который определяется как InterpolationModeBilinear. |
| InterpolationModeLowQuality | `1` | Указывает режим интерполяции низкого качества, который определяется как InterpolationModeNearestNeighbor. |
| InterpolationModeHighQuality | `2` | Указывает режим интерполяции высокого качества, который определяется как InterpolationModeHighQualityBicubic. |
| InterpolationModeBilinear | `3` | Определяет билинейную интерполяцию, которая использует ближайшую окрестность 2x2 известных пикселей, окружающих интерполируемый пиксель. Средневзвешенное значение этих 4 известных значений пикселей определяет значение, присваиваемое интерполированному пикселю. Результат выглядит более плавным, чем InterpolationModeNearestNeighbor. |
| InterpolationModeBicubic | `4` | Задает бикубическую интерполяцию, которая использует ближайшую окрестность 4x4 известных пикселей, окружающих интерполируемый пиксель. Средневзвешенное значение этих 16 известных значений пикселей определяет значение, присваиваемое интерполированному пикселю. Поскольку известные пиксели, вероятно, будут находиться на разных расстояниях от интерполированного пикселя, более близким пикселям при расчете присваивается более высокий вес. Результат выглядит более плавным, чем InterpolationModeBilinear. |
| InterpolationModeNearestNeighbor | `5` | Задает интерполяцию ближайшего соседа, которая использует только значение пикселя, наиболее близкого к интерполируемому пикселю. Этот режим просто дублирует или удаляет пиксели, что приводит к результату самого низкого качества среди всех этих вариантов. |
| InterpolationModeHighQualityBilinear | `6` | Задает билинейную интерполяцию с предварительной фильтрацией. |
| InterpolationModeHighQualityBicubic | `7` | Задает бикубическую интерполяцию с предварительной фильтрацией, которая дает результат наивысшего качества среди этих опций. |
