## EmfPlusLineCapType enumeration

Перечисление LineCapType определяет типы заглавных букв для использования на концах линий, нарисованных графическими перьями.

```csharp
public enum EmfPlusLineCapType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| LineCapTypeFlat | `0` | Определяет квадратный конец строки. Конец строки ДОЛЖЕН быть последней точкой в строке. |
| LineCapTypeSquare | `1` | Определяет окончание квадратной линии. Центр квадрата ДОЛЖЕН находиться в последней точке линии. Ширина квадрата равна ширине линии. |
| LineCapTypeRound | `2` | Определяет конец круговой линии. Центр круга ДОЛЖЕН находиться в последней точке линии. Диаметр круга равен ширине линии. |
| LineCapTypeTriangle | `3` | Задает треугольный конец строки. Основание треугольника ДОЛЖНО находиться в последней точке линии. Основанием треугольника является ширина линии. |
| LineCapTypeNoAnchor | `16` | Указывает, что конец строки не привязан. |
| LineCapTypeSquareAnchor | `17` | Указывает, что конец строки закрепляется квадратной заглушкой. Центр квадрата ДОЛЖЕН находиться в последней точке линии. Высота и ширина квадрата - ширина линии. |
| LineCapTypeRoundAnchor | `18` | Указывает, что конец строки закреплен закругленной линией. Центр круга ДОЛЖЕН находиться в последней точке линии. Круг ДОЛЖЕН быть шире линии. |
| LineCapTypeDiamondAnchor | `19` | Указывает, что конец линии закреплен ромбовидной заглушкой линии, которая представляет собой квадрат, повернутый на 45 градусов. Центр ромба ДОЛЖЕН быть расположен в последней точке линии. Ромб ДОЛЖЕН быть шире линии. |
| LineCapTypeArrowAnchor | `20` | Указывает, что конец линии привязан к форме стрелки. Точка стрелки ДОЛЖНА располагаться в последней точке линии. Стрелка ДОЛЖНА быть шире линии. |
| LineCapTypeAnchorMask | `240` | Маска, используемая для проверки того, является ли заглушка строки якорной заглушкой. |
| LineCapTypeCustom | `255` | Указывает пользовательский заголовок строки. |
