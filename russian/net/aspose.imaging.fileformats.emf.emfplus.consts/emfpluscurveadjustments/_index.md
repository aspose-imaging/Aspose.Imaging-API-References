## EmfPlusCurveAdjustments enumeration

Перечисление CurveAdjustments определяет настройки, которые можно применить к цветовой кривой изображения.

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |

| AdjustDensity | `1` | Определяет симуляцию увеличения или уменьшения плотности изображения. |

| AdjustHighlight | `3` | Задает увеличение или уменьшение значения цветового канала изображения, если этот канал уже имеет значение, превышающее половину интенсивности. Эту регулировку можно использовать для увеличения четкости в светлых областях изображения, не затрагивая темные области. |
| AdjustShadow | `4` | Задает увеличение или уменьшение значения цветового канала изображения, если этот канал уже имеет значение ниже половины интенсивности. Эту регулировку можно использовать для увеличения четкости в темных областях изображения, не затрагивая светлые области. |
| AdjustMidtone | `5` | Определяет настройку, которая делает изображение светлее или темнее. Эту регулировку можно использовать для осветления или затемнения изображения без потери контраста между самой темной и самой светлой частями изображения. |
| AdjustWhiteSaturation | `6` | Задает настройку насыщенности белого цвета изображения, определяемую как максимальное значение в диапазоне интенсивностей для данного цветового канала, диапазон которого обычно составляет от 0 до 255. |
| AdjustBlackSaturation | `7` | Определяет настройку насыщенности черного изображения, которая является минимальным значением в диапазоне интенсивностей для данного цветового канала, обычно от 0 до 255. |
