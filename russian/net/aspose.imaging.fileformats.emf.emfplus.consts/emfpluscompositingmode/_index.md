## EmfPlusCompositingMode enumeration

Перечисление CompositingMode определяет режимы для комбинирования исходных цветов с цветами фона. Режим композитинга представляет собой включенное состояние альфа-смешивания.

```csharp
public enum EmfPlusCompositingMode : byte
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| CompositingModeSourceOver | `0` | Включает альфа-смешивание, которое указывает, что при визуализации цвет смешивается с цветом фона. Степень смешивания определяется значением альфа-компонента отображаемого цвета. |
| CompositingModeSourceCopy | `1` | Отключает альфа-смешивание, что означает, что при рендеринге исходного цвета он перезаписывает цвет фона. |
