## EmfPlusPathPointFlags enumeration

32-битное целое число без знака, указывающее, как интерпретировать точки и связанные типы точек, определенные этим объектом. C (1 бит):Если установлено, массив PathPoints указывает абсолютные местоположения в координатном пространстве с 16-битными целочисленными координатами. Если этот параметр не установлен, массив PathPoints указывает абсолютные местоположения в координатном пространстве с 32-битными координатами с плавающей запятой. Примечание. Если флаг P (ниже) установлен, этот флаг МОЖЕТ быть очищен и ДОЛЖЕН игнорироваться. R (1 бит):если установлено, типы точек в массиве PathPointTypes определяются объектами EmfPlusPathPointTypeRle (раздел 2.2.2.32), которые используют кодирование длин серий (RLE ) сжатие и/или объекты EmfPlusPathPointType (раздел 2.2.2.31). См. раздел 3.1.6 [MS-WMF] для получения дополнительной информации о сжатии RLE. Если этот параметр не установлен, типы точек в массиве PathPointTypes задаются объектами EmfPlusPathPointType. P (1 бит):если установлено, каждый элемент в массиве PathPoints указывает местоположение в координатном пространстве, которое относится к местоположению, указанному предыдущим элементом в массиве. . В случае первого элемента в PathPoints предполагается предыдущее местоположение с координатами (0,0). Если этот параметр не установлен, каждый элемент в массиве PathPoints указывает абсолютное местоположение.
