## EmfPlusStringAlignment enumeration

Перечисление StringAlignment определяет способы выравнивания строк относительно прямоугольника текстового макета.

```csharp
public enum EmfPlusStringAlignment
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| StringAlignmentNear | `0` | Указывает, что строка выравнивается по направлению к исходной точке прямоугольника макета. Это можно использовать для выравнивания символов вдоль строки или для выравнивания текста внутри прямоугольника. Для прямоугольника макета с направлением справа налево начало координат ДОЛЖНО быть справа вверху. |
| StringAlignmentCenter | `1` | Указывает, что выравнивание выполняется по центру между исходной точкой и экстентом прямоугольника макета. |
| StringAlignmentFar | `2` | Указывает, что выравнивание выполняется по правой стороне прямоугольника макета. |
