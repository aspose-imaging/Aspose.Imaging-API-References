## EmfPlusFilterType enumeration

Перечисление FilterType определяет типы алгоритмов фильтрации, которые можно использовать для улучшения качества текста и графики и рендеринга изображений.

```csharp
public enum EmfPlusFilterType : byte
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| FilterTypeNone | `0` | Указывает, что фильтрация не выполняется. |
| FilterTypePoint | `1` | Указывает, что каждый целевой пиксель вычисляется путем выборки ближайшего пикселя из исходного изображения. |
| FilterTypeLinear | `2` | Указывает, что линейная интерполяция выполняется с использованием средневзвешенного значения области 2x2 пикселей, окружающих исходный пиксель. |
| FilterTypeTriangle | `3` | Указывает, что каждый пиксель исходного изображения в равной степени влияет на целевое изображение. Это самый медленный из алгоритмов фильтрации. |
| FilterTypeBox | `4` | Задает алгоритм блочного фильтра, в котором каждый целевой пиксель вычисляется путем усреднения прямоугольника исходных пикселей. Этот алгоритм полезен только при уменьшении размера изображения. |
| FilterTypePyramidalQuad | `6` | Указывает, что используется палаточный фильтр с 4 сэмплами. |
| FilterTypeGaussianQuad | `7` | Указывает, что используется фильтр Гаусса с 4 выборками, который создает эффект размытия изображения. |
