## EmfPlusDriverStringOptionsFlags enumeration

Флаги DriverStringOptions задают свойства позиционирования и рендеринга графического текста. Эти флаги можно комбинировать для указания нескольких параметров.

```csharp
[Flags]
public enum EmfPlusDriverStringOptionsFlags
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| DriverStringOptionsCmapLookup | `1` | Если установлено, положения глифов символов СЛЕДУЕТ указывать в таблице поиска карты символов. Если флажок снят, позиции глифов СЛЕДУЕТ получать из массива координат. |
| DriverStringOptionsVertical | `2` | Если установлено, строка ДОЛЖНА отображаться вертикально. Если флажок установлен, строка ДОЛЖНА отображаться горизонтально. |
| DriverStringOptionsRealizedAdvance | `4` | Если установлено, позиции глифов СЛЕДУЕТ вычислять относительно позиции первого глифа. Если флажок снят, позиции глифов СЛЕДУЕТ получать из массива координат. |
| DriverStringOptionsLimitSubpixel | `8` | Если установлено, меньше памяти СЛЕДУЕТ использовать для кэширования сглаженных глифов, что приводит к более низкому качеству рендеринга текста. Если этот параметр не установлен, СЛЕДУЕТ использовать больше памяти, что обеспечивает более высокое качество рендеринга текста. |
