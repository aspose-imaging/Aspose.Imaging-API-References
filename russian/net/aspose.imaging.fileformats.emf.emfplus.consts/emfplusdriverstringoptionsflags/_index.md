## EmfPlusDriverStringOptionsFlags enumeration

Флаги DriverStringOptions определяют свойства позиционирования и рендеринга графического текста. Эти флаги можно комбинировать для указания нескольких параметров.

```csharp
[Flags]
public enum EmfPlusDriverStringOptionsFlags
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| DriverStringOptionsCmapLookup | `1` | Если установлено, положения глифов СЛЕДУЕТ указывать в таблице поиска карты символов. Если флажок не установлен, положения глифов СЛЕДУЕТ получать из массива координат. |
| DriverStringOptionsVertical | `2` | Если установлено, строка ДОЛЖНА отображаться вертикально. Если снято, строка ДОЛЖНА отображаться горизонтально. |
| DriverStringOptionsRealizedAdvance | `4` | Если установлено, позиции глифов ДОЛЖНЫ быть рассчитаны относительно позиции первого глифа. Если флажок не установлен, позиции глифов ДОЛЖНЫ быть получены из массива координат. |
| DriverStringOptionsLimitSubpixel | `8` | Если установлено, для кэширования сглаженных глифов СЛЕДУЕТ использовать меньше памяти, что приводит к более низкому качеству рендеринга текста. Если этот флажок установлен, СЛЕДУЕТ использовать больше памяти, что обеспечивает более высокое качество рендеринга текста. |

### Примечания

Вывод графического текста указан в[`EmfPlusDrawDriverString`](../../aspose.imaging.fileformats.emf.emfplus.records/emfplusdrawdriverstring) записи
