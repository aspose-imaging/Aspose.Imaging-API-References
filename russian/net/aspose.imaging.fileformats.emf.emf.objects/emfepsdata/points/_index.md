## EmfEpsData.Points property

Получает или задает массив из трех объектов Point28_4 (раздел 2.2.23), который определяет координаты выходного параллелограмма с использованием 28,4-битной нотации FIX

```csharp
public EmfPoint28To4[] Points { get; set; }
```

### Примечания

Верхний левый угол параллелограмма является первой точкой в этом массиве, верхний правый угол является второй точкой, а нижний левый угол является третьей точкой. Нижний правый угол параллелограмма вычисляется из первых трех точек (A, B и C), рассматривая их как векторов.
