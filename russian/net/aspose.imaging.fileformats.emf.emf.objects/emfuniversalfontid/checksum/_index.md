## EmfUniversalFontId.Checksum property

Получает или задает 32-разрядное целое число без знака, которое является контрольной суммой шрифта. Значение контрольной суммы имеет следующие значения. 0x00000000 Объект является шрифтом устройства. 0x00000001 Объект представляет собой шрифт Type 1, который был установлен на клиентском компьютере и перечисляется драйвером принтера PostScript как шрифт устройства. 0x00000002 Объект не является шрифтом, а является растеризатором типа 1. 3 ≤ значение Объект представляет собой растровый, векторный шрифт, шрифт TrueType или растеризованный шрифт Type 1, который был создан растеризатором Type 1.
