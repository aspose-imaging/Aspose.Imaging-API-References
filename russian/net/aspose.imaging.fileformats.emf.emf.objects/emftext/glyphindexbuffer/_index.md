## EmfText.GlyphIndexBuffer property

Получает или задает необязательный буфер индекса глифа. Если options имеет флаг ETO_GLYPH_INDEX, то коды символов в выходной текстовой строке фактически являются индексами глифов символов в шрифте TrueType (перечисление 2.1.11 ExtTextOutOptions). Индексы глифов зависят от шрифта, поэтому для отображения правильных символов при воспроизведении используемый шрифт ДОЛЖЕН быть идентичен шрифту, используемому для создания индексов.

public int[] GlyphIndexBuffer { get; set; }
