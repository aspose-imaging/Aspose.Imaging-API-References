## EmfLogPenEx.StyleEntry property

Получает или задает необязательный массив 32-битных целых чисел без знака, который определяет длину тире и пробелов в линии, нарисованной этим пером, когда значением PenStyle является стиль линии PS_USERSTYLE для пера. Массив содержит ряд элементов , указанных NumStyleEntries, но он используется так, как если бы он повторялся бесконечно Первый элемент массива указывает длину первого тире. Вторая запись указывает длину первого пробела. После этого длины штрихов и пробелов чередуются. Если тип пера в поле PenStyle — PS_GEOMETRIC, длины указываются в логических единицах; в противном случае длина указывается в единицах устройства.

```csharp
public int[] StyleEntry { get; set; }
```
