## EmfLogPenEx.Width property

Получает или задает 32-разрядное целое число без знака, указывающее ширину линии, нарисованной пером. Если тип пера в поле PenStyle — PS_GEOMETRIC, это значение равно ширине в логических единицах; в противном случае ширина указывается в единицах устройства. Если тип пера в поле PenStyle — PS_COSMETIC, это значение ДОЛЖНО быть 0x00000001.

public int Width { get; set; }
