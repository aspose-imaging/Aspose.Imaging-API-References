## PsdOptions.RemoveGlobalTextEngineResource property

Получает или задает значение, указывающее, - Удалить ли глобальный ресурс текстового движка - Используется для некоторых текстовых файлов psd, только в том случае, когда их нельзя открыть в Adobe Photoshop после обработки (в основном для отсутствующих шрифтов, связанных с текстовыми слоями). После использования этой опции пользователю необходимо сделать следующее в открытом в Photoshop файле:Меню "Текст" -&gt;&gt; «Обработать отсутствующие шрифты». После этой операции весь текст появится снова. Обратите внимание, что эта операция может привести к некоторым окончательным изменениям макета.

```csharp
public bool RemoveGlobalTextEngineResource { get; set; }
```

### Стоимость имущества

` true` if [удалить ресурс глобального текстового движка]; в противном случае` false` .
