## EmfPlusObject class

Запись EmfPlusObject определяет объект для использования в графических операциях. Определение объекта может охватывать несколько записей, на что указывает значение поля Флаги.

public sealed class EmfPlusObject : EmfPlusObjectRecordType

### Примечания

Запись EmfPlusObject является универсальной; используется для всех типов объектов. Значения, относящиеся к определенным типам объектов , содержатся в поле ObjectData. Концептуальная модель для управления графическими объектами описана в разделе «Управление графическими объектами» (раздел 3.1.2).
