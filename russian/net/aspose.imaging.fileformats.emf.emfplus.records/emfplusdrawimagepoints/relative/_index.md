## EmfPlusDrawImagePoints.Relative property

Получает или задает значение, указывающее, является ли это[`EmfPlusDrawImagePoints`](../../emfplusdrawimagepoints)относительным. Этот бит указывает, указывает ли поле PointData относительное или абсолютное местоположение. Если установлено, каждый элемент в PointData указывает положение в пространстве координат, которое относительно положения, указанного предыдущим элементом в массиве. В случае с первым элементом в PointData предполагается предыдущее местоположение с координатами (0,0). Если флажок не установлен, PointData указывает абсолютные местоположения в соответствии с флагом C. Примечание. Если этот флаг установлен, флаг C (выше) не определен и ДОЛЖЕН игнорироваться.

```csharp
public bool Relative { get; set; }
```

### Стоимость имущества

` true` если относительно; в противном случае` false` .
