## EmfPlusDrawImagePoints.Compressed property

Получает или задает значение, указывающее, сжат ли PointData. Этот бит указывает, указывает ли поле PointData сжатые данные. Если установлено, PointData указывает абсолютные местоположения в координатном пространстве с 16-битным целым числом координат. Если этот параметр не установлен, PointData указывает абсолютные местоположения в координатном пространстве с 32-битными координатами с плавающей запятой. Примечание. Если флаг P (ниже) установлен, этот флаг не определен и ДОЛЖЕН игнорироваться.

public bool Compressed { get; set; }
