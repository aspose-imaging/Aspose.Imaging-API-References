## EmfPlusDrawArc.Size property

Получает или задает размер. 32-разрядное целое число без знака, указывающее выровненное по 32 битам количество байтов во всей записи, включая 12-байтовый заголовок записи и данные, относящиеся к записи. Для этого типа записи значение ДОЛЖНО быть одним из следующих: 0x0000001C Если бит C установлен в поле Flags. 0x00000024 Если бит C очищен в поле Flags
