## EmfPlusDrawArc.DataSize property

Получает или задает размер данных. 32-разрядное целое число без знака, указывающее 32-разрядное выровненное число байтов следующих за ним данных, относящихся к записи. Для этого типа записи значение ДОЛЖНО быть одним из следующих: 0x00000010 Если бит C установлен в поле Flags. 0x00000018 Если бит C в поле Flags очищен.
