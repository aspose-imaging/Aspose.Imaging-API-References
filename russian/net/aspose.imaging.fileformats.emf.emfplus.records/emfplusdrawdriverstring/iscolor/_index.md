## EmfPlusDrawDriverString.IsColor property

Получает или задает значение, указывающее, является ли этот экземпляр цветом. Этот бит указывает тип данных в поле BrushId. Если установлено, BrushId определяет значение цвета в EmfPlusARGB object (раздел 2.2.2.1). Если флажок не установлен, BrushId содержит табличный индекс EMF+ Object объекта EmfPlusBrush (раздел 2.2.1.1).
