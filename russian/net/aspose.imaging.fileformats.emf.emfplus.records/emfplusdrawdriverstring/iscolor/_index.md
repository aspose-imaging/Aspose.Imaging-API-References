## EmfPlusDrawDriverString.IsColor property

Получает или задает значение, указывающее, является ли данный экземпляр цветным. Этот бит указывает тип данных в поле BrushId. Если установлено, BrushId определяет значение цвета в объекте EmfPlusARGB (раздел 2.2.2.1). Если флажок не установлен, BrushId содержит индекс EMF+ Object Table объекта EmfPlusBrush (раздел 2.2.1.1).

public bool IsColor { get; set; }
