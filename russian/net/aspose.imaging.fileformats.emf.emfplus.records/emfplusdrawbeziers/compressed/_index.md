## EmfPlusDrawBeziers.Compressed property

Получает или задает значение, указывающее, сжат ли PointData. Если установлено, PointData указывает абсолютные местоположения в координатном пространстве с 16-битными целочисленными координатами. Если флажок не установлен, PointData указывает абсолютные местоположения в пространстве координат с 32-битными координатами с плавающей запятой. Примечание. Если установлен флаг Relative (ниже), этот флаг не определен и ДОЛЖЕН игнорироваться.

```csharp
public bool Compressed { get; set; }
```
