## RasterImage.BinarizeOtsu method

Бинаризация изображения с пороговым значением Otsu

```csharp

```

### Примеры

В следующем примере выполняется бинаризация растрового изображения с пороговым значением Otsu. Бинаризованные изображения содержат только 2 цвета — черный и белый.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.png"))
{
    Aspose.Imaging.RasterImage rasterImage = (Aspose.Imaging.RasterImage)image;

    // Бинаризация изображения с пороговым значением Otsu.
    rasterImage.BinarizeOtsu();
    rasterImage.Save(dir + "sample.BinarizeOtsu.png");
}
```
