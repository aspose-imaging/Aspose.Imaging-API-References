## FontSettings.GetFontsFolders method

Получает копию массива, содержащего список папок, в которых Aspose.Words ищет шрифты TrueType.

```csharp
public static string[] GetFontsFolders()
```

### Возвращаемое значение

Копия текущего местоположения шрифта.

### Примечания

Возвращаемое значение является копией данных, которые использует Aspose.Words. Если вы измените записи в возвращаемом массиве, это не повлияет на визуализацию документа. Чтобы указать расположение новых шрифтов, используйте метод[`SetFontsFolders`](../setfontsfolders).
