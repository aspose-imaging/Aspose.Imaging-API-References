## FromArgb(int, int, int, int) {#fromargb_3}

Создает[`Color`](../../color) структура из четырех значений компонентов ARGB (альфа, красный, зеленый и синий). Хотя этот метод позволяет передавать 32-битное значение для каждого компонента, значение каждого компонента ограничено 8 битами.

```csharp
public static Color FromArgb(int alpha, int red, int green, int blue)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| alpha | Int32 | Альфа компонент. Допустимые значения: от 0 до 255. |
| red | Int32 | Красная составляющая. Допустимые значения: от 0 до 255. |
| green | Int32 | Зеленый компонент. Допустимые значения: от 0 до 255. |
| blue | Int32 | Синяя составляющая. Допустимые значения: от 0 до 255. |

### Возвращаемое значение

[`Color`](../../color) что создает этот метод.

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentOutOfRangeException | *alpha* ,*red* ,*green* , или же*blue* меньше 0 или больше 255. |
