Указывает качество рендеринга текста.

public enum TextRenderingHint

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| SystemDefault | `0` | Каждый символ рисуется с использованием своего растрового изображения глифа с системной подсказкой по умолчанию. Текст будет отображаться с использованием любых настроек сглаживания шрифтов, выбранных пользователем для системы. |
| SingleBitPerPixelGridFit | `1` | Каждый символ рисуется с использованием своего растрового изображения глифа. Подсказка используется для улучшения внешнего вида персонажа на стеблях и кривизне. |
| SingleBitPerPixel | `2` | Каждый символ рисуется с использованием своего растрового изображения глифа. Подсказка не используется. |
| AntiAliasGridFit | `3` | Каждый символ рисуется с использованием своего сглаженного растрового изображения глифа с хинтингом. Гораздо лучшее качество благодаря сглаживанию, но за счет более высокой производительности. |
| AntiAlias | `4` | Каждый символ рисуется с использованием своего сглаженного растрового изображения глифа без хинтинга. Лучшее качество за счет сглаживания. Различия в ширине стержня могут быть заметны, потому что хинтинг отключен. |
| ClearTypeGridFit | `5` | Каждый символ рисуется с использованием своего растрового изображения ClearType глифа с подсказками. Настройка высочайшего качества. Используется для использования возможностей шрифта ClearType. |
