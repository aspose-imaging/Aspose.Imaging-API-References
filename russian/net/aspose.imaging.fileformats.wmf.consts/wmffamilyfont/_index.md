## WmfFamilyFont enumeration

Перечисление FamilyFont определяет семейство шрифтов. Семейства шрифтов описывают внешний вид шрифта в общем виде. Они предназначены для указания шрифтов, когда нужный шрифт недоступен.

```csharp
public enum WmfFamilyFont : byte
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Dontcare | `0` | Указан шрифт по умолчанию, который зависит от реализации. |
| Roman | `1` | Шрифты с переменной шириной штриха, пропорциональные фактической ширине глифов, и имеющие засечки. «MS Serif» является примером. |
| Swiss | `2` | Шрифты с переменной шириной штриха, которые пропорциональны фактической ширине глифов и не имеют засечек. Например, MS Sans Serif. |
| Modern | `3` | Шрифты с постоянной шириной штриха, с засечками или без них. Шрифты фиксированной ширины обычно современные. Примерами являются «Pica», «Elite» и «Courier New». |
| Script | `4` | Шрифты, разработанные так, чтобы они выглядели как рукописный текст. «Script» и «Cursive» являются примерами. |
| Decorative | `5` | Новые шрифты. Например, «древнеанглийский». |
