## EmfRestoreDc class

Запись EMR_RESTOREDC восстанавливает контекст устройства воспроизведения в указанное состояние. Контекст устройства воспроизведения восстанавливается путем извлечения информации о состоянии из стека, созданного предыдущими записями EMR_SAVEDC (раздел 2.3.11).

public sealed class EmfRestoreDc : EmfStateRecordType

### Примечания

Стек может содержать информацию о состоянии для нескольких экземпляров контекста устройства воспроизведения. Когда состояние восстанавливается, все экземпляры состояния, которые были сохранены позднее, ДОЛЖНЫ быть отброшены.
