## EmfComment.PrivateData property

Получает или задает необязательный массив байтов, определяющий частные данные. Первое DWORD этих данных НЕ ДОЛЖНО быть одним из предопределенных значений идентификатора комментария, указанных в разделе 2.3.3. Частные данные неизвестны EMF; это имеет смысл только для приложений, которые знают формат данных и как их использовать. Записи личных данных EMR_COMMENT МОГУТ игнорироваться.

public byte[] PrivateData { get; set; }
