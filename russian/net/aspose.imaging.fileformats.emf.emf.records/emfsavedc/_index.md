## EmfSaveDc class

Сохраняет текущее состояние контекста устройства воспроизведения в стеке состояний, сохраненных предыдущими записями EMR_SAVEDC , если таковые имеются. Состояние состоит из графических свойств и объектов, включая текущее выбранное растровое изображение, кисть, палитру, шрифт, перо и область. Запись An EMR_RESTOREDC используется для восстановления состояния. Эта запись EMF не содержит параметров.

public sealed class EmfSaveDc : EmfStateRecordType

### Примечания

Стек может содержать информацию о состоянии нескольких экземпляров контекста устройства воспроизведения. Когда состояние восстанавливается, все экземпляры состояния, которые были сохранены позднее, ДОЛЖНЫ быть отброшены.
