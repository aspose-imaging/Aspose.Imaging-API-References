## EmfSetTextAlign.TextAlignmentMode property

Получает или задает 32-разрядное целое число без знака, указывающее выравнивание текста по с использованием маски флагов выравнивания текста. Это либо[`WmfTextAlignmentModeFlags`](../../../aspose.imaging.fileformats.wmf.consts/wmftextalignmentmodeflags) ([MS-WMF], раздел 2.1.2.3) для текста с горизонтальной базовой линией или[`WmfVerticalTextAlignmentModeFlags`](../../../aspose.imaging.fileformats.wmf.consts/wmfverticaltextalignmentmodeflags) ([MS-WMF], раздел 2.1.2.4) для текста с базовой линией vertical . Можно выбрать только одно значение из тех, которые влияют на выравнивание по горизонтали и вертикали .

public int TextAlignmentMode { get; set; }
