## EmfBeginPath class

Эта запись открывает скобку пути в контексте текущего устройства воспроизведения. После того, как скобка пути открыта, приложение может начать обработку записей для определения точек, лежащих в пути. Приложение ДОЛЖНО закрыть открытую скобку пути путем обработки EMR_ENDPATH record. Когда приложение обрабатывает запись EMR_BEGINPATH, все предыдущие пути ДОЛЖНЫ быть исключены из контекста устройства воспроизведения.

public sealed class EmfBeginPath : EmfPathBracketRecordType
