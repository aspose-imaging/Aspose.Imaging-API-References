## CmxImagePage.CacheData method

Кэш нельзя использовать.

```csharp
public override void CacheData()
```

### Примеры

В следующем примере показано, как кэшировать все страницы изображения CMX.

```csharp
[C#]

string dir = "c:\\temp\\";

 // Загрузить изображение из файла CMX.
using (Aspose.Imaging.FileFormats.Cmx.CmxImage image = (Aspose.Imaging.FileFormats.Cmx.CmxImage)Aspose.Imaging.Image.Load(dir + "sample.cmx"))
{
     // Этот вызов кэширует только страницу по умолчанию.
    image.CacheData();

     // Кэшируем все страницы, чтобы не выполнялась дополнительная загрузка данных из базового потока данных.
    foreach (Aspose.Imaging.FileFormats.Cmx.CmxImagePage page in image.Pages)
    {
        page.CacheData();
    }
}
```
