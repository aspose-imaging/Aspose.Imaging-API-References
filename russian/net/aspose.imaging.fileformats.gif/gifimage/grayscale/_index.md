## GifImage.Grayscale method

Преобразование изображения в его представление в градациях серого

```csharp
public override void Grayscale()
```

### Примеры

Следующий пример преобразует цветное изображение GIF в его представление в градациях серого. Изображения в градациях серого состоят исключительно из оттенков серого и несут только информацию об интенсивности.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.FileFormats.Gif.GifImage gifImage = (Aspose.Imaging.FileFormats.Gif.GifImage)image;

    gifImage.Grayscale();
    gifImage.Save(dir + "sample.Grayscale.png", new Aspose.Imaging.ImageOptions.PngOptions());
}
```
