## GifImage.AdjustBrightness method

Регулировка*яркости*для изображения.

```csharp
public override void AdjustBrightness(int brightness)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| brightness | Int32 | Значение яркости. |

### Примеры

В следующем примере выполняется коррекция яркости изображения GIF.

```csharp
[C#]

string dir = "c:\\temp\\";

using (Aspose.Imaging.Image image = Aspose.Imaging.Image.Load(dir + "sample.gif"))
{
    Aspose.Imaging.FileFormats.Gif.GifImage gifImage = (Aspose.Imaging.FileFormats.Gif.GifImage)image;

     // Установить значение яркости. Допустимые значения яркости находятся в диапазоне [-255, 255].
    gifImage.AdjustBrightness(50);
    gifImage.Save(dir + "sample.AdjustBrightness.png", new Aspose.Imaging.ImageOptions.PngOptions());
}
```
