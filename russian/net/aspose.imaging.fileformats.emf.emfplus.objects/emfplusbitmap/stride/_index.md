## EmfPlusBitmap.Stride property

Получает или задает шаг изображения Шаг (4 байта): 32-битное целое число со знаком, указывающее смещение в байтах между началом одной строки сканирования и следующей. Это значение представляет собой количество байтов на пиксель, указанное в поле PixelFormat, умноженное на ширину в пикселях, указанную в поле Ширина. Значение этого поля ДОЛЖНО быть кратно четырем. Если изображение сжато в соответствии с полем Тип, это значение не определено и ДОЛЖНО игнорироваться.

public int Stride { get; set; }
