## EmfPlusColorCurveEffect.AdjustmentIntensity property

Получает или задает 32-разрядное целое число со знаком, указывающее интенсивность настройки кривой для цветового канала, заданного CurveChannel. Диапазоны значений значащих для этого поля варьируются в зависимости от значения CurveAdjustment следующим образом: Диапазон регулировки экспозиции: -255 ≤ значение &lt; 0 По мере уменьшения значения экспозиция изображения ДОЛЖНА уменьшаться. 0 Значение 0 указывает что экспозиция НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 255 По мере увеличения значения экспозиция изображения ДОЛЖНА увеличиваться. Диапазон регулировки плотности: -255 ≤ значение &lt; 0 По мере уменьшения значения плотность изображения ДОЛЖНА уменьшаться , в результате чего получается более темное изображение. 0 Значение 0 указывает, что плотность НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 255 При увеличении значения плотность изображения ДОЛЖНА увеличиваться. Диапазон регулировки контрастности: -100 ≤ значение &lt; 0 При уменьшении значения контрастность изображения ДОЛЖНА уменьшаться. 0 Значение 0 указывает, что контрастность НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 100 При увеличении значения контрастность изображения ДОЛЖНА увеличиваться. e. Диапазон регулировки подсветки: -100 ≤ значение &lt; 0 По мере уменьшения значения светлые области изображения ДОЛЖНЫ казаться темнее. 0 Значение 0 указывает, что подсветка НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 100 As при увеличении значения светлые области изображения ДОЛЖНЫ казаться светлее. Диапазон настройки тени: -100 ≤ значение &lt; 0 При уменьшении значения темные области изображения ДОЛЖНЫ казаться темнее. 0 Значение 0 указывает, что тень НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 100 При увеличении значения темные области изображения ДОЛЖНЫ казаться светлее. Диапазон регулировки насыщенности белого: 0 — 255 При увеличении значения верхняя граница диапазона цветового канала увеличивается интенсивность. Диапазон регулировки насыщенности черного: 0 — 255 По мере увеличения значения нижний предел диапазона интенсивности цветового канала увеличивается.
