## EmfPlusColorCurveEffect.AdjustmentIntensity property

Получает или задает 32-разрядное целое число со знаком, указывающее интенсивность настройки кривой для цветового канала, заданного CurveChannel. Диапазоны значений для этого поля варьируются в зависимости от значения CurveAdjustment следующим образом: Диапазон регулировки экспозиции: -255 ≤ значение &lt; 0 По мере уменьшения значения экспозиция изображения ДОЛЖНА уменьшаться. 0 Значение 0 указывает, что экспозиция НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 255 По мере увеличения значения экспозиция изображения ДОЛЖНА увеличиваться. Диапазон регулировки плотности: -255 ≤ значение &lt; 0 По мере уменьшения значения плотность изображения ДОЛЖНА уменьшаться, в результате чего изображение становится темнее. 0 Значение 0 указывает, что плотность НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 255 По мере увеличения значения плотность изображения ДОЛЖНА увеличиваться. Диапазон регулировки контрастности: -100 ≤ значение &lt; 0 По мере уменьшения значения контрастность изображения ДОЛЖНА уменьшаться. 0 Значение 0 указывает, что контраст НЕ ДОЛЖЕН изменяться. 0 &lt; значение ≤ 100 По мере увеличения значения контрастность изображения ДОЛЖНА увеличиваться. Диапазон регулировки выделения: -100 ≤ значение &lt; 0 По мере уменьшения значения светлые области изображения ДОЛЖНЫ казаться темнее. 0 Значение 0 указывает, что подсветка НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 100 По мере увеличения значения светлые области изображения ДОЛЖНЫ казаться светлее. Диапазон регулировки тени: -100 ≤ значение &lt; 0 По мере уменьшения значения темные области изображения ДОЛЖНЫ казаться темнее. 0 Значение 0 указывает, что тень НЕ ДОЛЖНА изменяться. 0 &lt; значение ≤ 100 По мере увеличения значения темные области изображения ДОЛЖНЫ казаться светлее. Диапазон регулировки насыщенности белого: 0 — 255 По мере увеличения значения увеличивается верхняя граница диапазона интенсивностей цветовых каналов. Диапазон регулировки насыщенности черного: 0 — 255 По мере увеличения значения нижняя граница диапазона интенсивностей цветовых каналов увеличивается.
