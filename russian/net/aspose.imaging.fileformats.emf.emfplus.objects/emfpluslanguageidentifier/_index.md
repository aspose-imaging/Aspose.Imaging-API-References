## EmfPlusLanguageIdentifier class

Объект EmfPlusLanguageIdentifier указывает идентификатор языка, который соответствует языку natural в локали, включая страны, географические регионы и административные районы. Каждый идентификатор языка представляет собой кодировку значения основного языка и значения подъязыка.

public sealed class EmfPlusLanguageIdentifier : EmfPlusStructureObjectType
