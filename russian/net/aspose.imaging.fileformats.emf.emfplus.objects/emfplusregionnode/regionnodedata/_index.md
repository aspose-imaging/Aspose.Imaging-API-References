## EmfPlusRegionNode.RegionNodeData property

Получает или задает необязательные данные переменной длины, определяющие узел региона объект данных, указанный в поле Тип. Содержание и формат данных могут быть разными для каждого региона типа узла. Это поле НЕ ДОЛЖНО присутствовать, если тип узла имеет тип RegionNodeDataTypeEmpty или RegionNodeDataTypeInfinite Этот объект является универсальным и используется для указания различных типов данных узла региона, включая: объект EmfPlusRegionNodePath (раздел 2.2.2.42) для конечного узла; Объект EmfPlusRectF (раздел 2.2.2.39) для терминального узла; и Объект EmfPlusRegionNodeChildNodes (раздел 2.2.2.41) для нетерминального узла.

public EmfPlusStructureObjectType RegionNodeData { get; set; }
