## EmfPlusBrightnessContrastEffect.ContrastLevel property

Получает или задает 32-разрядное целое число со знаком, указывающее уровень контрастности. Это значение ДОЛЖНО находиться в диапазоне от -100 до 100 со следующими эффектами: -100 ≤ значение &lt; 0 По мере уменьшения значения контрастность изображения ДОЛЖНА уменьшаться. 0 Значение 0 указывает, что контраст НЕ ДОЛЖЕН изменяться. 0 &lt; значение ≤ 100 По мере увеличения значения контрастность изображения ДОЛЖНА увеличиваться.
