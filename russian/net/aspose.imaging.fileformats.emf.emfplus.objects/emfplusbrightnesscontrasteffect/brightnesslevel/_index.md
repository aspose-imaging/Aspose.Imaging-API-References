## EmfPlusBrightnessContrastEffect.BrightnessLevel property

Получает или задает 32-разрядное целое число со знаком, указывающее уровень яркости. Это значение ДОЛЖНО находиться в диапазоне от -255 до 255 со следующими эффектами: -255 ≤ значение &lt; 0 По мере уменьшения значения яркость изображения ДОЛЖНА уменьшаться. 0 Значение 0 указывает, что яркость НЕ ДОЛЖНА изменяться. . 0 &lt; значение ≤ 255 По мере увеличения значения яркость изображения ДОЛЖНА увеличиваться.
