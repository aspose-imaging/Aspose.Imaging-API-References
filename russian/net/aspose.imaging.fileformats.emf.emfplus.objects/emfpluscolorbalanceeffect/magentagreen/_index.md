## EmfPlusColorBalanceEffect.MagentaGreen property

Получает или задает 32-разрядное целое число со знаком, указывающее изменение количества зеленого цвета в изображении. Это значение ДОЛЖНО быть в диапазоне от -100 до 100 с эффектами как следует: -100 ≤ значение &lt; 0 По мере уменьшения значения количество зеленого в изображении ДОЛЖНО уменьшаться, а количество пурпурного ДОЛЖНО увеличиваться. 0 Значение 0 указывает, что количество зеленого и пурпурного НЕ ДОЛЖНО изменяться. 0 &lt; значение ≤ 100 По мере увеличения значения количество зеленого в изображении ДОЛЖНО увеличиваться, а количество пурпурного СЛЕДУЕТ уменьшаться.
