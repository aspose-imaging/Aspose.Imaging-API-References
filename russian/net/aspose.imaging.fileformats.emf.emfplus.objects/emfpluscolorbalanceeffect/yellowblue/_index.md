## EmfPlusColorBalanceEffect.YellowBlue property

Получает или задает 32-разрядное целое число со знаком, указывающее изменение количества синего в изображении. Это значение ДОЛЖНО находиться в диапазоне от -100 до 100 со следующими эффектами: -100 ≤ значение &lt; 0 По мере уменьшения значения количество синего в изображении ДОЛЖНО уменьшаться, а количество желтого ДОЛЖНО увеличиваться. 0 Значение 0 указывает, что количество синего и желтого НЕ ДОЛЖНО изменяться. 0 &lt; значение ≤ 100 По мере увеличения значения количество синего в изображении ДОЛЖНО увеличиваться, а количество желтого ДОЛЖНО уменьшаться.
