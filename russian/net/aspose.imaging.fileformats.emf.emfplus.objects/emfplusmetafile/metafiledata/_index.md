## EmfPlusMetafile.MetafileData property

Получает или задает данные переменной длины, определяющие встроенный метафайл. Содержимое и формат данных могут быть разными для каждого типа метафайла.

```csharp
public byte[] MetafileData { get; set; }
```

### Примечания

Графические изображения задаются объектами EmfPlusImage (раздел 2.2.1.4). Объект EmfPlusMetafile ДОЛЖЕН присутствовать в поле ImageData объекта EmfPlusImage, если ImageTypeMetafile указан в его поле Type. Этот объект является общим и используется для различных типов данных, включая: Метафайл WMF [MS-WMF]; Метафайл WMF, который можно разместить; Метафайл EMF [MS-EMF]; Метафайл EMF+, определяющий графические операции только с записями EMF+; и Метафайл EMF+, определяющий графические операции с записями EMF+ и EMF. См. раздел 2.2.2 для спецификации дополнительных структурных объектов.
