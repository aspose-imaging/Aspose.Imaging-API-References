## EmfPlusMetafile.MetafileData property

Получает или задает данные переменной длины, определяющие внедренный метафайл. Содержимое и формат данных могут различаться для каждого типа метафайла.

```csharp
public byte[] MetafileData { get; set; }
```

### Примечания

Графические изображения задаются объектами EmfPlusImage (раздел 2.2.1.4). Объект EmfPlusMetafile ДОЛЖЕН присутствовать в поле ImageData объекта EmfPlusImage, если в его поле Type указан ImageTypeMetafile. Метафайл WMF, который можно разместить; Метафайл EMF [MS-EMF]; Метафайл EMF+, определяющий графические операции только с записями EMF+; and Метафайл EMF+, определяющий графические операции с записями EMF+ и EMF. Спецификацию дополнительных объектов структуры см. в разделе 2.2.2.
