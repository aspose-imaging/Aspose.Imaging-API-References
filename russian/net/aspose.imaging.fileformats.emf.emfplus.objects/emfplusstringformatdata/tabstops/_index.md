## EmfPlusStringFormatData.TabStops property

Получает или задает необязательный массив значений с плавающей запятой, которые определяют необязательные положения табуляции для этого объекта. Каждое значение остановки табуляции представляет собой количество пробелов между позициями табуляции или, для первой позиции табуляции, количество пробелов между началом строки текста и первой остановкой табуляции. Это поле ДОЛЖНО присутствовать, если значение поля TabStopCount в объекте EmfPlusStringFormat больше 0.

```csharp
public float[] TabStops { get; set; }
```
