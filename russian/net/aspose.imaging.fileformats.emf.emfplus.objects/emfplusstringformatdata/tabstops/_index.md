## EmfPlusStringFormatData.TabStops property

Получает или задает необязательный массив значений с плавающей запятой, которые определяют необязательные положения табуляции для этого объекта. Каждое значение табуляции stop представляет количество пробелов между табуляциями или, для первой табуляции, количество пробелов между началом табуляции. строку текста и первую позицию табуляции. Это поле ДОЛЖНО присутствовать, если значение поля TabStopCount в объекте EmfPlusStringFormat больше 0.

public float[] TabStops { get; set; }
