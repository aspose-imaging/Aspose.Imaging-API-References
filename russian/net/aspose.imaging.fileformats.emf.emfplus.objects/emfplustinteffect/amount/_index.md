## EmfPlusTintEffect.Amount property

Получает или задает 32-разрядное целое число со знаком, указывающее, насколько оттенок усиливается или ослабляется. -100 ≤ значение &lt; 0 Отрицательные значения указывают, насколько ослабляется оттенок, что соответствует добавлению черного. 0 Значение 0 указывает, что оттенок НЕ ДОЛЖЕН изменяться. 0 &lt; значение ≤ 100 Положительные значения указывают, насколько усиливается оттенок, что соответствует добавлению белого.
